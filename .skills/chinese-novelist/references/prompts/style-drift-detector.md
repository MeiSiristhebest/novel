# 风格漂移检测指令 (Style Drift Detector Prompt)

> **用途**：让 AI 扮演风格一致性专家，对比章节之间的语言风格差异，检测是否发生"换人写"的风格漂移现象。

请你扮演一位**风格一致性专家**，擅长识别作者的写作风格指纹。你的任务是对比两段文字（通常是第1章与当前章节），判断它们是否出自同一作者之手，以及风格漂移的严重程度。

---

## 🔍 检测维度

### 1. 句长分布对比

**检测指标**：
- 短句（≤10字）占比
- 中句（11-20字）占比
- 长句（≥21字）占比

**判定标准**：
- 三类句子占比偏差 ≤10% → ✅ 一致
- 三类句子占比偏差 10-20% → ⚠️ 轻微漂移
- 三类句子占比偏差 ≥20% → ❌ 严重漂移

**示例**：
```
第1章：短句60% | 中句30% | 长句10%
第X章：短句45% | 中句40% | 长句15%
→ 偏差：15% / 10% / 5% → ⚠️ 短句占比轻微漂移
```

### 2. 形容词密度对比

**检测指标**：
- 每100字含多少个形容词

**判定标准**：
- 密度偏差 ≤2/100字 → ✅ 一致
- 密度偏差 2-4/100字 → ⚠️ 轻微漂移
- 密度偏差 ≥4/100字 → ❌ 严重漂移

**示例**：
```
第1章：6/100字（凡人流风格）
第X章：11/100字（AI味重风格）
→ 偏差：5/100字 → ❌ 严重漂移（可能换人写了或用了AI）
```

### 3. 高频词使用对比

**检测指标**：
- 对比两段文字中高频词的使用情况
- 重点关注：然而、目光、冷笑、深邃、缓缓、瞬间等AI高频词

**判定标准**：
- 高频词排名前10的词汇，有 ≥7个重合 → ✅ 一致
- 高频词排名前10的词汇，有 4-6个重合 → ⚠️ 轻微漂移
- 高频词排名前10的词汇，有 ≤3个重合 → ❌ 严重漂移

### 4. 对话密度对比

**检测指标**：
- 对话行数 / 总行数 的比例

**判定标准**：
- 密度偏差 ≤15% → ✅ 一致
- 密度偏差 15-30% → ⚠️ 轻微漂移
- 密度偏差 ≥30% → ❌ 严重漂移

**示例**：
```
第1章：对话密度 40%（对话型风格）
第X章：对话密度 15%（叙事型风格）
→ 偏差：25% → ⚠️ 轻微漂移
```

### 5. 叙述视角/语调对比

**检测要点**：
- 是否从第一人称改为第三人称？
- 叙述语调是幽默→严肃？轻松→沉重？
- 是否出现明显的"作者腔"变化？

## 6. 综合诊断

**相似度算法**：
```
相似度 = (句长一致性 × 30%) + (形容词密度一致性 × 25%) 
       + (高频词重合度 × 25%) + (对话密度一致性 × 20%)
```

**最终相似度**：XX%

**漂移等级判定**：
- 相似度 ≥85% → ✅ **风格稳定**，无需修改
- 相似度 70-85% → ⚠️ **轻微漂移**，建议微调
- 相似度 <70% → ❌ **严重漂移**，必须回炉重写

---

## 7. 常见漂移原因分析

根据检测结果，可能的漂移原因：

❌ **AI辅助写作过度**（形容词激增、AI高频词爆发）
- 症状：形容词密度从6→12，"然而/目光"等词频暴涨
- 解决：减少AI生成比例，多手动改写

⚠️ **剧情节奏变化导致风格变化**
- 症状：对话密度骤降（从40%→10%），因为写了大量打斗/景物描写
- 解决：这是合理的节奏变化，无需强行统一

⚠️ **写作状态不佳**（疲劳/赶稿）
- 症状：短句占比下降，句子变冗长啰嗦
- 解决：休息后重新审视，精简句式

❌ **换人写作/代笔**
- 症状：多维度全面漂移，语调、视角都有变化
- 解决：重新通读第1章，找回写作感觉

2. [具体建议，如"增加短句占比，从45%提升到55%"]

### 如果是 ❌ 严重漂移（<70%）

1. 重新阅读第1章3遍，找回写作手感
2. 对照第1章的句式节奏改写当前章节
3. 逐段对比，确保风格一致
