> **核心理念**：写得好 ≠ 赚得到钱。商业成功需要产品思维。
> 重要：签约标准、分成、榜单口径、平台规则都可能随时间调整。若你要引用“最新榜单/当前趋势/平台审核与AI政策”，必须先按 `references/guides/platform-playbook-cn.md` 第 0 节制作快照（`.context/PLATFORM_SNAPSHOT.md` / `.context/PLATFORM_POLICY_SNAPSHOT.md`），否则不得输出“最新结论”。

## 📊 网文商业模式

### 主流平台对比

| 平台 | 类型 | 分成比例 | 特点 | 适合 |
|------|------|----------|------|------|
| **起点** | 订阅制 | 作者50% | 付费用户质量高 | 长篇男频 |
| **晋江** | 订阅制 | 作者50% | 女频天堂 | 言情、耽美 |
| **番茄** | 免费+广告 | 按阅读量 | 流量大，单价低 | 爽文、短剧向 |
| **七猫** | 免费+广告 | 按阅读量 | 下沉市场 | 通俗爽文 |
| **知乎盐选** | 订阅制 | 作者分成 | 短篇优势 | 悬疑、脑洞 |

### 收入构成

```
订阅平台收入 = 订阅收入 + 打赏收入 + 版权收入

订阅收入：
- 均订 × 章节数 × 千字单价
- 例：均订3000 × 1000章 × 0.05元 = 15万

打赏收入：
- 取决于粉丝忠诚度
- 头部作者月打赏可达数万

版权收入：
- 影视改编：几十万到几千万
- 有声书：几万到几十万
- 游戏：几十万到几百万
- 漫画：几万到几十万
```

## 🎯 签约攻略

### 各平台签约标准

| 平台 | 最低字数 | 审核周期 | 通过率 | 注意事项 |
|------|----------|----------|--------|----------|
| 起点 | 3万字 | 3-7天 | 约30% | 内容质量要求高 |
| 晋江 | 3万字 | 3-5天 | 约40% | 女频内容 |
| 番茄 | 2万字 | 1-3天 | 约60% | 相对宽松 |
| 七猫 | 2万字 | 1-3天 | 约60% | 相对宽松 |

### 签约前准备

```markdown
签约检查清单：

□ 字数达到平台要求
□ 黄金三章质量过关
□ 大纲完整（至少30万字规划）
□ 无敏感内容
□ 无抄袭嫌疑
□ 书名、简介、标签完善
```

### 提高签约率的技巧

```
1. 开头三章反复打磨
   - 编辑只看前3章决定是否签约

2. 展示商业价值
   - 在大纲中说明卖点
   - 类比成功作品

3. 选对分类
   - 了解平台热门分类
   - 不要选冷门分类

4. 时机选择
   - 月初投稿（编辑KPI压力小）
   - 避开节假日
```

## 📈 上架与追读优化

### 上架条件

```
起点/晋江：
- 签约后10-20万字可申请上架
- 需要编辑审批

番茄/七猫：
- 通常签约即上架
- 或达到一定字数自动上架
```

### 上架前的准备

```
上架前1周：

□ 存稿至少20章
□ 检查前30章质量
□ 优化书名和简介
□ 准备上架感言
□ 规划上架后更新节奏

上架当天：

□ 稳定双更或三更
□ 感谢免费期读者
□ 引导订阅
□ 在作者的话里互动
```

### 追读率优化

```
追读率 = 当天阅读人数 / 收藏人数

追读率意义：
- 30%以上：优秀
- 20-30%：正常
- 低于20%：需要优化

提高追读率的方法：

1. 稳定更新
   - 固定时间更新
   - 不断更

2. 结尾钩子
   - 每章结尾留悬念
   - 让读者期待明天

3. 节奏控制
   - 不要连续低潮
   - 保持爽点密度

4. 互动维护
   - 回复评论
   - 让读者有参与感
```

## 📖 书名、简介、封面

### 书名设计

```
好书名的特点：
1. 简短有力（4-8字最佳）
2. 有记忆点
3. 暗示类型
4. 避免撞名

书名模式：

主角+设定：《凡人修仙传》《赘婿》
设定+动词：《斗破苍穹》《吞噬星空》
情绪+名词：《诡秘之主》《大奉打更人》
疑问式：《我真不是大魔王》

检查：
□ 是否有歧义？
□ 是否容易搜索？
□ 是否与热门书撞名？
```

### 简介写作

```
简介模板（150-200字）：

【第一句：核心设定/钩子】
"当林凡发现自己能看到别人头顶的死亡倒计时，他的人生彻底改变了。"

【第二段：核心冲突】
"为了拯救只剩3天寿命的妹妹，他必须找到改变命运的方法。"

【第三段：核心卖点】
"然而，能看到死亡的人，自己的命数也在被别人窥视..."

【结尾：引导阅读】
"一个关于命运与选择的故事。"
```

### 封面设计

```
自己做（省钱）：
- 使用Canva等工具
- 参考同类热门书封面
- 保持简洁，字体清晰

请人做（质量好）：
- 淘宝约50-200元
- 专业设计师更贵
- 建议签约后再投入

封面要素：
- 书名清晰可读
- 风格符合类型
- 颜色醒目
- 主视觉突出
```

## 🏆 冲榜策略

### 榜单类型

| 榜单 | 规则 | 作用 | 操作空间 |
|------|------|------|----------|
| 新书榜 | 新书期数据 | 获取初期流量 | 中 |
| 畅销榜 | 订阅销售额 | 证明商业价值 | 低 |
| 月票榜 | 月票数量 | 粉丝活跃度 | 中 |
| 推荐票榜 | 推荐票 | 读者认可度 | 低 |
| 完本榜 | 完本作品评分 | 长尾流量 | 低 |

### 新书期冲榜

```
新书期（前30天）是关键窗口期！

策略：

1. 开局存稿爆发
   - 开书前存30章
   - 开书后连续7天爆更

2. 稳定更新
   - 不要断更
   - 保持日更6000+

3. 争取推荐
   - 表现好会获得编辑推荐
   - 推荐位流量巨大

4. 引导互动
   - 引导投票、评论
   - 数据好看更容易上榜
```

### 月票冲刺

```
月末冲刺策略：

1. 提前预告
   - 月末前3天开始预热

2. 加更激励
   - 月票满XX张加更

3. 剧情配合
   - 月末安排高潮剧情

4. 感谢互动
   - 及时感谢投票读者
```

## 💰 版权开发

### 版权类型

| 类型 | 价格范围 | 周期 | 成功率 |
|------|----------|------|--------|
| 有声书 | 3-50万 | 1-3月 | 高 |
| 漫画 | 5-30万 | 2-6月 | 中 |
| 影视 | 10-5000万 | 1-5年 | 低 |
| 游戏 | 20-500万 | 1-3年 | 低 |
| 出版 | 3-20万 | 6-12月 | 中 |

### 提高版权价值

```
影视改编潜力提升：

1. 故事结构清晰
   - 有明确的三幕结构
   - 每卷都有高潮

2. 人物塑造立体
   - 主角有成长弧光
   - 配角有记忆点

3. 场景可视化强
   - 画面感强的描写
   - 适合影视呈现

4. 题材契合市场
   - 关注影视市场趋势
   - 古装、都市、悬疑热门
```

## 七、实战转化：AI 如何运用本文 (Practical Application)

> **给 AI 的行为指令**：当用户咨询商业化问题时，必须读取并应用本文的策略。

### 7.1 触发场景

| 用户说/做 | AI 必须执行 |
|:----------|:------------|
| "怎么签约" | 引用签约攻略：检查字数、黄金三章、大纲是否完善 |
| "选什么平台" | 引用平台对比表：根据流派和目标推荐 |
| "书名怎么起" | 引用书名设计：4-8字、有记忆点、暗示类型 |
| "简介怎么写" | 提供简介模板：钩子→冲突→卖点→引导 |
| "怎么提高追读" | 引用追读优化：稳定更新、结尾钩子、节奏控制、互动维护 |
| "怎么冲榜" | 引用冲榜策略：新书期存稿爆发、稳定更新、争取推荐 |

### 7.2 AI 可执行的商业化辅助

当用户准备商业化时，AI 应主动检查：

```markdown
## 商业化就绪检查

### 签约前 □ 字数达到平台要求？
□ 黄金三章打磨过？
□ 大纲完整（至少30万字）？
□ 书名、简介、标签完善？

### 上架前
□ 存稿至少20章？
□ 前30章质量过关？
□ 准备好上架感言？

### 完本后
□ 完结评分维护？
□ 版权开发跟进？
```

### 7.3 输出建议模板

当用户咨询商业问题时，AI 应输出：

```markdown
## 商业化建议

**当前状态**：[签约前/连载中/完本后]
**平台推荐**：[根据流派和目标]
**关键行动**：
1. [最重要的一步]
2. [次要行动]

**数据参考**：
- 该平台签约通过率：约XX%
- 建议日更字数：XXXX字
```
