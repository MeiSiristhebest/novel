# PLATFORM_STRATEGY_CARD（平台策略卡，<=12行）

> 用途：把平台偏好从“参考知识”变成“硬约束”。
> 规则：若目标平台已明确，每轮写作/质检都必须提供本卡。

1. 目标平台：{{platform}}（起点/番茄/晋江/飞卢）
2. 商业模型：{{business_model}}（付费订阅/免费广告/女性向付费+互动/快节奏订阅）
3. 开篇强度：{{opening_strength}}（0-300/300-1000 的冲突与代价要求）
4. 章节节奏：{{pacing_rule}}（每 300-500 字必须发生的变化类型）
5. 章末钩子：{{end_hook_rule}}（优先钩子类型 + 禁止弱钩子）
6. 更新红线：{{update_redline}}（断更/更新密度底线）
7. 情绪回报：{{emotional_payoff}}（本平台读者最吃的结算方式）
8. 标题题眼：{{title_hook}}（反常设定/强爽点/强关系张力）
9. 禁忌：{{taboos}}（最讨厌的 3 件事）
