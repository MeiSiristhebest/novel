# 意象与比喻库 (Metaphor \u0026 Imagery Library)

> **使用指南**：比喻不是装饰品，是理解的桥梁。好的比喻让抽象变具体，让陌生变熟悉。
> **原则**：宁可不用，不用烂俗。

## 七、比喻使用的禁忌 (Don'ts)

### 7.1 陈词滥调黑名单

**绝对避免**：

| ❌ 禁用 | 原因 |
|--------|------|
| 美如天仙 | 太老套，毫无新意 |
| 快如闪电 | 用烂了 |
| 坚如磐石 | 老套 |
| 破茧成蝶 | 网文用烂了 |
| 浴火重生 | 太常见 |
| 刻骨铭心 | 成语直接用，没有画面感 |
| 心如刀绞 | 太直白 |

### 7.2 比喻的一致性原则

**错误示例**（意象混乱）：

```markdown
他像一只孤狼（第3章）
他像一头雄狮（第10章）
他像一只猎鹰（第20章）
```

**正确做法**：选定一个核心意象，贯穿始终

### 7.3 比喻的适度性

**错误**：比喻过密
```markdown
他的眼神如刀，心如铁，手如鹰爪，身如猎豹……
```

**正确**：一个精准的比喻足矣
```markdown
他是一把锈迹斑斑的剑——老了，但依然能杀人。
```

## 九、使用建议 (Best Practices)

1. **场景优先于词汇**  
   不要为了用比喻而用，先问：这个场景需要吗？

2. **新鲜感 > 准确性**  
   宁可稍微不那么精确，也要避免陈词滥调

3. **一章1-2个即可**  
   精彩的比喻是点睛，不是铺满画布

4. **让比喻服务于人物**  
   不同性格的角色，内心独白中的比喻应该不同

5. **创造你自己的意象**  
   本库是工具，最好的意象是你独创的

---

**最后一句话**：

> 比喻是诗人的语言，让它为你的故事增色，而不是喧宾夺主。
