# 转场技巧 (Transition Techniques)

> **核心理念**：好的转场让读者无缝滑入下一个场景，差的转场让读者出戏。

---

## 📊 转场类型

### 三种转场需求

| 转场类型 | 说明 | 难度 | 常见问题 |
|----------|------|------|----------|
| **时间转场** | 从现在跳到之后 | ⭐⭐ | 跳跃太突兀 |
| **空间转场** | 从这里跳到那里 | ⭐⭐ | 方位不清 |
| **视角转场** | 从A角色跳到B角色 | ⭐⭐⭐ | 读者混乱 |

---

## ⏰ 时间转场

### 短时间跳跃（几小时-几天）

```
技巧1：时间词引导
"三个时辰后，许凡从修炼中睁开眼睛。"
"第二天清晨，他准时出现在约定地点。"
"两天后，结果终于出来了。"

技巧2：动作省略
"他倒头便睡。
再次醒来时，阳光已经洒满房间。"

技巧3：场景切换暗示
"他转身离开。
---
城东茶楼，午后。"
```

### 长时间跳跃（数月-数年）

```
技巧1：明确标注
"三年后——"
"【五年之后】"

技巧2：变化展示
"三年时光，白驹过隙。
当许凡再次出现在青云镇时，已无人能认出他。
当年的废材少年，如今已是筑基期高手。"

技巧3：他人视角
"'许师兄？是你？'
林小妹难以置信地看着眼前的男子。
三年不见，他的气质已经完全不同。"
```

### 时间转场常见错误

```
❌ 错误1：无标识跳跃
"他开始修炼。他已经修炼到筑基期了。"
（读者：什么？怎么就筑基了？）

✅ 修正：
"他开始修炼。
三个月后，他终于突破到筑基期。"

❌ 错误2：跳跃后无交代
"五年后，许凡回来了。"
（读者：这五年发生了什么？）

✅ 修正：
"五年后，许凡回来了。
这五年，他走遍了大陆北部，经历了无数生死，
终于将那门功法修炼到了大成。"
```

## 🗺️ 空间转场

### 短距离转场（同一区域内）

```
技巧1：动作引导
"他走出房间，穿过长廊，来到了大厅。"
"她三步并作两步，冲进了后院。"

技巧2：省略过程
"片刻后，他已站在悬崖边。"

技巧3：视线引导
"他抬头望向远处的山峰，心中一动，
身形一闪，已落在山巅。"
```

### 长距离转场（跨区域/跨地图）

```
技巧1：分割线+地点标注
"---
（场景转换：京城，皇宫）
"

技巧2：交通工具
"飞舟破空而去，三日后，落在了南海之滨。"
"传送阵闪烁，许凡出现在万里之外的中州。"

技巧3：新场景描写
"与北方的苍凉不同，南域是一片水乡泽国。
碧波荡漾，荷叶田田，空气中弥漫着潮湿的水汽。"
```

### 空间转场常见错误

```
❌ 错误1：瞬移无解释
"他在青云镇，下一秒就到了京城。"
（读者：怎么过去的？？）

✅ 修正：
"他踏上传送阵，白光一闪，已身在京城。"

❌ 错误2：方位混乱
"他向东走，来到了西边的山洞。"
（读者：？？？）

✅ 修正：
"他转向西行，半日后，终于找到了那处山洞。"
```

## 👤 视角转场

### 同章内视角切换

```
技巧1：分割线标识
"许凡转身离去，消失在夜色中。

---

（与此同时，林府）

林老爷子紧皱眉头，手中的茶杯已经凉透。"

技巧2：明确主语
"许凡心中大定。
而他不知道的是，林小姐此刻正站在窗边，
望着他离去的方向，久久无法平静。"
```

### 跨章视角切换

```
技巧1：章节标题标注
"第一百章 许凡的选择
第一百零一章 林小姐的心事"

技巧2：开头明确
"（本章为林小姐视角）
林小姐已经三天没有出门了。"
```

### 视角转场规则

```yaml
核心规则:
  1. 同一段落内不要切换视角
  2. 切换要有明确标识
  3. 切换后立即明确"现在是谁的视角"
  4. 主角视角占绝对多数（80%+）

切换频率:
  同一章内: 最多2-3次
  连续配角章: 最多2章就要回主角
```

## 🎬 电影式转场技巧

### 1. 切镜式

```
"刀光一闪——
鲜血飞溅——
敌人倒下。"

特点：快速、有冲击力
适合：战斗高潮
```

### 2. 蒙太奇式

```
"那一夜，整个城市都不平静。
东城，有人在逃亡。
西城，有人在追杀。
南城，有人在等待。
北城，有人在哭泣。"

特点：平行展示，制造氛围
适合：大事件、多线汇聚
```

### 3. 闪回式

```
"看着眼前的场景，他突然想起了什么。

---
（十年前）

那时候，他还是个孩子...
---

'原来如此。'他终于明白了。"

特点：补充信息，揭示过去
适合：伏笔回收、人物背景
```

### 4. 预叙式

```
"三天后，当这座城市化为废墟时，
所有人都会记得这个夜晚的选择。

但此刻，他们还不知道命运的齿轮已经开始转动。

---
（三天前）
"

特点：制造悬念，吊起胃口
适合：章节开头、重大事件铺垫
```

## 📋 转场检查清单

### 每次转场检查

```markdown
时间转场：
□ 跳跃时长是否明确？
□ 期间发生的重要事件是否交代？
□ 读者能否理解时间线？

空间转场：
□ 新地点是否明确？
□ 如何到达是否交代？
□ 新环境是否有描写？

视角转场：
□ 切换是否有标识？
□ 新视角人物是否明确？
□ 切换是否有必要？
```
