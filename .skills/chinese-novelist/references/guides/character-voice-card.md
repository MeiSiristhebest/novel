# 角色声音档案 (Character Voice Card)

> **核心目标**：解决“千人一面”问题。
> **检验标准**：遮住名字，读者能通过说话方式猜出是谁。

---

## 一、声音设计模版 (Voice Design Template)

为每个主要角色填写此卡片。

### [角色姓名]

**1. 核心关键词 (The Vibe)**
*   *形容词*：[例如：粗鲁、文雅、急躁、冷漠、阴阳怪气]
*   *参照人物*：[例如：钢铁侠的毒舌+福尔摩斯的语速]

**2. 说话节奏 (Rhythm)**
*   **句长偏好**：
    *   [ ] 喜欢长句，逻辑严密，多用连接词（如：学者/贵族）
    *   [ ] 喜欢短句，甚至只有单词，省略主语（如：杀手/硬汉）
    *   [ ] 喜欢破碎句，结巴，重复（如：胆小鬼/疯子）
*   **语速感**：[快/慢/忽快忽慢]

**3. 词汇指纹 (Vocabulary Fingerprint)**
*   **口头禅/高频词**：[例如：“有趣”、“恕我直言”、“他妈的”、“从理论上讲”]
*   **禁用词 (Banned Words)**：[这个角色绝不会说的话。例如：硬汉绝不说“哎呀”、“好可怕”]
*   **特殊称呼**：[对主角/他人的特定称呼。例如：叫主角“小鬼”、“老板”、“喂”]

**4. 潜台词习惯 (Subtext Style)**
*   *类型*：
    *   [ ] **直肠子**：想啥说啥，没有潜台词。
    *   [ ] **被动攻击**：嘴上客气，心里骂人。
    *   [ ] **谜语人**：永远只说一半，喜欢用隐喻。
    *   [ ] **谎言家**：习惯性撒谎，掩盖真实意图。

**5. 肢体习惯 (Body Language)**
*   说话时喜欢做什么？[例如：推眼镜、摸鼻子、玩弄硬币、不看对方眼睛]

## 二、声音对比示例 (Examples)

### 场景：面对危险

*   **角色A (热血笨蛋)**
    > “来啊！老子正好手痒了！看我不把这群混蛋揍飞！” (短句，感叹号，粗俗)

*   **角色B (冷静智囊)**
    > “胜率不足 5%。建议撤退。如果你非要送死，请先把欠我的钱还了。” (数据，逻辑，冷幽默)

*   **角色C (优雅反派)**
    > “真遗憾。这件衬衫是我新买的，本来不想沾血的。” (关注点偏离，从容，礼貌的残忍)

*   **角色D (胆小鬼)**
    > “那个...我们要不...还是跑吧？我腿有点软...真的...” (省略号，迟疑，请求)

## 三、对话自检清单 (Voice Check)

写完一段对话后，检查：

1.  **这个角色会用这个词吗？** (例如：一个古代侠客不会说“这个逻辑有问题”)
2.  **这句话是否太长/太短？** (符合角色的呼吸节奏吗？)
3.  **有没有体现潜台词？** (他真正想说的是什么？)
4.  **能否删掉标签？** (删掉“他说”，还能分清是谁吗？)
