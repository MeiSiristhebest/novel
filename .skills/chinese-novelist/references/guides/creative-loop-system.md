# 顶级作家创作闭环系统 (The Creative Loop System)

> **核心理念**：写作不是线性的"生成"，而是一个"观察-构思-输出-修正-复盘"的无限循环。顶级作家与普通写手的区别在于是否有意识地维护这个闭环。

---

## 🔁 六阶段循环模型

### 1. 观察与输入 (Observe & Research)
*“输出是输入的投影。没有优质的输入，AI 也只能生成陈词滥调。”*

*   **行动指令**：
    *   建立素材库 (`materials/scraps.md`)。
    *   **生活切片**：记录今天的对话、观察到的路人表情、情绪波动。
    *   **行业调研**：如果是写医生文，必须去读相关论文或纪录片，而不是只看别人的小说。
    *   **竞品拆解**：使用 `deconstructing-masterpieces.md` 拆解同类爆款。

### 2. 构思与孵化 (Ideate)
*“不要急着写正文，先让点子在脑子里飞一会儿。”*

*   **行动指令**：
    *   **高概念测试**：用一句话讲清楚故事的卖点（Logline）。
    *   **差异化验证**：我的故事和市面上那100本同类书有什么不同？（使用 `aesthetic-charter.md` 校验）。
    *   **冰山构建**：只写出10%，但要设定好水下的90%。

### 3. 粗放输出 (Drafting / Zero Draft)
*“关闭内心的评论员。写垃圾也没关系，先写完。”*

*   **原则**：
    *   **速度优先**：不要回头改错别字。
    *   **情绪连贯**：保持心流，不要中断去查资料（打个标记 [TODO: 查资料] 继续写）。
    *   **数量堆砌**：这一阶段的目标是字数，不是质量。

### 4. 宏观修订 (Macro Revision)
*“现在，戴上编辑的帽子，无情地审视结构。”*

*   **检查清单**：
    *   **结构完整性**：激励事件-冲突-高潮-结局是否清晰？
    *   **节奏控制**：是否太拖沓？哪里需要加速？
    *   **人物动机**：主角的行为是否合乎逻辑？有没有被剧情强行推动？
    *   **删减**：勇敢地删掉那些"写得好但对剧情无用"的段落（Kill your darlings）。

### 5. 微观打磨 (Micro Revision)
*“上帝在细节中。魔鬼也在细节中。”*

*   **行动指令**：
    *   **展示不告知 (Show, Don't Tell)**：把"他很生气"改成"他捏碎了手里的玻璃杯"。
    *   **五感增强**：加入气味、触觉、声音描写。
    *   **语言降噪**：删除形容词，强化动词。删除"觉得"、"开始"、"突然"等废词。
    *   **声音统一**：检查对话是否符合人物性格（使用 `character-voice-card.md`）。

### 6. 发布与复盘 (Publish & Postmortem)
*“每一本书都是下一本书的练习。”*

*   **复盘四问**：
    1. 这一章读者反应最好的点是什么？（爽点命中率）
    2. 读者在哪里弃文了？（毒点/尿点分析）
    3. 我写得最痛苦的地方是哪里？（能力短板）
    4. 下一章如何改进？

---

## 🛠️ 每日行动清单 (Daily Action)

建议每天分配时间比例：
- **输入 (20%)**：阅读、观察、记录。
- **输出 (50%)**：专注写作。
- **修订 (20%)**：修改旧稿。
- **复盘 (10%)**：思考与总结。

## 📂 推荐文件结构

```text
novels/[项目名]/
├── materials/          # 素材库
│   ├── scraps.md       # 灵感碎片
│   ├── research.md     # 调研资料
│   └── reference.md    # 竞品分析
├── drafts/             # 初稿（不删改）
├── revisions/          # 修订版
├── postmortem/         # 复盘日志
└── aesthetic-charter.md # 审美宪章
```
