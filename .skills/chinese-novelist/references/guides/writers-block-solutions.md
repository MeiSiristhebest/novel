# 卡文急救包 (Writer's Block Solutions)

> **核心理念**：卡文不可怕，可怕的是没有应对方法。这里是你的急救工具箱。

---

## 🚨 卡文类型诊断

### 先诊断，后治疗

| 卡文类型 | 症状 | 核心原因 | 急救方法 |
|----------|------|----------|----------|
| **大纲卡** | 不知道接下来写什么 | 大纲不够细 | 细化大纲法 |
| **开头卡** | 第一句写不出来 | 完美主义 | 垃圾初稿法 |
| **场景卡** | 知道要写什么，但写不出来 | 缺乏画面感 | 五感激活法 |
| **对话卡** | 对话写得干巴巴 | 角色声音不清 | 角色扮演法 |
| **情绪卡** | 写得没感觉 | 情绪疲惫 | 情绪充电法 |
| **倦怠卡** | 完全不想写 | 身心俱疲 | 休息重启法 |

## 💊 十大急救技巧

### 1. 垃圾初稿法

**适用**：完美主义导致的卡文

```
核心理念：
"允许自己写垃圾，写完再改。"

操作步骤：
1. 告诉自己："这一稿就是要写烂的"
2. 不回看，不修改，一直往前写
3. 用最简单的句子写
4. 写完整章再回头修改

心理暗示：
"初稿是用来改的，不是用来发表的。"
```

### 2. 场景跳跃法

**适用**：某个场景写不下去

```
操作步骤：
1. 跳过卡住的场景
2. 直接写下一个你想写的场景
3. 写完后再回填跳过的部分
4. 有时候跳过的部分根本不需要写

示例：
卡在："主角如何从A地到B地"
解决：直接写"主角已经到了B地"
```

### 3. 倒叙破卡法

**适用**：不知道怎么开头

```
操作步骤：
1. 先写结果："他站在尸体面前"
2. 再倒推过程：怎么走到这一步的？
3. 从结果倒推往往更容易

示例：
卡在：战斗怎么开始
解决：先写"战斗结束，主角擦了擦刀上的血"
      再倒推战斗过程
```

### 4. 对话驱动法

**适用**：场景描写卡住

```
操作步骤：
1. 暂时放弃场景描写
2. 只写对话，像剧本一样
3. 对话写完后再补充动作和描写
4. 对话能推动剧情，描写可以后加

示例：
先写：
"你来了。"
"不得不来。"
"那件事...你想好了？"
"想好了。"

再补充表情、动作、环境
```

### 5. 五感激活法

**适用**：场景没有画面感

```
操作步骤：
1. 闭上眼睛，想象自己站在场景中
2. 依次感受：看到什么？听到什么？闻到什么？
3. 把感受到的写下来
4. 不需要都用，选最有冲击力的1-2个

五感清单：
□ 视觉：颜色、光线、形状、动态
□ 听觉：声音、节奏、远近
□ 嗅觉：气味、强度
□ 触觉：温度、质感、痛感
□ 味觉：口中滋味（少用）
```

### 6. 角色附体法

**适用**：对话写不出角色感

```
操作步骤：
1. 选择一个你熟悉的演员/角色
2. 想象他/她在演这个角色
3. 他/她会怎么说这句话？
4. 写下来

示例：
写霸道总裁：想象靳东
写古灵精怪：想象赵丽颖
写阴沉反派：想象张震
```

### 7. 问题驱动法

**适用**：不知道写什么

```
操作步骤：
问自己这些问题：
1. 主角现在最想要什么？
2. 什么阻止他得到？
3. 他会怎么做？
4. 会有什么后果？

然后把答案写成场景。
```

### 8. 读者视角法

**适用**：写得没意思

```
操作步骤：
1. 想象一个读者正在看这章
2. 他看到这里会问什么问题？
3. 他最期待看到什么？
4. 给他想看的！

常见读者期待：
- 打脸爽点
- 感情进展
- 谜题答案
- 能力展示
- 逆袭时刻
```

### 9. 碎片拼接法

**适用**：状态很差但必须写

```
操作步骤：
1. 不要想着写完整的一章
2. 写碎片：一个画面、一句对话、一个动作
3. 积累够多碎片后再拼接
4. 状态差时产出慢很正常

示例：
碎片1："他的手在颤抖。"
碎片2："窗外的雨越下越大。"
碎片3："她说了那三个字。"
→ 拼接成完整段落
```

### 10. 设置障碍法

**适用**：剧情推不动

```
操作步骤：
1. 给主角加一个障碍
2. 障碍可以是：敌人、误会、意外、时限
3. 有了障碍，剧情自然推进

万能障碍清单：
- 时间紧迫："必须在天亮前..."
- 敌人出现："突然，一道身影挡住了去路"
- 意外发生："正在这时，天空传来轰鸣"
- 信息错误："他不知道的是，那个消息是假的"
```

## 🔧 预防卡文的方法

### 每日写作前

```markdown
写作前热身清单：

□ 回顾昨天写的最后500字
□ 明确今天要写的核心事件
□ 列出今天的3个关键场景
□ 想好第一句话再开始
□ 设定今天的最低目标（如2000字）
```

### 每周计划

```markdown
周规划模板：

本周目标：写完第X章-第Y章

周一：第X章（场景A、场景B）
周二：第X+1章（场景C、场景D）
周三：第X+2章（场景E）
周四：修改前三天的内容
周五：第X+3章
周六：存稿
周日：休息/构思下周
```

### 存稿策略

```
存稿是预防卡文的最佳武器！

推荐存稿量：
- 新书期：至少存20章再开书
- 日常期：保持5-10章存稿
- 高潮期：提前存15章以上

存稿的好处：
1. 卡文时有缓冲
2. 可以冷静修改
3. 减少断更压力
4. 保持更新稳定
```

## 🆘 紧急情况处理

### 今天必须更新但写不出来

```
紧急方案（按优先级）：

1. 发存稿（如果有）

2. 写"过渡章"：
   - 主角修炼/休息
   - 配角视角插入
   - 回忆/闪回
   - 势力格局分析

3. 发"短章"：
   - 正常章节的一半
   - 加作者的话解释

4. 请假：
   - 诚恳说明情况
   - 承诺补更时间
```

### 长期写不下去

```
可能是大问题：

1. 检查大纲
   - 大纲是否有问题？
   - 当前剧情是否偏离？

2. 检查人物
   - 主角是否还有魅力？
   - 动机是否还成立？

3. 检查自己
   - 是否需要休息？
   - 是否对这本书失去兴趣？

解决方案：
- 休息3-7天
- 重读自己写过的精彩章节
- 重新规划后续剧情
- 如果实在写不下去，考虑完结或太监
```
