### 日更1000章需要多久？

```
按每章3000字，1000章=300万字

日更4000字 → 750天 ≈ 2年
日更6000字 → 500天 ≈ 1.4年
日更8000字 → 375天 ≈ 1年

这是一场持久战！
```

09:00-12:00  写作（存稿）

目标：周末存够5章

## 📝 产出效率技巧

### 提速方法

#### 1. 细纲写作法

```
写作前先列细纲：

【第X章细纲】
- 开场：主角进入xxx，遇到xxx
- 发展：对话内容要点（1、2、3）
- 冲突：xxx发生
- 结尾：xxx的反应，留钩子

有细纲，写作速度提升50%以上！
```

#### 2. 番茄工作法

```
25分钟专注写作 → 5分钟休息 → 循环

目标：
- 1个番茄 = 500-800字
- 4个番茄 = 2000-3200字
- 8个番茄 = 4000-6400字

工具推荐：番茄TODO、Forest
```

#### 3. 语音输入法

```
用嘴说比用手打快3-4倍！

流程：
1. 用语音输入APP说出内容
2. 转文字后修改错字
3. 润色调整

适合：
- 对话场景
- 内心独白
- 打斗描写

工具：讯飞语音、搜狗语音
```

#### 4. 模板写作法

```
常见场景都可以有模板：

【战斗场景模板】
- 对峙（50字）
- 试探交手（200字）
- 升级对抗（300字）
- 关键一击（200字）
- 结果（100字）

【日常对话模板】
- 开场白（30字）
- 主题展开（200字）
- 信息传递（150字）
- 情感互动（100字）
- 结束语（50字）
```

## 💪 身体保养指南

### 职业病预防

| 问题 | 症状 | 预防措施 |
|------|------|----------|
| **颈椎问题** | 脖子僵硬、疼痛 | 每小时起来活动、米字操 |
| **腰椎问题** | 腰酸背痛 | 好椅子、站立写作交替 |
| **腕管综合征** | 手腕疼、手指麻 | 正确键盘姿势、手腕操 |
| **眼睛疲劳** | 干涩、视力下降 | 20-20-20法则、滴眼液 |
| **久坐代谢** | 肥胖、三高 | 每天运动30分钟 |

### 20-20-20法则

```
每20分钟，看20英尺（6米）外的东西，持续20秒

这是保护眼睛最简单有效的方法！
```

### 作者运动推荐

```
低强度（每天）：
- 散步30分钟
- 拉伸操10分钟
- 眼保健操

中强度（每周3次）：
- 跑步/游泳30分钟
- 力量训练

建议：
- 写作前运动，提升专注力
- 或者写作中间运动，恢复精力
```

## 🧠 心理状态管理

### 常见心理问题

| 问题 | 表现 | 应对方法 |
|------|------|----------|
| **自我怀疑** | 觉得自己写得很烂 | 看读者好评、对比进步 |
| **数据焦虑** | 不停刷后台数据 | 设定固定看数据时间 |
| **比较焦虑** | 看别人成绩焦虑 | 只和过去的自己比 |
| **完美主义** | 一章改无数遍 | 设定修改次数上限 |
| **倦怠期** | 完全不想写 | 休息、换题材、找动力 |

### 保持动力的方法

```
短期动力：
- 每日打卡成就感
- 读者好评截图保存
- 小目标奖励（写完1万字吃顿好的）

中期动力：
- 月收入目标
- 完本目标
- 粉丝数目标

长期动力：
- 作家梦想
- 财务自由
- 创作价值感
```

### 情绪急救包

```
当情绪很差时：

1. 停止写作，离开电脑
2. 做一件让自己开心的事
3. 运动/散步/听音乐
4. 和朋友聊天
5. 睡一觉

第二天再继续。
一天不更不会死，身心健康更重要。
```

## 📅 请假与断更策略

### 请假的正确方式

```
请假前：
□ 发存稿顶上
□ 提前告知读者
□ 说明返回时间
□ 承诺补更

请假条模板：
"各位读者好，因[原因]，今天请假一天。
明天[时间]恢复更新，会补上今天的份。
抱歉给大家带来不便！"
```

### 断更的代价

```
断更1天：追读下降5-10%
断更3天：追读下降20-30%
断更1周：追读下降50%+
断更1月：基本凉凉

结论：尽量不要断更！
实在要断，请假比消失强。
```

### 补更策略

```
断更后的补救：

1. 回来第一天双更
2. 连续3天保持双更
3. 发个福利章节
4. 在作者的话里道歉
5. 逐渐恢复读者信任
```

## 📋 日更检查清单

### 每日检查

```markdown
写作前：
□ 回顾昨天内容
□ 确认今天目标
□ 准备好细纲
□ 清除干扰（手机静音等）

写作中：
□ 专注模式开启
□ 按番茄钟工作
□ 每小时起来活动
□ 保持水分补充

写作后：
□ 完成字数目标
□ 检查并发布
□ 回复重要评论
□ 规划明天内容
```

### 每周检查

```markdown
□ 本周总字数达标？
□ 存稿量是否够？
□ 身体状态如何？
□ 下周大纲是否清晰？
□ 有无需要调整的地方？
```
