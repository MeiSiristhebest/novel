# 章节微观结构 (Chapter Micro-Structure)

> **核心法则**：每一章都是一个微型的电影，必须有完整的起承转合。
> **拒绝流水账**：不要记录角色吃饭睡觉，除非饭里有毒或者睡在敌人床上。

---

## 一、标准章节骨架 (The Standard Skeleton)

一个标准的 3000 字章节应包含以下 4 个部分：

### 1. 开篇钩子 (The Hook) - 前 5%
**目标**：在 3 秒内抓住读者的注意力。
*   **切入点**：
    *   **In Medias Res (从中间开始)**：直接切入动作或冲突中，不要铺垫背景。
    *   **悬念切入**：承接上一章的悬念，立即给出反应。
    *   **反常切入**：描写一个极度不合理/诡异的画面。
*   **禁忌**：禁止从天气描写、起床刷牙、或者长篇大论的世界观介绍开始。

### 2. 铺垫与阻力 (The Build-up) - 30%
**目标**：建立期待，给主角制造麻烦。
*   **情境建立**：快速交代“谁、在哪、要做什么”。
*   **阻力出现**：主角想做某事，但遇到了困难（敌人、环境、规则）。
*   **尝试与失败**：主角尝试解决，但第一招失败了，或者情况变得更糟。

### 3. 高潮与转折 (The Climax/Twist) - 50%
**目标**：情绪释放，信息增量。
*   **核心冲突**：本章最激烈的时刻（战斗、争吵、揭秘）。
*   **主动时刻**：主角必须**主动**做点什么来改变局势（不能全靠运气）。
*   **信息揭示**：读者必须获得新的剧情碎片（线索、真相、新地图）。

### 4. 结尾钩子 (The Cliffhanger) - 后 15%
**目标**：强迫读者点击下一章。
*   **悬念设置**：
    *   **危机悬念**：刀架在脖子上。
    *   **信息悬念**：发现了一个惊人的秘密，但没看完。
    *   **选择悬念**：必须在两个糟糕的选项中选一个。
*   **收束**：稍微平复情绪，然后迅速拉高期待。

## 二、场景切换与节奏控制 (Pacing Control)

### 1. 场景切换 (Scene Cuts)
*   **硬切 (Hard Cut)**：直接跳到下一个场景，不要写路上的过程。
    *   *❌ 错误*：他走出房间，穿过走廊，下了楼梯，走出大门，打了一辆车，半小时后到了目的地。
    *   *✅ 正确*：他摔门而出。/ 暴雨中的墓地，他站在墓碑前全身湿透。
*   **蒙太奇 (Montage)**：用排比句快速压缩时间（用于修炼、赶路）。

### 2. 变速齿轮 (The Gear Shift)
*   **加速 (Action)**：打斗、争吵、逃亡。
    *   *技巧*：短句、省略主语、动词密集、少形容词。
    *   *示例*：刀光一闪。血溅五步。他没退，反进一步。
*   **减速 (Reflection)**：情感、氛围、思考。
    *   *技巧*：长句、感官描写（声光色味）、内心独白。
    *   *示例*：雨水顺着他的发梢滴落，在积水中晕开一圈圈涟漪，倒影里那张脸苍白得像个死人。

## 三、微观结构自检表 (Self-Check)

每章写完后，检查以下几点：

- [ ] **开头**：我是不是直接切入了正题？有没有废话？
- [ ] **中间**：主角有没有遇到阻力？还是太顺了？
- [ ] **高潮**：本章有没有一个“名场面”或记忆点？
- [ ] **结尾**：读者看到这里会想睡觉，还是想骂娘/想追更？
- [ ] **节奏**：全篇是不是都是一样的句长？有没有快慢变化？

## 四、常见场景模板

### 1. 打脸章结构
1.  **压抑**：反派嚣张，主角被误解/轻视，旁人嘲讽。
2.  **转机**：主角亮出底牌（实力/证据/身份）。
3.  **爆发**：反派震惊，旁人被打脸，主角反击。
4.  **余韵**：收获战利品/名声，引出更强的敌人。

### 2. 探索/解谜章结构
1.  **困惑**：发现不合理之处，气氛诡异。
2.  **探索**：小心翼翼地接触真相，危险逼近。
3.  **惊吓**：突然的惊吓点（Jump Scare），发现恐怖真相。
4.  **逃离/对抗**：带着真相逃命，或被迫战斗。

### 3. 感情/日常章结构
1.  **氛围**：温馨或暧昧的环境描写。
2.  **互动**：通过对话或小动作拉近关系。
3.  **波折**：外部干扰或内部误会打断气氛。
4.  **升华**：解决波折，关系更进一步（或彻底破裂）。
