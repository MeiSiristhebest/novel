# 小说大纲模板 (Novel Outline Template)

> **使用说明**：根据小说类型和长度选择合适的大纲模板，支持短篇(10章)到超长篇(1000章+)。

---

## 📋 基本信息

```yaml
项目信息:
  书名: 
  作者: 
  流派: [玄幻/仙侠/都市/言情/历史...]
  细分类型: [打脸流/凡人流/甜宠/虐恋...]
  
规模规划:
  预计总字数: [X]万字
  预计章节数: [X]章
  每章字数: 2000-4000字
  更新频率: [日更X章/周更X章]

核心卖点:
  一句话简介: [50字以内概括全书]
  核心爽点: [读者为什么追？]
  差异化: [和同类作品的区别？]
```

## 🎯 核心设定

### 控制思想 (Controlling Idea)

```
[价值A] 战胜了 [价值B]，因为 [原因]。

示例：
- 玄幻：坚持战胜了天赋，因为努力可以弥补一切差距
- 言情：爱情战胜了门第，因为真心能打动一切
- 宫斗：智慧战胜了权势，因为善于布局者终将胜出
```

### 核心冲突

```yaml
主角想要: [终极目标]
阻碍他的: [主要障碍]
如果失败: [最坏后果]
```

### 金手指/特殊设定

```yaml
金手指类型: [系统/功法/重生记忆/老爷爷...]
核心能力: 
能力限制: [冷却/代价/副作用]
升级路线: [如何变强？]
```

## 🎭 剧情结构（各类型适用）

### 三幕结构

```
第一幕：建置（1-25%）
├── 日常世界：展示主角的普通生活
├── 激励事件：打破平衡的事件
├── 拒绝召唤：主角的犹豫
└── 第一转折点：主角决定行动

第二幕A：对抗（25-50%）
├── 新世界规则：主角适应新环境
├── 尝试与失败：初次尝试，遭遇挫折
├── 盟友与敌人：建立关系网
└── 中点转折：重大发现或事件

第二幕B：深化（50-75%）
├── 敌人反击：反派加大力度
├── 一无所有：主角失去重要的东西
├── 灵魂黑夜：最低谷
└── 第二转折点：找到希望

第三幕：解决（75-100%）
├── 最终集结：为最后决战准备
├── 高潮对决：与反派的终极对决
├── 真相揭示：谜底揭开
└── 新的日常：故事收尾
```

### 网文节奏公式

```
【黄金三章】第1-3章
目标：留住读者
公式：压迫→金手指→释放→期待

【前期】第4-50章
目标：建立追读习惯
公式：小目标→完成→新目标→完成→更大目标

【中期】第51-200章
目标：扩大格局
公式：升级地图→新势力→新挑战→新突破

【后期】第201-结局
目标：完美收尾
公式：最终决战→伏笔回收→HE结局→番外
```

## 📎 快速模板

### 10章短篇模板

```
第1章：开局+激励事件
第2章：初步行动+第一次冲突
第3章：发现真相的一部分
第4章：遭遇挫折
第5章：获得帮助/线索（中点）
第6章：再次行动
第7章：重大发现
第8章：一无所有（低谷）
第9章：最终决战
第10章：解决+收尾
```

### 100章标准模板

```
第1-3章：黄金三章（吸引读者）
第4-10章：建立世界观+目标
第11-30章：第一个大目标+第一个反派
第31-50章：升级+扩展人际关系
第51-70章：更大的世界+更强的敌人
第71-90章：高潮前的积累+低谷
第91-100章：最终决战+大结局
```

### 500章长篇模板

```
第一卷(1-100)：起步篇 - 从无名小卒到初露锋芒
第二卷(101-200)：成长篇 - 站稳脚跟，建立势力
第三卷(201-300)：争霸篇 - 参与更大的博弈
第四卷(301-400)：巅峰篇 - 成为顶尖存在
第五卷(401-500)：终局篇 - 最终决战+完美收官
```
