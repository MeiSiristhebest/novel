# 世界观圣经模板 (World Bible Template)

> **用途**：建立小说的世界设定，确保创作过程中的一致性  
> **适用**：所有需要世界观设定的流派（玄幻/仙侠/科幻/都市异能/历史等）

---

## 📋 基础信息

```yaml
项目信息:
  小说名称: 
  流派类型: 
  世界类型: [架空/现实/半架空]
  时代背景: [古代/现代/未来/混合]
  
世界观一句话:
  "[用一句话概括这个世界最独特的地方]"
  例："一个灵气复苏后，普通人与异能者共存的现代都市"
```

## ⚡ 力量体系 (Power System)

### 力量来源与获取

```yaml
力量名称: [灵气/斗气/异能/魔力/内力...]

来源设定:
  本质: [天地灵气/基因变异/神秘物质/科技改造]
  获取方式: 
    - 天生觉醒
    - 后天修炼
    - 意外获得
    - 传承继承

等级体系:
  # 必须有清晰的等级划分
  等级1: [名称] - [描述] - [能做到什么]
  等级2: [名称] - [描述] - [能做到什么]
  等级3: [名称] - [描述] - [能做到什么]
  ...
  巅峰: [名称] - [描述] - [能做到什么]
```

### 力量代价（核心！）

> **顶级设定必填**：没有代价的力量是无聊的。

```yaml
使用代价:
  低级代价: [体力消耗/短暂虚弱]
  中级代价: [寿命损耗/精神污染]
  高级代价: [人性丧失/身体异变/理智崩溃]

获取代价:
  天赋型: [天生异于常人，被当作怪物]
  修炼型: [需要大量时间和资源]
  传承型: [前任必须死亡/付出代价]
  
失控风险:
  描述: [力量失控会发生什么？]
  触发条件: [什么情况下会失控？]
```

### 力量限制

```yaml
硬性限制:
  - [例：斗气无法攻击灵魂]
  - [例：异能在特定区域失效]
  - [例：需要特定媒介才能施法]

软性限制:
  - [例：使用后需要冷却]
  - [例：每天有使用次数上限]
  - [例：需要消耗特定资源]

克制关系:
  [力量A] 克制 [力量B]
  [力量B] 克制 [力量C]
  [力量C] 克制 [力量A]
```

## 📜 历史与秘密 (History & Secrets)

### 洋葱式真相设计

> 设计三层真相，层层反转，增加悬念。

```yaml
第一层（表象）:
  世人公认的历史: "[例：三千年前，圣人开天辟地，创造了这个世界]"
  主角最初相信的: "[和大众一样]"

第二层（反转）:
  只有高层知道的真相: "[例：所谓的圣人其实是入侵者，原住民被灭族]"
  主角发现的时机: "[中期，产生动摇]"

第三层（终极）:
  世界的底层逻辑: "[例：这个世界其实是某种存在的梦境，一切都不是真的]"
  主角领悟的时机: "[后期，决定命运]"
```

## 🎭 各流派世界观速查

### 玄幻/仙侠

```yaml
必备元素:
  - 境界体系（9个等级以上）
  - 宗门/家族势力
  - 天地灵气设定
  - 天材地宝资源体系
  - 法宝/功法分级

常见地图层级:
  凡界 → 灵界 → 仙界 → 神界
  或：小城 → 王国 → 帝国 → 大陆 → 星域 → 宇宙
```

### 都市异能

```yaml
必备元素:
  - 异能分类体系
  - 隐秘组织（官方+民间）
  - 异常生物分级
  - 普通人与异能者的关系
  - 保密机制

常见设定:
  - 灵气复苏型
  - 一直隐藏型
  - 末世变异型
```

### 历史穿越

```yaml
必备元素:
  - 准确的历史背景
  - 当时的社会制度
  - 经济水平锚定
  - 语言/礼仪差异
  - 蝴蝶效应考量

注意事项:
  - 食物/器物的时代性
  - 称谓的准确性
  - 历史人物的形象
```

## ✅ 世界观自检清单

- [ ] 力量体系有清晰的等级划分
- [ ] 力量有代价和限制
- [ ] 地图有层级，适合长篇升级
- [ ] 势力之间有矛盾和利益冲突
- [ ] 历史有三层真相设计
- [ ] 经济体系有锚定，不会崩坏
- [ ] 特殊规则能产生剧情冲突
- [ ] 没有逻辑漏洞
