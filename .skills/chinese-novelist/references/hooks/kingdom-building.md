# Kingdom Building - 种田/基建指南

> **用途**: 描写从零开始建设势力、国家或领地的过程，提供养成快感。
> **适用流派**: 种田文、领主文、历史穿越、末世重建
> **关键特征**: 视觉化成长、科技攀升、民心所向、以弱胜强

---

## 一、核心循环 (Core Loop)

**获取资源 → 投入建设 → 产出更多资源 → 扩大规模**

1.  **开局 (The Start)**: 一穷二白，破屋漏雨，刁民想反。
2.  **破局 (Breakthrough)**: 引入第一项黑科技（水泥/肥皂/土豆）或新制度。
3.  **滚雪球 (Snowball)**: 粮食增产，人口流入，组建军队。
4.  **挑战 (Conflict)**: 邻居眼红，蛮族入侵，天灾降临。
5.  **碾压 (Domination)**: 降维打击，文明同化。

## 二、爽点设计

### 2.1 视觉化成长
*   **地图**: 从一个小村庄变成巨城，地图边界不断外扩。
*   **建筑**: 茅草屋 → 砖瓦房 → 高楼大厦/魔法塔；泥泞小路 → 水泥大道。
*   **生活**: 吃糠咽菜 → 白面白米 → 满汉全席；衣不蔽体 → 丝绸锦缎。

### 2.2 科技树攀升 (Tech Tree)
*   **农业**: 曲辕犁、杂交水稻、化肥 → *解决温饱，稳定民心*。
*   **工业**: 水泥、高炉炼钢、蒸汽机 → *基建狂魔，战争潜力*。
*   **军事**: 连弩、火药、后装枪、火炮 → *排队枪毙，降维打击*。
*   **民生**: 青霉素、玻璃、造纸术、印刷术 → *收割财富，文化输出*。

### 2.3 人口与人才
*   **流民**: 是负担也是劳动力。
*   **集邮模式**: 搜集历史名将/谋士/工匠，点亮图鉴。
*   **教育**: 建立学校，培养忠诚的下一代。

## 三、写作误区

### 3.1 流水账
*   ❌ 既然是种田，就每天写记账：今天收了多少斤米，明后天造了什么墙。
*   ✅ **事件驱动**: 通过具体的**危机**来推动建设。
    *   *例*: 冬天快来了（危机）→ 必须造火炕/挖煤（建设）→ 成功过冬，还要卖煤给邻居赚一笔（爽点）。

### 3.2 忽视人性
*   老百姓不是NPC，给口饭吃就死忠。
*   要写出**思想转变**: 从麻木不仁到眼中有光，从怀疑主角到誓死追随。

### 3.3 科技太快
*   开局一个月造出原子弹，崩了。
*   要有**前置条件**: 想要造枪？先搞定炼钢。想要炼钢？先找矿。

## 四、模板：基建三板斧

1.  **要想富，先修路**: 改善物流，控制力延伸。
2.  **广积粮，高筑墙**: 提升防御，在此期间猥琐发育。
3.  **树立典型**: 提拔一个听话的土著，给予重奖，千金买马骨。
