# Folk Customs - 民俗禁忌系统构建

> **用途**: 在中式恐怖、志怪悬疑、乡土惊悚中，把“地方规矩/祖训/庙规”做成可执行的叙事引擎：规则可复述、破戒有动机、代价可追踪。
> **适用流派**: 民俗志怪、规则怪谈、乡镇刑侦暗线、克苏鲁本土化
> **关键特征**: 祖训、禁忌、节令、祭祀、因果、现实人情网

---

## 一、核心特征分析

### 1.1 定义

“民俗禁忌系统”不是一句“村里有规矩”，而是一套**可运行的规则体系**：
- **规则**：一句话能背下来（读者记得住）。
- **来源**：能追溯到人、物、案（读者信得过）。
- **执行**：有人或结构在执行（读者怕得起来）。
- **结算**：后果分期到账且可回溯（读者上瘾）。

### 1.2 核心价值

- **把恐怖变成博弈**：主角不是“被吓”，而是“在破戒与代价之间算账”。
- **把氛围变成机制**：地方秩序本身就是压力源（人情、利益、舆论先动刀）。
- **把线索变成闭环**：禁忌=线索目录；每条规则都能指向一个真相碎片。
- **把爽点变成推进**：破解禁忌不是科普，而是夺回叙事控制权（活下来/翻盘/破局）。

### 1.3 类型划分

**按规则形态**:
1. **祖训型**：族谱批注、家规、祠堂碑文（“不得”“不可”“切记”）。
2. **庙规型**：香火规矩、禁地禁时、供品规格（“缺一不可”）。
3. **行会型**：抬棺、捞尸、送殡队口传（“规矩不教外人”）。
4. **地势型**：桥、井、坟、河、山口的“地理规则”（“那条路别走”）。

**按代价形态**:
1. **现实代价**：被排挤、断供、封口、勒索、替罪、被当“灾星”。
2. **感官代价**：味道、湿冷、噪声、皮肤异样（低烈度但高持续）。
3. **认知代价**：记忆缺口、时间错位、把“错的”当成“对的”。
4. **关系代价**：亲缘裂开、同伴互疑、证人反水。

### 1.4 常见误区

❌ **只有一句规矩，没有系统**：规则不成网，读者无法建立预期。

❌ **破戒靠降智**：主角为了推进剧情硬犯蠢，读者直接出戏。

❌ **后果只有超自然**：没有现实层惩罚，禁忌就像“作者开关”。

❌ **解释过度**：把恐怖写成讲义，读者不怕只觉得累。

## 三、Top作品解析

> 这里不追求“致敬情节”，只抽取可复用机制。

### 3.1 经典案例1: 《鬼吹灯》

**核心手法**: 民间传说 + 行当规矩 + 环境压迫

**分析**:
- **规矩先行**：很多禁忌先以“行当经验”出现，而非作者解释。
- **现实层压力**：团队生存/利益分配/信任关系让“破戒”变得合理。
- **可复用要素**: “规矩是为了活命”而非“规矩是为了神秘”。

**为什么有效**:
1. 规则来自“人类经验”，读者接受成本低。
2. 破戒往往由现实动机逼出来，张力稳定。

### 3.2 经典案例2: 规则怪谈类（机制抽象）

**核心手法**: 短规则 + 明确惩罚 + 不完全信息

**分析**:
- **规则可复述**：一两句就能在评论区被复述传播。
- **惩罚可追踪**：读者能回溯“错在第几条”。
- **可复用要素**: “规则是关卡”，读者会主动参与解题。

**为什么有效**:
1. 规则化降低了写作难度，提升读者参与感。
2. 违禁结算的确定性，反而增强恐惧的确定性。

### 3.3 经典案例3: 《神秘复苏》（机制化恐怖）

**核心手法**: 厉鬼规律 + 观察代价 + 生存博弈

**分析**:
- **规律不是背景**：规律=行动边界，人物每一步都在踩线。
- **观察有代价**：知道更多并不安全，反而更危险。
- **可复用要素**: “破解=暂活；掌握=更深绑缚”。

**为什么有效**:
1. 恐怖从“长相”转为“规律”，更耐写。
2. 规则让每次推进都可量化，读者更容易上瘾。

## 四、实战技法

### 4.1 基础公式

**禁忌条目公式**:
```text
规则（可复述） + 来源（可追溯） + 破戒动机（合理） + 征兆（可验证） + 代价（分期） = 可持续张力
```

**两层结算公式（推荐）**:
```text
现实层结算（人情/利益/舆论） -> 低烈度异象（味/湿/噪/错位） -> 高烈度反噬（失控/死亡/交换）
```

### 4.2 进阶技巧

#### 1. 规则“可验证”

让读者在 1-2 次事件内看到：规则不是随口胡扯。
- 例：守夜不回头，回头的人并非立刻死，而是“第二天开始被当作丧主”。

#### 2. 破戒“不可避免”

破戒不是想作死，而是“现实逼你选”。
- 例：救人、查案、取证、换命、还债。

#### 3. 规则“互相矛盾”

制造解题空间：两条规矩在当前情境下无法同时满足。
- 例：必须关门 vs 必须回应呼救。

#### 4. “欠账式反噬”

违规不是立刻爆炸，而是形成债务：
- 利息：异象加重。
- 催收：人情网、仪式组织者、地头势力。
- 结算：以物易物、以命换命、以秘密换路。

### 4.3 常见套路

#### 套路1: 规矩的“空位”

规则要求“双数/成对/同席”，但现场缺一个。
- 读者爽点：知道“必有东西来补齐”。

#### 套路2: 替罪与转嫁

你破戒的代价可以转嫁，但要付更大代价。
- 读者爽点：主角的道德/利益抉择。

#### 套路3: 口传与改写

口传规矩被人故意改过一字。
- 读者爽点：推理与反转。

## 五、写作要点

### 节奏控制

- **立规**: 在安全场景里自然出现（饭桌、路边、案卷、族谱边角）。
- **诱犯**: 现实动机把主角推到必须犯的边缘。
- **结算**: 先现实、后异象、再反噬，分批到账。

### 情绪设计

- **压迫**: “不是鬼在盯你，是全村在盯你”。
- **不安**: 低烈度异常持续出现（潮、锈、腥、噪）。
- **疑惧**: 规矩互相打架，主角怎么选都像错。

### 陷阱规避

❌ **避免**: 只写“很诡异”“很恐怖”这类判断词。
✅ **改为**: 让异常落在可感知细节上（味、湿、痒、错位、迟一拍）。

❌ **避免**: 一次性解释真相。
✅ **改为**: 解释只解“这一条规矩”，把更大真相延后。

## 六、检查清单

写完后自检：

- [ ] 规则是否可被一句话复述？
- [ ] 规则来源是否可追溯（谁说/哪里写/哪次案）？
- [ ] 破戒是否由现实动机逼迫，而非主角降智？
- [ ] 是否出现至少 1 个“可验证征兆”（非纯主观感受）？
- [ ] 后果是否分期到账（现实层 -> 异象 -> 反噬）？
- [ ] 是否存在至少一处“规则矛盾/两难选择”？
- [ ] 结尾是否留下可追查线索（案卷、族谱、证人、物证）？
