# Transportation - 交通方式

> **用途**: 解决地图移动问题，展示贫富差距，提供特殊的战斗/暧昧场景（车厢/飞舟）。
> **适用流派**: 仙侠、科幻、奇幻、大航海
> **关键特征**: 速度分级、载具多样性、移动成本、途中遇袭

---

## 一、核心特征分析

### 1.1 定义
地图大了，靠腿跑是不现实的。
交通方式（Transportation）直接决定了信息传播的速度和势力控制的范围。
**核心公式**: 距离 / 速度 = 剧情空白期。如果在路上要花三个月，这三个月必须发生点什么（或者一笔带过）。

### 1.2 类型划分
1.  **生物坐骑**: 汗血宝马、狮鹫、飞龙、仙鹤。
2.  **法宝载具**: 飞剑（个人）、飞舟（团队）、移动城市（要塞）。
3.  **传送阵/星门**: 瞬间移动。通常贵且有战略意义。
4.  **肉身横渡**: 强者不需要载具，撕裂虚空。

### 1.3 常见误区
*   **速度崩坏**: 前期说两个城隔着十万里，主角骑马一天就到了。马是火箭做的？
*   **载具无用论**: 有了传送阵，大家还坐飞舟干什么？（解释：传送阵太贵/不能直达/需要排队）。
*   **缺乏细节**: 坐飞舟就像坐飞机，只是"上船-过场动画-下船"，没有描写船上的生活。

## 二、Top作品解析

### 2.1 经典案例1: 《凡人修仙传》
**载具**: **神风舟 vs 传送阵**。
**细节**:
- 韩立前期全是靠飞，飞得很慢，还要躲避妖兽。
- 只有大城市才有传送阵，而且传送一次要几十块灵石（巨款）。
- **乱星海**地图：因为海太大，必须要有船，而且要猎杀妖兽作为动力材料。

### 2.2 经典案例2: 《吞噬星空》
**载具**: **战机 -> 飞船 -> 瞬移**。
**细节**:
- 从智能战机到E级金属飞船，载具的升级代表了防御力的提升。
- 飞船内部就是个移动豪宅，主角在里面修炼、生活。

### 2.3 经典案例3: 《雪中悍刀行》
**载具**: **马车 / 劣马**。
**细节**:
- 徐凤年游历江湖，坐的是破马车，带的是老黄马。
- 一路上的风餐露宿，反衬出世子身份的反差，以及江湖的真实质感。

## 二点五、场景示例

### 【飞行体验】初次御剑

```
林枫站在飞剑上，双腿发抖。

三尺长的寒铁剑悬浮在百米高空，下面是云海，看不到地面。

"不要往下看。"师姐在前面稳稳站着，回头一笑。

"我...我不敢动。"

"第一次御剑都这样。把灵力灌注到脚下，剑会感应你的意志。"

林枫深吸一口气，按照师姐教的方法，将真气缓缓渗入飞剑。

剑身微微一颤，像是活了过来。

他试着想象自己往前飞——

嗖！

飞剑猛地向前冲刺，速度快得他几乎被甩下去！

"啊啊啊啊！怎么停？！"

风在耳边呼啸，云层被撕裂，他像一颗炮弹划破天空。

师姐在后面笑得花枝乱颤："意念放松！放松就会减速！"

林枫强行让自己冷静，脑海中想象"慢下来、慢下来"——

飞剑果然开始减速，最后平稳地悬停在空中。

他大口喘气，后背全是冷汗。

但同时，他的嘴角不由自主地上扬。

这就是飞行啊...这就是修仙啊！

从今天起，天空不再是禁区。
```

**分析**:
- **新手视角**: 通过第一次御剑的恐惧和兴奋，让读者感同身受
- **操作细节**: "灵力灌注"、"意念控制"给出了清晰的飞行逻辑
- **情绪递进**: 恐惧→失控→掌握→自由，完整的成长弧线
- **可复用要素**: "第一次XX"的经典写法（第一次飞行/第一次杀人/第一次恋爱）

### 【载具劫持】空中危机

```
"警告！飞舟遭受攻击！护盾强度下降至30%！"

船舱内的灵石灯开始闪烁，警报声刺耳。

林枫冲到甲板上，看到了恐怖的一幕——

三艘黑色的魔修飞舟正在包围他们，船身上绘着骷髅图案。

"血骷髅海盗团！"船长脸色煞白。

砰！

一道黑色的魔雷击中护盾，整艘船剧烈摇晃。

"所有人准备弃船！"

"弃船？"林枫抓住栏杆，"这里是万米高空！跳下去必死无疑！"

"被俘虏，死得更惨。"

船长咬牙切齿，丢出三张破空符。

"你们先走，我留下来拖延时间——"

林枫看向那三艘黑船，突然眼睛一亮。

他认出了中间那艘船尾的防御薄弱点。

"船长，给我三息时间。"

"你疯了吗？！"

林枫没有回答，直接跃下飞舟——

在所有人惊恐的目光中，他御剑冲向了最大的那艘黑船！

"找死！"魔修们狞笑着，数十道攻击朝他轰来。

但林枫的速度快得像闪电，在攻击间隙穿梭，贴着黑船船体俯冲——

轰！

他一掌拍碎了船尾的动力核心！

黑船失去平衡，开始坠落。

剩下两艘黑船慌乱撤退，他们不敢再追了。

林枫飞回甲板，落地时双腿一软，差点摔倒。

船长和乘客们看他的眼神，充满了敬畏。

"小兄弟...报上名来。"

"不用。"林枫摆摆手，走回舱室，"只是顺路帮个忙。"

但他的心脏还在狂跳。

刚才那一瞬间，他离死亡只有一线之隔。
```

**分析**:
- **危机设定**: 万米高空+海盗围攻，双重绝境
- **战术细节**: 找到弱点(动力核心)而非硬拼
- **时间压力**: "三息时间"制造紧迫感
- **可复用要素**: "劫持反劫持"的经典桥段，常用于交通工具上的战斗

## 三、实战技法

### 3.1 基础公式 (The Journey)
1.  **分级**: 穷人走路，富人坐兽车，仙人御剑，大佬撕裂空间。
2.  **补给**: 长途旅行需要补给点（驿站/港口）。这里是触发支线任务的地方。
3.  **危险**: 路上会有强盗、空难（虚空风暴）、兽潮。
4.  **空间**: 载具内部是一个封闭空间，适合发生密室杀人、暧昧互动、卧底窃听。

### 3.2 进阶技巧
*   **活体载具**: 你的飞船实际上是一只巨大的鲲，它还需要喂食，甚至有情绪。
*   **公共交通**: 仙界的地铁/公交车（如：云鲸巴士）。体现世界的繁荣。
*   **特殊限制**: 禁空领域。只能步行通过，强行把节奏慢下来，制造危机。

### 3.3 常见套路
*   **劫机/劫船**: 喜闻乐见的送经验环节。
*   **同乘**: 男女主共乘一骑/一把剑，身体接触（发糖）。

## 四、写作要点

### 节奏控制
*   **略写**: "三日后，抵达XX城"。无事发生就跳过。
*   **详写**: 第一次乘坐高级载具时的震撼（俯瞰大地、穿梭云海）。

### 情绪设计
*   **优越感**: 主角坐着豪华飞辇从头顶飞过，地面上的路人如果不小心挡路就会被撞死。
*   **孤独感**: 一个人在漫长的星空中航行。

### 陷阱规避
*   **注意时间线**: 如果主角去救人，一定要算好路程时间。不要瞬移过去（除非有挂）。
*   **载具防御**: 飞舟通常有护盾，能抗几下攻击，给主角反应时间。

## 五、检查清单
写完交通设定后自检：
- [ ] 交通工具的速度是否符合地图尺度？
- [ ] 不同的交通方式是否有明显的价格/身份差异？
- [ ] 旅途中是否安排了合理的冲突或互动？
- [ ] 载具是否有独特的视觉外观？
