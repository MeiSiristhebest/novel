# Culture & Customs - 文化习俗

> **用途**: 增加世界的真实感（沉浸感），提供差异化的阅读体验，为剧情提供背景。
> **适用流派**: 历史、奇幻、异界大陆、修仙日常
> **关键特征**: 节日、禁忌、饮食、礼仪

---

## 一、核心特征分析

### 1.1 定义
文化（Culture）是世界观的血肉。
如果世界观只写了打打杀杀，那就只是一个战场；加上了风土人情，才是一个世界。
**核心公式**: 独特的环境/历史 -> 独特的生活方式 -> 独特的习俗。

### 1.2 类型划分
1.  **宗教/信仰习俗**: 祭祀大典、狂欢节、忏悔日。
2.  **社交/礼仪习俗**: 见面磕头还是握手？尊称是什么？（"道友请留步"）。
3.  **生存/生活习俗**: 有什么不能吃的？有什么不能说的？（避讳）。
4.  **种族特有习俗**: 精灵族爱洁癖，矮人族爱喝酒，兽人族以牙还牙。

### 1.3 常见误区
*   **直接照搬中国古代**: 异世界也过春节、吃饺子、贴对联，完全没有异域感（除非是架空历史）。
*   **说明文**: "在这个国家，人们习惯于..."，用大段文字介绍，而不是通过角色的行为表现出来。
*   **无意义的怪癖**: 为了设定而设定（比如规定走路必须先迈左脚），对剧情和人物塑造没有帮助。

## 二、Top作品解析

### 2.1 经典案例1: 《诡秘之主》
**核心展示**: **鲁恩王国的绅士文化 vs 拜朗帝国的死神崇拜**。
**细节**:
- 鲁恩人喝下午茶，讲究体面，报纸文化发达。
- 这种维多利亚时代的风貌，为"克苏鲁+蒸汽朋克"提供了完美的温床。

### 2.2 经典案例2: 《鬼吹灯》
**核心展示**: **倒斗行规与民间禁忌**。
**细节**:
- "鸡鸣灯灭不摸金"、"黑驴蹄子防尸变"。
- 这些虚构的"行业习俗"，听起来极具说服力，构建了整个盗墓流的基石。

### 2.3 经典案例3: 《将夜》
**核心展示**: **唐人的骄傲与草原的野性**。
**细节**:
- 唐人尚武，以胖为美（致敬大唐），无论男女都佩剑。
- 这种文化氛围直接决定了唐国的国运和角色的性格。

## 二点五、场景示例

### 【文化冲突】外来者的窘境

```
林枫走进客栈，准备点菜。

"掌柜，来一份烤肉。"

掌柜脸色一变，周围的客人也纷纷侧目。

"客官...您不是本地人吧？"

"是啊，我从中原来的。怎么了？"

掌柜压低声音："今天是'守夜日'。在这一天，全城禁荤，只能吃素。"

"为什么？"

"一千年前的今天，兽神降下神罚，屠了三座城。"

"从那以后，每年这一天，草原上的人都要斋戒三日，以示忏悔。"

林枫愣住了："那...我可以吃吗？我又不是草原人。"

掌柜苦笑："您当然可以吃。但吃完之后，恐怕走不出这条街。"

他指了指窗外。

几个彪形大汉正盯着客栈门口，眼神不善。

"他们是护法骑士，专门巡查是否有人破戒。"

"外来者也要遵守？"

"入乡随俗。"掌柜叹气，"草原人的规矩，没人敢破。"

林枫沉默了三秒，收起了点菜的念头。

"那...给我来碗青菜汤吧。"

掌柜松了口气："客官明智。"

林枫坐在角落，看着碗里的清汤寡水，苦笑。

这就是异乡啊。

你的力量再强，也要低头。
```

**分析**:
- **知识盲区**: 主角因不了解当地习俗而犯错
- **历史背景**: 用"兽神神罚"解释习俗来源
- **强制性**: 通过"护法骑士"展示习俗的约束力
- **妥协**: 主角选择入乡随俗，体现智慧
- **可复用要素**: "文化冲突"的标准叙事（无知→违规→警告→服从）

### 【成人礼】仪式感与命运转折

```
今天，是林家的成人礼。

十六岁的少年少女们，身穿白色长袍，赤脚走过三里长的火炭路。

"只有走完这条路，才能算真正的林家人。"

族长站在终点，手持祖传的青铜镜。

林枫咬着牙，每一步都踩在炙热的炭火上，皮肉烧焦的味道刺鼻。

但他不能喊疼。

喊出声的人，会被视为懦夫，永远无法获得家族资源。

终于，他走到了终点。

双脚血肉模糊，但他依然挺直了腰板。

"林枫，抬起头。"

族长举起青铜镜，镜面对准了他的眉心。

一道光芒射出——

镜子里显现出一团微弱的红光。

"火灵根...下品。"族长的声音没有波澜。

周围传来窃窃私语。

"又是个废物。"

"林家这一代，没一个能用的。"

林枫握紧拳头，指甲刺入掌心。

他知道，这个结果意味着什么。

没有好的灵根，就拿不到家族的丹药和功法。

他的修炼之路，注定比别人难一百倍。

但仪式还没结束。

族长将一枚黑色的令牌递给他："从今天起，你是林家外门弟子。"

"谨记家训：弱者，无资格抱怨。"

林枫接过令牌，深深鞠躬。

他没有说话。

但心中已经立下了誓言——

总有一天，他会让所有人刮目相看。

成人礼，不是终点。

是战场的起点。
```

**分析**:
- **仪式过程**: 火炭路→鉴定灵根→授予令牌，层次分明
- **身体苦痛**: 通过"火炭"制造仪式的残酷性
- **命运转折**: 鉴定结果直接决定未来地位
- **情绪压抑**: 被嘲讽但不能反驳，积累爆发点
- **可复用要素**: "成人礼=资质鉴定"的经典设定

## 三、实战技法

### 3.1 基础公式 (The Culture Filter)
1.  **节日**: 设计一个盛大的节日（花灯节/狩猎祭），作为剧情发生的时间点（通常会有暗杀或邂逅）。
2.  **禁忌**: 什么是绝对不能做的？（如：不能直视神的雕像，晚上不能照镜子）。
3.  **口头禅/俚语**: 角色无论高兴还是生气，都会说的一句话（"赞美愚者"、"无量天尊"）。
4.  **服饰**: 有什么特殊的穿着规定？（颜色等级、图腾纹身）。

### 3.2 进阶技巧
*   **仪式感**: 把简单的行为仪式化。比如杀人前要先擦剑，喝茶前要先祷告。
*   **文化冲突**: 主角刚到一个新地图，因为不懂当地规矩而闹笑话或惹麻烦（冲突点）。
*   **历史回响**: 现在的习俗是为了纪念几千年前的某个大事件（如：吃粽子是为了纪念屈原）。给习俗编一个神话故事。

### 3.3 常见套路
*   **相亲大会/比武招亲**: 最常见的推动感情线的习俗。
*   **成人礼**: 主角成年的标志，通常也是金手指觉醒或者家族矛盾爆发的时刻。

## 四、写作要点

### 节奏控制
*   **点缀**: 不要专门写，要夹杂在剧情里。比如主角去酒馆打听消息时，描写酒馆里的吟游诗人和特色饮料。
*   **集中展示**: 在重要的节日（庆典）章节，可以浓墨重彩地描写氛围。

### 情绪设计
*   **新奇感**: "居然还有这种操作？"（如：这个种族通过碰触触角来交流感情）。
*   **厚重感**: 越古老、越繁琐的仪式，越显得这个势力底蕴深厚。

### 陷阱规避
*   **避免太现代**: 除非是穿越文搞发明，否则不要让土著满嘴网络流行语。
*   **避免冒犯**: 所有的习俗都要有逻辑支撑（比如缺水的地方肯定不流行天天洗澡）。

---

## 五、检查清单
写完文化设定后自检：
- [ ] 是否有至少一个独特的节日或仪式贯穿剧情？
- [ ] 读者能否通过角色的言行判断出他来自哪个地区/种族？
- [ ] 习俗是否为剧情提供了便利或阻碍（如禁忌）？
- [ ] 是否有独特的饮食或服饰描写？
