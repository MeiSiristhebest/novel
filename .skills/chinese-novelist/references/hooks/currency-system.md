# Currency System - 货币体系

> **用途**: 量化价值，控制通货膨胀，体现贫富差距，作为金手指的消耗品。
> **适用流派**: 凡人流、经营文、神豪文、游戏异界
> **关键特征**: 多级货币、兑换汇率、特殊货币（贡献点/寿命）、通货膨胀

---

## 一、核心特征分析

### 1.1 定义
货币（Currency）是修仙界的血液。
它不仅仅是买东西的钱，通常它本身也是**修炼资源**（如灵石、魔晶）。
**核心公式**: 一般等价物 + 自身价值（含能量） = 硬通货。

### 1.2 类型划分
1.  **凡俗货币**: 金银铜钱。在低级地图流通，对修士无用（视金钱如粪土）。
2.  **能量货币**: 下品/中品/上品/极品灵石。既能花又能吸。
3.  **信用货币**: 宗门贡献点、联邦信用点、灵石票。由大势力背书，出了圈可能就不认了。
4.  **特殊货币**: 寿元（《求魔》）、功德、灵魂金币。

### 1.3 常见误区
*   **汇率锁死**: 永远是1:100。实际上汇率会波动，黑市兑换更贵。
*   **面额太大**: 吃顿饭要几万灵石，读者对数字脱敏了。要注意购买力设定（一颗灵石能买多少大米？）。
*   **货币无用**: 主角身上几百亿，结果后期买东西全是"以物易物"，钱烂在手里。

## 二、Top作品解析

### 2.1 经典案例1: 《凡人修仙传》
**体系**: **灵石分级**。
**分析**:
- 极品灵石不仅是钱，还是驱动古传送阵和傀儡的电池，有价无市。
- 这种"货币=战略物资"的设定，让甚至元婴老怪都会为了几块灵石打生打死。

### 2.2 经典案例2: 《诡秘之主》
**体系**: **金镑 / 苏勒 / 便士**。
**分析**:
- 严格遵循1:20:12的维多利亚时代进位制。
- 只有金镑能买非凡材料。克莱恩前期为了几镑的经费斤斤计较，非常真实。

### 2.3 经典案例3: 《我欲封天》
**体系**: **灵石复制**。
**分析**:
- 孟浩的金手指是复制灵石。
- 如果不加以限制，经济系统会崩溃。作者的处理是让他用灵石去砸人（物理消耗）或者喂给铜镜，转化为战力。

## 二点五、场景示例

### 【货币兑换】体现价值差异

```
"灵石？我们这里不收灵石。"

药铺掌柜看了一眼林枫手中的下品灵石，摆了摆手。

"这里是凡人城镇，你得用银子。"

林枫愣住了："那...这块灵石能换多少银子？"

掌柜拿出天平，掂量了一下："下品灵石，成色一般，给你三十两。"

"三十两？"林枫瞪大眼睛，"在宗门里，一块下品灵石只能买一个馒头！"

"那是宗门。"掌柜淡定地说，"在凡间，三十两够你吃喝半年了。"

林枫突然意识到——

自己在宗门里是穷鬼，但在凡间，他是富豪。

这就是修仙世界与凡人世界的货币鸿沟。
```

**分析**:
- **汇率震撼**: 通过巨大的兑换比例制造反差
- **世界观分层**: 修仙界vs凡间的经济体系完全不同
- **身份颠覆**: 宗门穷鬼在凡间是首富
- **可复用要素**: "跨世界货币套利"的经典桥段

### 【通货膨胀】经济崩坏

```
【系统公告】
由于大量玩家刷金币BUG，游戏经济体系崩溃。
当前金币与现实货币兑换比例暂时关闭。

"什么意思？"张伟看着公告，一脸茫然。

旁边的老玩家叹了口气："意思是，金币不值钱了。"

"昨天一把白银剑还卖1000金币，今天就变成10万金币了。"

"为什么？"

"因为有人用外挂刷了一万亿金币，扔进了拍卖行。"

张伟打开背包，看着自己辛辛苦苦攒的5000金币。

这些钱，昨天还能买五把好剑。

今天，连一瓶药水都买不起。

"这就是通货膨胀吗..."他苦笑。

在游戏里，三天就体验了现实中需要几十年才能看到的经济崩溃。
```

**分析**:
- **现实映射**: 通过游戏讲经济学概念
- **数字对比**: 1000→10万的暴涨让问题可视化
- **个人影响**: 展示宏观经济对个体的冲击
- **可复用要素**: "经济系统崩坏"的危机叙事

## 三、实战技法

### 3.1 基础公式 (The Money Rules)
1.  **购买力**: 设定一个基准。列出"馒头指南"（1文钱）、"宝剑指南"（100两）、"丹药指南"（10灵石）。
2.  **兑换**: 大额交易必须用高级货币。如果拿出几亿枚铜钱去买神器，会被人笑死（也不方便携带）。
3.  **造假**: 有没有假币？（灵石灵气流失变成了废石）。如何鉴别？
4.  **携带**: 储物袋的空间是有限的。这也是限制主角携带过多低级货币的原因。

### 3.2 进阶技巧
*   **私铸货币**: 主角掌握了灵石矿或者铸币权，直接挑战官方经济体系。
*   **特殊汇率**: 在魔界，灵石不值钱，魔晶值钱；回到修仙界反之。主角做倒爷赚差价。
*   **通货膨胀**: 随着世界等级提升，灵石如果不纯，就被淘汰了，这就逼着主角去赚"仙晶"。

### 3.3 常见套路
*   **撒币**: 战斗时燃烧灵石来提升阵法/法宝威力（氪金战士）。
*   **欠债**: 主角欠了系统/宗门一屁股债（如：为了救命预支），被迫疯狂打工。

## 四、写作要点

### 节奏控制
*   **缺钱**: 永远让主角处于"还差一点点"的状态，驱动他去冒险。
*   **暴富**: 用来解决之前的积压需求（换装备、升级家园）。

### 情绪设计
*   **肉痛**: "这一击打出去的不是法术，是五百灵石啊！"
*   **豪横**: 以前买不起的东西，现在"给我包圆了"。

### 陷阱规避
*   **避免复杂的数学题**: 虽然要有经济系统，但不要让读者算账。大概齐就行。
*   **避免货币种类切换太快**: 刚适应了灵石，马上换成仙元，读者会混乱。最好有过渡期。

## 五、检查清单
写完货币设定后自检：
- [ ] 货币之间是否有明确的兑换比例？
- [ ] 高级货币是否有除了购买力之外的特殊用途（能吃/能用）？
- [ ] 读者是否对"1万灵石是多大一笔钱"有清晰的概念？
- [ ] 经济系统是否支持主角的挥霍？
