# Fashion & Clothing - 服饰与时尚

> **用途**: 塑造人物形象（白衣胜雪），体现身份地位（门派校服），展示防御装备（法衣），提供互动道具（定情信物）。
> **适用流派**: 古言、仙侠、玄幻、宫廷
> **关键特征**: 材质特效、颜色等级、功能性、文化隐喻

---

## 一、核心特征分析

### 1.1 定义
人靠衣装。在玄幻世界，衣服不仅是遮羞布，更是**第二层皮肤**（防御层）。
服饰（Fashion）直接反映了人物的性格、所属势力以及当前的经济状况。
**核心公式**: 华丽外观 + 防御属性 + 身份标识 = 玄幻服饰。

### 1.2 类型划分
1.  **制服/校服**: 宗门弟子服、军队铠甲。强调集体归属感和等级（外门灰衣，内门白衣，亲传紫衣）。
2.  **法衣/战甲**: 镶嵌了阵法，能防尘、避火、自动修复。
3.  **常服/便装**: 主角微服私访时穿的，通常低调奢华。
4.  **特殊装束**: 夜行衣、伪装斗篷、舞衣（魅惑用）。

### 1.3 常见误区
*   **永远白衣**: 男主白衣胜雪，女主白衣飘飘。虽然仙，但看多了脸盲。
*   **现代感太强**: 出现"牛仔裤"、"超短裙"（除非是穿越文的特意描写）。
*   **忽视破损**: 打了一天架，衣服居然没破？（战损是展现战斗激烈程度的重要指标，也是福利环节）。

## 二、Top作品解析

### 2.1 经典案例1: 《诛仙》
**人物**: **碧瑶（绿衣） vs 陆雪琪（白衣）**。
**分析**:
- 颜色直接对应性格。绿色代表灵动、生机（也暗示悲剧）；白色代表高冷、出尘。
- 读者一看到颜色就知道是谁来了。

### 2.2 经典案例2: 《遮天》/《完美世界》
**装备**: **战衣与甲胄**。
**分析**:
- 到了高层次战斗，衣服就变成了甲胄。天帝战衣、雷帝甲胄。
- 描写重点在于材质（龙鳞、凰羽）和流光溢彩的特效。

### 2.3 经典案例3: 《将夜》
**校服**: **书院后山的旧棉袄**。
**分析**:
- 夫子和大师兄都穿得很朴素，甚至土气。
- 这种"土"与他们实力的"强"形成反差，体现了返璞归真的境界。

## 二点五、场景示例

### 【服饰升级】身份转变的视觉化

```
三个月前，林悦还穿着洗得发白的校服，背着补了又补的书包。

今天，她站在镜子前，几乎认不出自己。

量身定制的藏青色长裙，腰间是真丝暗纹腰带，领口绣着低调的云纹——这是只有贵族小姐才能穿的"云锦裙"。

不是炫耀的华丽，而是恰到好处的精致。

"这就是有钱人的衣服啊..."她轻声喃喃。

管家微笑道："小姐，您现在就是有钱人。"

林悦走进宴会厅时，所有的窃窃私语都停止了。

那些曾经嘲笑她是"乡下来的土包子"的名媛们，眼中闪过嫉妒的光。

男主角停下了手中的酒杯，目光追随着她的身影。

她的改变，不仅仅是一套衣服。
```

**分析**:
- **对比手法**: 三个月前的寒酸 vs 现在的优雅
- **细节描写**: "云纹"、"真丝暗纹"比直接说"很贵"更有质感
- **环境反应**: 通过他人的嫉妒和注视侧面烘托
- **可复用要素**: "服饰是身份转变的第一视觉信号"

### 【战袍描写】装备即战力的具象化

```
"这是...传说中的'龙鳞甲'？"

李寒从储物戒中取出那副漆黑的铠甲，每一片鳞片都闪烁着幽光。

【龙鳞战甲】
- 防御：+500
- 火抗：+80%
- 附加被动：龙威（震慑低阶妖兽）

他穿上铠甲的瞬间，整个人的气质都变了。

不再是那个畏畏缩缩的外门弟子，而是一尊黑色的战神。

"就凭你，也配穿龙鳞甲？"赵云冷笑。

李寒没有回答，只是抬起手。

赵云的剑斩在他胸口——铠甲纹丝不动，剑刃反而崩出了一个缺口。

"现在呢？"李寒的声音低沉而冷漠。

赵云脸色煞白，倒退三步："这不可能..."

装备，有时候就是实力的一部分。
```

**分析**:
- **数据化**: 游戏化的装备面板让防御力可视化
- **前后对比**: 穿上前后的气质变化
- **实战验证**: 用战斗证明装备的强度
- **可复用要素**: "换装=换战力"的系统流经典写法

## 三、实战技法

### 3.1 基础公式 (The Outfit)
1.  **材质**: 冰蚕丝、万年雪绒、蛟龙皮。听起来就很贵。
2.  **颜色**: 对应五行（火红、水蓝）或地位（明黄、紫气）。
3.  **配饰**: 玉佩（身份证明）、发簪（暗器）、腰带（储物）。
4.  **特效**: 走路带风、自带光环、无风自动。

### 3.2 进阶技巧
*   **爆衣**: 战斗到极限时，法衣破碎。男主展示肌肉（硬汉），女主...（注意尺度，重点写狼狈凄美）。
*   **变装**: 主角有多个马甲，每个马甲对应一套特定的衣服和面具。
*   **赠衣**: 男主把外套披在女主身上（经典暧昧桥段）。

### 3.3 常见套路
*   **买衣服**: 带着妹子去逛街，被势利眼店员嘲讽，然后包下全场。
*   **缝制**: 女主亲手给男主缝衣服（纳鞋底），体现贤惠（慈母/贤妻手中线）。

## 四、写作要点

### 节奏控制
*   **出场**: 重要人物第一次出场，必须详细描写衣着，建立第一印象。
*   **换装**: 随着实力提升/地图切换，主角的造型也要升级。不要一套新手装穿到大结局。

### 情绪设计
*   **惊艳**: 女主盛装出席宴会，男主看呆了。
*   **嘲笑**: 主角穿得破破烂烂去赴宴，被门口保安拦住。

### 陷阱规避
*   **避免描写过剩**: 不要写成淘宝详情页。抓住一两个重点特征（颜色、材质）即可。
*   **避免不合时宜**: 刺客穿一身发光的金甲去暗杀？

写完服饰设定后自检：

- [ ] 衣服是否具有实用的防御/辅助功能？
