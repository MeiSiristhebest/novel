# Romance Scene Guide - 感情戏描写指南

> **用途**: 构建令人心动的浪漫场景，处理角色之间的化学反应与情感张力。
> **适用流派**: 现代言情、古代言情、耽美、都市后宫
> **关键特征**: 暧昧推拉、肢体语言、心理博弈、氛围营造

---

## 一、核心要素

### 1.1 化学反应 (Chemistry)
只有两个人"来电"，才叫浪漫。
*   **反差**: 冰山 vs 太阳，傲娇 vs 直球，强强对抗。
*   **互补**: 一个懂另一个的梗，一个能接住另一个的话。
*   **张力 (Tension)**:
    *   **性张力**: 渴望触碰却又克制。
    *   **情报张力**: 互相试探底细（如《史密斯夫妇》）。

### 1.2 推进阶段 (Stages)
1.  **初见 (Meet Cute)**: 误会、相撞、英雄救美（或反过来）。
2.  **动心 (Crush)**: 发现对方未曾示人的一面（反差萌）。
3.  **试探 (Testing)**: 暧昧期，忽远忽近，患得患失。
4.  **确立 (Confession)**: 捅破窗户纸，高甜时刻。
5.  **磨合 (Conflict)**: 价值观冲突，或者外部阻力。

## 二、描写技巧

### 2.1 肢体语言 (Body Language)
*   **眼神**: 躲闪、凝视、余光。
    *   *例*: "他没有说话，只是目光沉沉地盯着她，仿佛要把她吸进眼底。"
*   **微动作**: 指尖颤抖、喉结滚动、耳根发红、不自觉地靠近。
    *   *例*: "他的手虚虚地搭在她的椅背上，是一个半拥抱的姿势。"

### 2.2 氛围营造 (Atmosphere)
*   **吊桥效应**: 在危险/紧张的环境下心跳加速（鬼屋、逃亡）。
*   **密闭空间**: 电梯、车厢、暴雨中的电话亭。
*   **感官放大**: 彼此的呼吸声、须后水的味道、体温。

## 三、实战示例

### 3.1 示例：暧昧试探
```markdown
顾寒洲微微俯身，将她困在双臂与墙壁之间。
那股清冽的冷香瞬间包裹了她。
“我想我们需要谈谈。”他的声音很低，带着一丝不易察觉的沙哑。
沈星晚不得不仰起头，试图气势上不输给他：“谈什么？谈顾总怎么只许州官放火？”
他却突然轻笑了一声，温热的气息拂过她的耳廓：“谈谈……你昨晚为什么喊我的名字。”
```

### 3.2 示例：无声关怀
```markdown
陆远只是默默地把那碗挑完了葱花的牛肉面换到了她面前。
他甚至头都没抬，依然在看手里的文件。
苏小小握着筷子的手顿了一下。
她不喜欢吃葱这件事，只说过一次。
那时他们还只是陌生人。
```

## 四、常见误区
1.  **工业糖精**: 强行撒糖，为了甜而甜，甚至不顾逻辑（霸总无脑宠）。
2.  **油腻霸总**: "女人，你成功引起了我的注意。"（现在已经不流行了，要尊重女性）。
3.  **莫名其妙**: 昨天还想杀对方，今天就爱得死去活来，缺乏情感转折。

## 五、检查清单
- [ ] 两人之间有没有**阻碍**（不能太容易在一起）？
- [ ] 是否有**心动瞬间**（Detailed Moment）？
- [ ] 互动是否符合人设？（高冷人设崩了吗？）
