# Resource Economy - 资源经济系统

> **用途**: 驱动主角行动（赚钱/抢资源），维持世界观的逻辑自洽，量化价值。
> **适用流派**: 凡人流、种田文、经营文、游戏异界
> **关键特征**: 稀缺性、流通性、通货膨胀控制、硬通货

---

## 一、核心特征分析

### 1.1 定义
钱（Resource）不是万能的，但没钱是万万不能的。
在网文中，资源不仅是货币，更是**修炼进度条**。
**核心公式**: 资源 = 实力。抢资源 = 抢命。

### 1.2 类型划分
1.  **一般等价物**: 金银、灵石、能量块。最基础的交易媒介。
2.  **消耗品**: 丹药、符箓、子弹。打一架就没了，所以要不断补充。
3.  **固定资产**: 洞府、灵田、地盘。能持续产出资源。
4.  **无形资产**: 功法、情报、人情、贡献点。

### 1.3 常见误区
*   **数字崩坏**: 动不动就几亿几千亿，读者对数字麻木了。（"我又赚了一个小目标"）。
*   **购买力混乱**: 馒头1文钱，住店100两，这物价体系完全是崩的。
*   **只进不出**: 主角光赚钱不花钱，或者光抢不消耗，不仅不爽，还让资源失去了意义。

## 二、Top作品解析

### 2.1 经典案例1: 《凡人修仙传》
**体系**: **灵石体系 + 以物易物**。
**分析**:
- 低阶修士用灵石，高阶修士用灵石买不到东西，只能以物易物。
- 无论什么境界，都需要辅助药物（筑基丹、结金丹）。
- 这种**刚需**导致了资源的极度稀缺和争夺的残酷性。

### 2.2 经典案例2: 《吞噬星空》
**体系**: **地球币 -> 宇宙币 -> 混元单位 -> 至宝点**。
**分析**:
- 随着地图升级，低级货币迅速贬值/被淘汰，换成更高级的货币。
- 每一级货币的购买力都重新锚定，避免了数字无限膨胀。
- **至宝点**: 后期直接用宝物作为计价单位，更符合顶尖强者的交易习惯。

### 2.3 经典案例3: 《亏成首富从游戏开始》
**体系**: **系统资金 vs 个人财产**。
**分析**:
- 反套路经济：主角需要把系统资金亏光才能转化为个人财产。
- 这种**逆向**的经济规则制造了大量的笑料和爽点（为了亏钱而拼命花钱，结果赚得更多）。

## 三、实战技法

### 3.1 基础公式 (The Economic Loop)
1.  **赚取**: 打怪掉落、做任务、杀人夺宝、副业（炼丹/锻造）。
2.  **消费**: 购买装备、丹药、情报、贿赂NPC。
3.  **缺口**: 无论赚多少，主角永远缺钱（因为升级需要的资源呈指数级增长）。
4.  **暴富**: 偶尔来一次横财（挖到古墓、赌石成功），爽感爆棚。

### 3.2 进阶技巧
*   **货币回收**: 系统/世界要有回收机制（如传送阵要收钱、进城要交税、渡劫要消耗法宝），防止通货膨胀。
*   **物价锚点**: 设定一个硬通货作为参照系（如：一颗最普通的练气丹 = 10灵石）。所有物价以此为基准浮动。
*   **拍卖会**: 集中展示高端资源和溢价交易的场所，是装逼打脸和炫富的最佳舞台。

### 3.3 常见套路
*   **捡漏**: 在地摊上用低价买到不知名的神器。
*   **垄断**: 主角掌握了独家技术（如完美丹药），垄断市场，甚至发行货币。

## 四、写作要点

### 节奏控制
*   **贫富循环**: 穷 -> 冒险 -> 暴富 -> 升级消耗 -> 穷。这个循环驱动主角不断换地图。
*   **花钱的爽感**: 赚钱是为了花。花钱升级、花钱打脸、花钱装逼。

### 情绪设计
*   **紧迫感**: "还差三千灵石就能买筑基丹了"，这种倒计时般的紧迫感能拉住读者。
*   **满足感**:看着储物袋里的灵石堆成山，或者系统面板上的余额跳动。

### 陷阱规避
*   **避免无限刷钱**: 如果主角有无限刷钱的Bug，必须加上严苛的限制（如不能买提升实力的东西，或者只能给别人花）。
*   **避免货币种类太多**: 最好不要超过三种主要货币（凡人钱、修仙钱、神豪钱），否则会乱。

## 五、检查清单
写完经济系统后自检：
- [ ] 货币的购买力是否稳定且合理？
- [ ] 主角是否有持续赚取资源的动力（缺钱）？
- [ ] 高端资源的交易方式是否符合强者身份？
- [ ] 资源的产出和消耗是否大致平衡？
