# Master-Disciple - 师徒羁绊

> **用途**: 建立从属与传承关系，提供保护伞（前期）或成为被守护对象（后期），制造温情或虐点。
> **适用流派**: 仙侠、玄幻、武侠、职业文
> **关键特征**: 传道受业、护短、背影、青出于蓝

---

## 一、核心特征分析

### 1.1 定义
师徒关系是网文中最稳固的非血缘关系。
老师（Master）不仅是技能书（Skill Provider），更是引路人（Guide）和精神支柱（Mental Anchor）。
**核心公式**: 仰望（前期） + 守护（中期） + 超越/送别（后期） = 传承

### 1.2 类型划分
1.  **随身老爷爷型**: 戒指里的灵魂。既是老师也是金手指。
2.  **护短狂魔型**: "我徒弟只有我能打，别人动他一根手指头试试？"（极具爽点）
3.  **高冷反差型**: 表面高冷，背地里给徒弟准备了一堆保命法宝。
4.  **坑徒型**: 师父不靠谱，徒弟操碎心（如《从前有座灵剑山》）。

### 1.3 常见误区
*   **工具人**: 师父发完秘籍就下线了，毫无存在感。
*   **无脑跪舔**: 师父对徒弟好得莫名其妙，像个保姆。
*   **师徒恋泛滥**: 虽然流行，但如果不是主打言情，不要轻易把师徒写成恋人，会破坏传承的庄重感。

## 二、Top作品解析

### 2.1 经典案例1: 《斗破苍穹》
**角色**: **萧炎 vs 药老**。
**核心手法**: **相互救赎**。
**分析**:
- **从利用到依赖**: 一开始药老吸萧炎斗气，萧炎恨他。后来两人相依为命，游历大陆。
- **关键剧情**: 萧炎为了复活药老，独闯魂殿；药老为了救萧炎，多次透支灵魂。
- **双向奔赴**: 这种感情超越了父子，是真正的生死之交。

### 2.2 经典案例2: 《将夜》
**角色**: **宁缺 vs 夫子**。
**核心手法**: **人间最强护短**。
**经典片段**:
> 夫子带宁缺周游列国，吃遍天下美食，顺便斩尽桃花。
> 夫子登天化月，是为了守护人间，也是为了给弟子们撑起一片天。
**分析**:
- **高山仰止**: 师父太强了，强到让人绝望。但正因为如此，成为他的弟子才无比荣耀。

### 2.3 经典案例3: 《求魔》
**角色**: **苏铭 vs 天邪子**。
**核心手法**: **疯癫中的真情**。
**分析**:
- 天邪子是个疯子，记忆混乱。
- 但他记得苏铭是他的徒弟。为了救徒弟，他不惜燃烧自己，哪怕全世界都遗忘了他。
- 这种残缺的守护更让人动容。

## 二点五、场景示例

### 【拜师】经典考验

```
"你想拜我为师？"

老者坐在悬崖边，钓着鱼，连头都没回。

"是。"林枫恭敬地跪在三步之外。

"为什么？"

"因为您是剑道宗师，我想学剑。"

"想学剑的人多了，为什么是你？"

林枫咬了咬牙："因为我愿意付出一切。"

"一切？"老者终于转过头，眼神锐利如刀，"你知道'一切'是什么意思吗？"

"知道。"

"那好。"老者指着悬崖，"跳下去。"

林枫愣住了："什么？"

"你不是说愿意付出一切吗？那就跳下去。"

"跳下去我会死的！"

"所以，你刚才说的是假话。"老者转回身，继续钓鱼，"滚吧。"

林枫握紧拳头，看着那万丈深渊。

云雾翻滚，看不到底。

他想起了自己为什么来到这里。

想起了家族的耻辱，想起了被羞辱的过往。

"如果不变强，活着又有什么意义？"

他站起身，走到悬崖边。

"前辈，如果我跳了，您会收我为徒吗？"

老者沉默了三秒。

"会。"

林枫深吸一口气，纵身一跃！

风在耳边呼啸，他闭上了眼睛。

下一秒——

一只手抓住了他的衣领。

老者将他提了上来，像提一只小鸡。

"你这个蠢货！真的跳了？！"

林枫睁开眼，看到老者又怒又笑的脸。

"老夫只是想看看你的决心...没想到你真敢跳！"

"那...我算通过了吗？"

老者叹了口气："通过了。从今天起，你就是我的徒弟。"

"记住，剑可以杀人，但人不能被剑杀。"

"一个连命都可以赌的人，还有什么学不会的？"

林枫跪下，重重磕了三个响头。

"徒儿林枫，拜见师父！"

师徒缘分，就此结下。
```

**分析**:
- **心理博弈**: "跳崖"是对决心的极限测试
- **层次递进**: 拒绝→考验→跃崖→救下→收徒，节奏紧凑
- **性格展现**: 师父又狠又护短，立体生动
- **主题升华**: 通过"剑不能杀人心"点明武道真谛
- **可复用要素**: "生死考验"式拜师流程

### 【护短】师父的背影

```
"你就是林枫的师父？"

三个黑衣人包围了老者。

为首的冷笑："交出林枫，我们可以饶你一命。"

老者放下茶杯，缓缓抬起头。

"你们...是哪个不长眼的势力？"

"血煞门！我们少主被你徒弟——"

话没说完，老者动了。

没有人看清他怎么出剑的。

三个黑衣人的脑袋同时飞起。

鲜血喷涌，尸体倒地。

老者收剑，重新坐下，端起茶杯，手都没抖一下。

"血煞门...记住了。"

远处，林枫躲在树后，浑身颤抖。

不是因为恐惧，是因为感动。

师父明明已经很久不出手了，因为每出一次手，寿元就会减少。

但为了他，师父毫不犹豫地杀了人。

而且是秒杀。

"师父..."林枫的眼眶红了。

老者没有回头，只是淡淡地说：

"还站在那里干什么？过来喝茶。"

"血煞门不会善罢甘休的..."林枫走过来，声音哽咽。

"不会善罢甘休的是我。"老者给他倒了杯茶。

"欺负我徒弟，血煞门，一个都别想跑。"

"但是师父，他们有三个元婴老怪——"

"那又如何？"老者笑了，"我当年一人一剑，屠过九个元婴。"

"你是我徒弟，天王老子来了，也得掂量掂量。"

林枫握着茶杯，泪水滴落。

这一刻，他什么都没说。

但他知道，这辈子，只要师父活着，他就永远有家。

而师父百年之后，他会替师父，守护这个家。
```

**分析**:
- **反差对比**: 平静喝茶→瞬间杀人→继续喝茶
- **护短宣言**: "欺负我徒弟"直接升级为灭门级仇恨
- **代价暗示**: "每出手寿元减少"让护短更显珍贵
- **双向承诺**: 徒弟感激→师父守护→徒弟继承
- **可复用要素**: "师父出场秒全场"的经典护短桥段

## 三、实战技法

### 3.1 基础公式 (The Inheritance Path)
1.  **拜师**: 经过考验，或者机缘巧合（"我看你骨骼清奇"）。
2.  **授业**: 传授独门绝技，教导做人的道理（通常是某种偏执的道理，如"打不过就跑"）。
3.  **护短**: 徒弟被欺负，师父出场镇压全场。
4.  **变故**: 师父受伤/失踪/死去，或者是敌人太强，师父挡不住了。
5.  **出师**: 徒弟站在师父倒下的地方，拔出剑，独自面对世界。

### 3.2 进阶技巧
*   **信物**: 一把剑、一个酒壶、一句话。师父不在了，看到信物如见人。
*   **反向照顾**: 师父生活不能自理（路痴、贪吃），徒弟反过来照顾师父，增加生活趣昧。
*   **隐秘的爱**: 师父当面骂徒弟废物，转身就去警告那些看不起徒弟的人。

### 3.3 常见套路
*   **代师收徒**: 师父已经飞升/死了，大师兄代为传授。
*   **被逐出师门**: 为了不连累师门，徒弟主动叛出，或者师父含泪逐出。

## 四、写作要点

### 节奏控制
*   **前期**: 师父是天花板，解决徒弟解决不了的麻烦。
*   **后期**: 徒弟超越师父，解决师父当年的遗憾（复活师娘、报仇）。

### 情绪设计
*   **安全感**: 只要师父在，天塌下来都有人顶着。
*   **孤独感**: 师父走了，也是主角真正长大的那一刻。

### 陷阱规避
*   **避免师父抢戏**: 师父可以强，但高光时刻要留给主角。师父的高光通常也是为了衬托主角或保护主角。
*   **避免辈分混乱**: 宗门文里辈分要搞清楚，不要乱套。

## 五、检查清单
写完师徒情节后自检：
- [ ] 师父是否有鲜明的性格特点（不仅是厉害）？
- [ ] 师徒之间是否有温情的互动细节？
- [ ] 师父的离去/退场是否推动了主角的成长？
- [ ] 徒弟是否继承了师父的某种精神内核？
