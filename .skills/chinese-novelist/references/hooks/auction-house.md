# Auction House - 拍卖会设计

> **用途**: 获得稀缺资源，引出后续剧情（杀人夺宝），展示财力（装逼），引出隐藏大佬。
> **适用流派**: 修仙、玄幻、都市鉴宝
> **关键特征**: 物品稀缺、竞价心理、鉴宝打眼、场外截杀

---

## 一、核心特征分析

### 1.1 定义
拍卖会（Auction）是网文中财富与眼力的竞技场。
它通常发生在主角急需某种材料，或者刚发了一笔横财需要消费的时候。
**核心公式**: 稀有宝物 + 恶意竞价 + 捡漏/截胡 + 场外PK = 拍卖会剧情。

### 1.2 类型划分
1.  **正规拍卖会**: 万宝阁、商会举办。讲究规矩，有包厢等级。
2.  **黑市拍卖会**: 藏头露尾，卖的都是违禁品（杀人越货的赃物、鼎炉）。容易发生火拼。
3.  **地下交易会**: 也就是"鬼市"。没有主持人，大家摆地摊，全靠眼力。

### 1.3 常见误区
*   **无脑加价**: "一千万！""两千万！""一亿！" 这种为了装逼而无视货币价值的竞价，会让经济系统崩坏。
*   **只有主角买**: 其他人都是死人，主角看上的东西大家都不抢，或者主角一买大家就抢。
*   **宝物太普通**: 既然上了拍卖会，就得是平时买不到的东西。不要卖大路货。

## 二、Top作品解析

### 2.1 经典案例1: 《斗破苍穹》
**场景**: **米特尔拍卖场 / 能够炼制丹药的黑袍人**。
**核心手法**: **身份反差**。
**分析**:
- 萧炎因为实力弱，披着黑袍装成高阶炼药师去卖药。
- 享受了顶级待遇（雅妃的诱惑），这种"弱者装强者"的爽感非常强。

### 2.2 经典案例2: 《黄金瞳》/《鉴宝》类
**场景**: **古玩拍卖 / 赌石**。
**核心手法**: **捡漏**。
**分析**:
- 别人眼里的废品，主角通过透视眼看到了里面的翡翠/古董。
- 用低价买下，当场解开/鉴定，价值翻万倍。全场震惊。

### 2.3 经典案例3: 《凡人修仙传》
**场景**: **天星城拍卖会**。
**核心手法**: **财不露白与杀人夺宝**。
**分析**:
- 韩立非常谨慎，买到了东西立刻就要考虑怎么跑路。
- 拍卖会结束往往是追逐战的开始。没人会真的老实付钱，除非打不过。

## 二点五、场景示例

### 【捡漏翻盘】经典写法

```
"下一件拍品，残破古画一幅，无法鉴定年代。"

主持人的声音有些敷衍，显然对这件东西不抱期望。

台下一片沉默。

"起拍价，一千灵石。"

无人举牌。

林枫的眼睛却亮了——他的鉴宝术在这幅画上看到了密密麻麻的灵气波动！

"一千！"他举起号牌。

"一千一次...两次...成交！"

几乎没有人注意到这件不起眼的小物。

林枫拿到古画，轻轻撕开表层的宣纸——

一副金光璀璨的"万灵朝宗图"显露真容！

"这...这是上古炼器宗师的炼器图谱！"

他身旁的鉴宝师老者颤抖着声音："至少价值百万灵石！"

瞬间，整个拍卖场所有的目光都锁定了林枫。

贪婪、震惊、杀意...各种情绪交织。

林枫暗道一声"糟了"，握紧了腰间的遁空符。
```

**分析**:
- **反差对比**: 无人问津→价值百万，爽感来自认知差
- **专业感**: "万灵朝宗图"、"炼器图谱"这些术语增加可信度
- **危机伏笔**: 最后一句暗示了后续的追杀情节
- **可复用要素**: "捡漏三段式"（无人要→主角识货→真相震撼）

### 【竞价博弈】经典写法

```
"筑基丹！三品上等！"

主持人高举丹瓶，场内瞬间燃起。

"此丹可使炼气大圆满修士增加三成筑基成功率！起拍价，十万灵石！"

"十五万！"
"十八万！"
"二十万！"

价格飞速上涨。

林枫咬了咬牙："二十五万！"

"呵，穷鬼也敢跟我抢？"三楼雅间传来嘲讽的笑声，"三十万！"

林枫认出了那个声音——赵家少爷，仇人。

他沉默了三秒，突然举牌："五十万！"

全场哗然。

"好！"赵少爷冷笑，"六十万！"

"六十五万！"林枫毫不犹豫。

"你..."赵少爷声音有些迟疑了。

林枫嘴角上扬，放下了号牌。

"六十五万一次！二次！三次！成交！"

主持人兴奋地敲下拍卖锤。

赵少爷脸色铁青——他刚才只是想抬价，没想到林枫突然不跟了！

六十五万买一颗市价三十万的筑基丹...血亏！

林枫靠在椅背上，轻声道："我说过，你会后悔的。"
```

**分析**:
- **心理博弈**: 不是比钱多，而是比谁先慌
- **节奏控制**: 快速竞价→沉默三秒→突然爆发→戛然停止
- **反向打脸**: 让对手高价买下自己不想要的东西
- **可复用要素**: "假装要买，实则坑对手"这一经典套路

## 三、实战技法

### 3.1 基础公式 (The Bidding War)
1.  **预热**: 提前放出消息，压轴品是XX神物，把各路牛鬼蛇神都吸引来。
2.  **捡漏**: 开场前或中间有些不起眼的东西（残片、生锈的铁剑），主角感觉到特殊波动，低价拿下。
3.  **冲突**: 主角看上了某个东西，必定有个富二代/反派也要买。
4.  **恶意竞价**: 反派把价格抬高，想坑主角。主角假装还要，最后突然放弃，反派高位接盘（反向打脸）。
5.  **压轴**: 真正的神物出现，天价成交（通常不是主角买，用来展示世界观的高端战力）。

### 3.2 进阶技巧
*   **盲盒机制**: 不告诉你是啥，只展示一部分，全靠赌。
*   **以物易物**: 到了最后，钱没用了，大佬们拿功法、领地、寿命来换。
*   **托**: 拍卖行自己安排人抬价，增加真实感。

### 3.3 常见套路
*   **寄拍**: 主角没钱，先拿出一瓶极品丹药去寄拍，瞬间成为贵宾，然后用卖药的钱买自己需要的东西。
*   **半路截杀**: "怀璧其罪"。只要主角买了好东西，出门必被堵。

## 四、写作要点

### 节奏控制
*   **流拍**: 偶尔出现一些东西没人要，增加真实感。
*   **高低起伏**: 也就是"前菜 - 主菜 - 甜点"。不要全程高能。

### 情绪设计
*   **紧张**: 钱不够了怎么办？（借钱/抵押）。
*   **愤怒**: 对方仗势欺人（"我是赵公子的狗，你敢跟我抢？"）。

### 陷阱规避
*   **避免嘴炮过多**: 拍卖就是用钱说话。不要在竞价时聊几百字的家常。
*   **避免主持人废话**: 主持人的介绍词要专业且诱惑，不要干巴巴的。

## 五、检查清单
写完拍卖会设定后自检：
- [ ] 拍卖品的等级是否符合当前地图的水平？
- [ ] 竞价过程是否有心理博弈（不仅仅是比数字）？
- [ ] 主角是否捡到了漏（金手指发挥作用）？
- [ ] 拍卖会结束后是否必须有一场战斗（杀人夺宝）？
