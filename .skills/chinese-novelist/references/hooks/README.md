## 📖 使用指南

### 这个知识库是什么？

这是一个**模块化的网文写作技法库**，每个`.md`文件都是一个独立的"写作模块"（Hook），可以在创作时按需调用。

### 如何使用？

1. **按流派查找**: 根据你的小说类型（如系统流、悬疑、言情），查看下面的分类索引
2. **按场景查找**: 根据当前要写的场景（如战斗、拍卖、宴会），直接跳转到对应文件
3. **按问题查找**: 遇到卡文？查看"剧情推进"分类；想写打脸？查看"社交反应"分类

### 文件结构说明

每个Hook文件包含：
- **核心特征分析**: 这个技法是什么，适用于哪些流派
- **Top作品解析**: 经典案例拆解
- **场景示例**: 2-3个完整的写作范例（代码块形式）
- **实战技法**: 公式、进阶技巧、常见套路
- **写作要点**: 节奏控制、情绪设计、陷阱规避
- **检查清单**: 写完后的自检项
- **关联资源**: 相关的其他Hook文件
