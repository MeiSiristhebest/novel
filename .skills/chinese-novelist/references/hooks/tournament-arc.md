# Tournament Arc - 赛事/大比设计

> **用途**: 集中展示战力进步，提供打脸舞台，确立江湖地位，引入新人物。
> **适用流派**: 学院流、竞技流、修仙流
> **关键特征**: 规则限制、观众反应、宿命对决、意外频出

---

## 一、核心特征分析

### 1.1 定义
赛事（Tournament）是网文中最经典的高潮桥段之一。
它通过规则将各路强者聚集在一起，提供了一个**合法且受瞩目**的装逼场所。
**核心公式**: 排名奖励 + 宿敌挑衅 + 惊艳表现 = 赛事爽点。

### 1.2 类型划分
1.  **宗门/学院大比**: 内部竞争，争夺核心弟子名额。通常是主角起飞的第一站。
2.  **天才战/天骄榜**: 全服范围的比赛，各路挂逼的大乱斗，扩宽世界观。
3.  **选拔赛**: 为了争夺某个进入秘境/遗迹/组织的资格。
4.  **生死擂台**: 解决私人恩怨，这就不是比赛了，是决斗。

### 1.3 常见误区
*   **比赛过程冗长**: 几十个人轮流打，写了几十章还没到主角，读者早跑了。
*   **对手千篇一律**: 都是"嘲讽->被打脸->不服"，缺乏个性。
*   **规则漏洞百出**: 说好点到为止，结果杀了人也没人管。

## 二、Top作品解析

### 2.1 经典案例1: 《火影忍者》
**赛事**: **中忍考试**。
**核心手法**: **群像塑造与意外**。
**分析**:
- 不仅展示了鸣人的成长，还立住了李洛克、鹿丸、假如等配角的人设。
- 只有第一场是正经笔试，后面全是实战，最后还被大蛇丸入侵打断了。**比赛往往不会顺利结束**。

### 2.2 经典案例2: 《吞噬星空》
**赛事**: **宇宙天才战**。
**核心手法**: **宏大叙事与阶层跨越**。
**分析**:
- 所有的宇宙国顶尖天才汇聚，规模感极强。
- 主角从地球走向宇宙，这是一次彻底的身份跃迁。

### 2.3 经典案例3: 《全职高手》
**赛事**: **职业联赛**。
**核心手法**: **战术博弈**。
**分析**:
- 这里的比赛不是单纯比谁拳头大，而是比战术、配合、心理素质。
- 每一场比赛都有独特的看点（如：擂台赛一挑三，团队赛绝地翻盘）。

## 二点五、场景示例

### 【预赛碾压】经典写法

```
"3号擂台，叶凡对战王虎！"

主持长老话音刚落，王虎便跳上擂台，炼气九层的气息毫不掩饰。

"听说你是废体？"王虎冷笑，"让我看看你怎么死——"

话音未落。

叶凡身形一闪，已经出现在他身后。

王虎瞳孔骤缩，还来不及反应，就感到后颈一痛，整个人飞出了擂台。

从开始到结束，三息。

全场死寂。

"这...这是什么速度？"
"刚才发生了什么？我只看到一道残影！"
"他隐藏实力了...这哪里是炼气三层！"

台下，几位长老互相对视，眼中露出惊讶之色。

主持长老深吸一口气："3号擂台，叶凡胜！"
```

**分析**:
- **时间压缩**: 用"三息"强调秒杀，制造反差
- **观众分层反应**: 普通弟子看不懂→长老看懂了→主持长老强行镇定
- **打脸节奏**: 对手话还没说完就被秒，这是最爽的打断方式
- **可复用要素**: "从开始到结束，X秒"的时间戳

### 【决赛逆转】经典写法

```
慕容云的剑已经刺到了叶凡的咽喉，只差三寸。

全场屏息。

"结束了。"慕容云嘴角上扬。

就在这时——

叶凡的身影突然虚化，像一片雾一样散开。

"残影？！"慕容云脸色大变。

真正的叶凡已经出现在他身后十丈之外，手中多了一颗赤红色的丹药。

"极品破境丹！他什么时候——"

来不及多想，叶凡的气息开始暴涨。

炼气九层...筑基一层...筑基三层！

"临阵突破？！这怎么可能！"

慕容云想要逃，但叶凡一掌已经拍至。

"天霜掌！"

掌力倾泻而下，慕容云的防御法宝应声碎裂，整个人被轰下擂台，昏死过去。

裁判长老愣了三秒，才举起手：

"本届宗门大比冠军...叶凡！"

沉寂一瞬后，山呼海啸般的欢呼声爆发了！
```

**分析**:
- **悬念最大化**: 先写危机（剑距咽喉三寸），再写翻盘
- **连环反转**: 残影→服丹→突破→一击制胜，节奏紧凑
- **数据可视化**: "筑基一层...三层"，让突破过程可见
- **情绪递增**: 死寂→惊呼→欢呼，三段式释放
- **可复用要素**: "临阵突破"这一经典套路的标准流程

## 三、实战技法

### 3.1 基础公式 (The Bracket)
1.  **预赛**: 快速筛选杂鱼。主角通常会秒杀对手，或者用奇葩方式晋级（如：睡觉睡过头了但运气好）。
2.  **初赛**: 遇到小反派，也就是之前的嘲讽者，打脸时刻。
3.  **半决赛**: 遇到强敌。主角被迫暴露底牌，或者临阵突破。
4.  **决赛**: 宿命对决。对手通常是大家公认的第一天才。
5.  **变故**: 颁奖时反派掀桌子，或者外部势力入侵。

### 3.2 进阶技巧
*   **种子选手**: 提前渲染几个夺冠热门（冰山美人、狂傲霸主、神秘黑马），给主角树立靶子。
*   **场外因素**: 有人下毒、有人收买裁判、有人绑架主角朋友。比赛从来不仅是台上的事。
*   **观众席**: 也就是"Reaction"。利用观众（路人、解说、大佬）的震惊来侧面描写主角的强。

### 3.3 常见套路
*   **废材逆袭**: 大家以为主角是废物，结果他拿了冠军。
*   **扮猪吃虎**: 主角隐藏实力，每次都险胜（演戏）。

## 四、写作要点

### 节奏控制
*   **略写**: 配角的战斗能跳过就跳过，除非是为了展示最终BOSS的技能。
*   **详写**: 主角的关键战斗，要一招一式写清楚，尤其是心理博弈。

### 情绪设计
*   **压抑**: 赛前所有人都看衰主角。
*   **爆发**: 夺冠那一刻，万众瞩目，全场欢呼。

### 陷阱规避
*   **避免胜之不武**: 主角赢要赢在硬实力或智慧上，不要靠对手突然拉肚子。
*   **避免奖励缩水**: 既然说是神级奖励，发奖时不要变成垃圾。

## 五、检查清单
写完赛事设定后自检：
- [ ] 比赛规则是否清晰且公平（或者不公平得有理由）？
- [ ] 是否塑造了至少3个有记忆点的对手？
- [ ] 观众的反应是否有多样性（不仅是震惊，还有质疑、担忧、崇拜）？
- [ ] 比赛结果是否改变了主角的社会地位？
