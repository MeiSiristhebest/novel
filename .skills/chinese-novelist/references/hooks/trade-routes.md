# Trade Routes - 贸易路线

> **用途**: 连接地图板块，提供护送/劫镖任务，引入外来物品/情报，展示世界广度。
> **适用流派**: 凡人流、经营文、大航海、西幻
> **关键特征**: 利润与风险并存、地理阻隔、商会博弈、特产差异

---

## 一、核心特征分析

### 1.1 定义
贸易路线（Trade Routes）是世界的血管。
哪里有需求，哪里就有商路。越是危险的地方，利润越高。
**核心公式**: 产地低价 + 危险运输 + 销地高价 = 暴利。

### 1.2 类型划分
1.  **陆路商队**: 骆驼、马车、镖局。穿越沙漠、戈壁、盗匪区。
2.  **水路航运**: 漕运、海运。运量大，不仅防海盗，还要防风暴/海怪。
3.  **空中航线**: 飞舟、狮鹫。高端货运，速度快，成本高。
4.  **跨界走私**: 通过不稳定的空间裂缝偷渡。极度危险，一本万利。

### 1.3 常见误区
*   **无中生有的繁荣**: 一个偏僻的小镇，既不产矿也不在交通要道，居然极其繁华有钱，不合逻辑。
*   **只有强盗没有保镖**: 路上全是劫匪，商队居然不请护卫，或者护卫全是弱鸡。
*   **物品单一**: 商队运的永远是"粮草"或"财宝"，没有具体货物细节（丝绸、香料、矿石）。

## 二、Top作品解析

### 2.1 经典案例1: 《亵渎》/《狼与香辛料》
**核心展示**: **经济战争与行商逻辑**。
**细节**:
- 罗格不仅仅是抢劫，他利用商路控制物价，低买高卖。
- 描写了商队如何在夹缝中求生存，以及金钱如何腐蚀信仰。

### 2.2 经典案例2: 《凡人修仙传》
**核心展示**: **乱星海捕妖航线**。
**细节**:
- 所有的岛屿依靠通过传送阵和海船连接。
- 每一条航线都对应着特定的妖兽分布。
- 主角跟随商队出海，是为了隐藏身份+蹭地图。

### 2.3 经典案例3: 《诡秘之主》
**核心展示**: **海上航路**。
**细节**:
- 不同的航路被不同的海盗将军控制。
- 航运连接了南北大陆的殖民贸易（橡胶、煤炭）。

## 二点五、场景示例

### 【商队遇袭】经典护镖桥段

```
"停！"

商队队长举起手，整个队伍瞬间停下。

五十匹骆驼、三十个脚夫、十名护卫，在沙漠中排成一条长龙。

"怎么了？"林枫警觉地四处张望。

"你看前面。"队长指着远处的沙丘。

那里，一杆黑旗在风中猎猎作响。

"黑风寨的旗帜...该死，他们怎么知道我们的路线？"

话音未落，沙丘后冲出了三十多名匪徒，骑着沙狼，呼啸而来。

"放下货物，饶你们不死！"

为首的匪首舔着刀刃，露出狞笑。

队长咬牙道："货里有三箱极品灵参，价值十万两黄金。交出去，我们商会就完了！"

他转头看向林枫："林少侠，按照合约，你要出手了。"

林枫叹了口气，走到队伍最前面。

"就一个人？"匪首哈哈大笑，"兄弟们，上！"

三十个匪徒冲锋——

林枫只是抬了抬手。

一道剑气横扫而出，黄沙爆炸，冲在最前面的五名匪徒连人带狼被斩成两半！

笑声戛然而止。

匪首的脸色瞬间变得煞白："筑...筑基境？！"

"还要打吗？"林枫平静地问。

"撤！快撤！"

黑风寨的土匪们丢盔弃甲，仓皇逃窜。

队长松了一口气，擦了擦额头的冷汗："不愧是天剑宗的高徒...这趟货算是保住了。"

"别高兴太早。"林枫皱眉，"黑风寨不可能这么快知道我们的路线，除非——"

他突然转头，盯着人群中的一个脚夫。

那人神色慌张，转身就跑。

"内鬼！"
```

**分析**:
- **危机三要素**: 货物贵重+匪徒围攻+内鬼出卖
- **实力展示**: 通过秒杀小怪展现主角战力
- **悬念转折**: 解决外部威胁后，揭露内部隐患
- **可复用要素**: "护镖任务"的标准流程

### 【跨界倒爷】套利暴富

```
"你确定这些'魔晶'在修仙界值钱？"

张伟看着手中黑色的晶体，有些怀疑。

系统回答：【确定。魔界的垃圾，在修仙界是稀缺资源。】

【魔晶在魔界价格：1魔币/公斤】
【魔晶在修仙界价格：100灵石/克】

张伟算了算，汇率差距是...一百万倍？！

"发财了啊！"

他立刻用身上所有的钱（100魔币），在魔界的地摊上买了100公斤魔晶。

摊主还以为他疯了："这么多魔晶你拿来干嘛？这玩意儿又不能吃。"

"做生意。"张伟神秘一笑。

穿过位面裂缝，他回到了修仙界。

"散修张伟，求见万宝阁掌柜！"

掌柜原本不想见他这种穷鬼，但听说他有货要卖，还是见了一面。

"掌柜，我有魔晶，你要吗？"

"魔晶？"掌柜眼睛一亮，"要！品质如何？"

张伟倒出一小堆黑色晶体。

掌柜拿起一块，仔细鉴定，脸色越来越兴奋：

"上品魔晶！纯度极高！你有多少？"

"一百公斤。"

掌柜的茶杯掉在地上。

"你...你说什么？"

三天后。

张伟的储物袋里装着一千万灵石，走出万宝阁。

他从一个身无分文的散修，变成了修仙界的暴发户。

而这一切的成本，只是一百魔币。

"这就是信息差的力量啊..."

他决定，下次去魔界，要买更多。
```

**分析**:
- **信息不对称**: 一个世界的垃圾=另一个世界的珍宝
- **数字震撼**: 百万倍汇率差制造冲击感
- **认知升级**: 展示"跨界思维"的价值
- **可复用要素**: "倒爷发家"的万界流经典套路

## 三、实战技法

### 3.1 基础公式 (The Caravan)
1.  **货物**: 必须是产地极其便宜、销地极其昂贵的东西（如：从雪原运冰块到沙漠）。
2.  **阻碍**: 自然天险（过不去的山/河）和人为阻碍（关税/强盗）。
3.  **节点**: 休息站/绿洲。这是情报交换和发生冲突（黑店）的地方。
4.  **势力**: 谁罩着这条路？（如果不交保护费会怎样）。

### 3.2 进阶技巧
*   **开辟新航路**: 主角发现了前人不知道的安全路线，从而垄断贸易。
*   **运人**: 贩卖人口/奴隶（暗黑风），或者护送重要人物（任务模式）。
*   **万界商行**: 跨位面贸易。把科技世界的手机卖给修仙界，把修仙界的丹药卖给魔法界。

### 3.3 常见套路
*   **劫富济贫**: 主角抢了反派商队的黑货，发了一笔横财。
*   **扮猪吃虎**: 主角混在商队里当杂役/护卫，遇到劫匪时出手救场。

## 四、写作要点

### 节奏控制
*   **旅途见闻**: 通过描写商队沿途看到的风景、听到的传闻，侧面交代世界观。
*   **遭遇战**: 商队通常会遇到两三波攻击。第一波是试探，第二波是硬茬，第三波是绝杀（或反转）。

### 情绪设计
*   **贪婪**: 商人为了利润可以践踏一切法律。
*   **辛酸**: 底层脚夫/水手的艰难生存，与商人的暴利形成对比。

### 陷阱规避
*   **避免流水账**: 不要写"第一天走了多少里，第二天走了多少里"。只写有冲突发生的日子。
*   **避免主角变保姆**: 主角是来搭顺风车的，不是来当全职保姆的。除非给的钱多到位。

## 五、检查清单
写完贸易设定后自检：
- [ ] 贸易路线的存在是否有地理和经济逻辑支撑？
- [ ] 运输的货物是否有具体的描述？
- [ ] 商队遇袭的理由是否充分（内鬼/仇杀/求财）？
- [ ] 这条路线是否为主角提供了情报或资源？
