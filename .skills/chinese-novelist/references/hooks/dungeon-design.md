# Dungeon Design - 副本设计

> **用途**: 提供封闭的探索环境，集中展示战斗和解谜，投放特定资源。
> **适用流派**: 网游流、无限流、盗墓流、修仙遗迹
> **关键特征**: 封闭性、规则性、高风险高回报、阶段性

---

## 一、核心特征分析

### 1.1 定义
副本（Dungeon）是与主世界隔离的特殊空间。
它不仅是打怪升级的场所，更是**剧情的加速器**。在副本里，时间和空间通常是特殊的。
**核心公式**: 封闭环境 + 独特规则 + 最终BOSS + 宝藏 = 副本

### 1.2 类型划分
1.  **探索型/遗迹**: 前人留下的洞府、墓穴。重点是破解机关和阵法（《鬼吹灯》）。
2.  **杀戮型/斗兽场**: 进去就是为了杀光所有怪或者活到最后（吃鸡模式）。
3.  **规则型/解谜**: 必须遵守特定规则才能生存，甚至不需要战斗（《地狱公寓》）。
4.  **试炼型/塔**: 一层比一层难，测试主角的极限（通天塔）。

### 1.3 常见误区
*   **流水账**: "进门，杀怪，开箱子，出门"。没有剧情波折，像写说明书。
*   **逻辑不通**: 一个几万年前的遗迹，里面的火把还亮着，机关还润滑如新，怪还活蹦乱跳（除非有解释）。
*   **缺乏特色**: 所有的副本都长得像山洞。

## 二、Top作品解析

### 2.1 经典案例1: 《全职高手》
**副本**: **格林之森、埋骨之地**。
**核心手法**: **机制攻略**。
**分析**:
- 重点不是怪有多强，而是怎么利用机制（卡怪、仇恨连锁）去打赢。
- 副本记录的争夺是推动公会矛盾的核心动力。

### 2.2 经典案例2: 《无限恐怖》/《惊悚乐园》
**副本**: **恐怖片世界 / 剧本**。
**核心手法**: **沉浸式剧情**。
**分析**:
- 每一个副本都是一个完整的小故事，有自己的NPC和世界观。
- 主角不仅要活下来，还要挖掘副本背后的隐藏剧情（世界观碎片）。

### 2.3 经典案例3: 《盗墓笔记》
**副本**: **七星鲁王宫、云顶天宫**。
**核心手法**: **氛围营造**。
**分析**:
- 副本本身就是巨大的悬念。为什么这里会有这个？墓主人生前在干什么？
- 环境描写（尸蟞、青眼狐尸）极大地调动了读者的感官。

## 二点五、场景示例

### 【副本探索】经典三房结构

```
【副本：遗忘神殿（推荐等级：Lv.25）】
【当前进度：2/3】

林枫推开最后一扇石门。

眼前，是一个圆形大厅，直径百米。

正中央，一尊十米高的石像矗立，手持巨剑。

【警告！BOSS已苏醒】
【守护者·石像巨人（Lv.30 精英）】

"等级比我高5级..."林枫咽了口唾沫。

石像的眼睛突然亮起红光，沉重的脚步声响起。

"没有退路了。"

他握紧武器，开始观察——

BOSS的脚踝有裂纹，左手的剑刃有缺口。

"弱点！"

战斗开始！

十分钟后，石像轰然倒塌。

【恭喜通关！】
【获得：石像之心×1、精良装备宝箱×1、经验×5000】

林枫瘫坐在地上，看着掉落的金色宝箱，嘴角上扬。

"值了。"
```

**分析**:
- **游戏化UI**: 副本信息、BOSS血条、通关奖励
- **等级压制**: 制造紧张感（高5级）
- **弱点发现**: 观察力的奖励
- **可复用要素**: "副本三段式"（进入→挑战→通关奖励）

### 【机关陷阱】解谜副本

```
"地上的石板...颜色不一样。"

李寒蹲下身，仔细观察。

黑色石板八块，白色石板八块，交错排列。

墙上刻着一行古文：

"踏错即死，唯有棋路可通天。"

"棋路？这是象棋的残局！"

他脑海中浮现出祖父教他的棋谱——

"马走日，象走田..."

他深吸一口气，踏上第一块石板——白色，"马三进二"的位置。

安全。

第二步，黑色石板。

脚刚落地，四周墙壁突然射出箭雨！

"糟了！"他急忙滚向一旁。

"不对...我记错了，应该是'车二平五'！"

他重新站在起点，额头冷汗直流。

这一次，必须一步不错。

十七步后，他站在了最后一块石板上。

石门轰隆打开。
```

**分析**:
- **知识门槛**: 需要象棋知识，增加真实感
- **试错惩罚**: 第一次失败制造紧张
- **步数计数**: "十七步"让过程可视化
- **可复用要素**: "解谜副本的容错机制"

## 三、实战技法

### 3.1 基础公式 (The Dungeon Structure)
1.  **入口**: 很隐蔽，或者需要特殊钥匙。进入的过程本身就是一种筛选。
2.  **前厅**: 遇到第一波小怪，适应副本环境（如：水下、无重力）。
3.  **中庭/岔路**: 分散团队，制造单刷机会，或者遇到解谜关卡。
4.  **BOSS房**: 最终决战。环境通常对BOSS有利。
5.  **隐藏房间**: 只有主角能发现的额外奖励区。

### 3.2 进阶技巧
*   **多方博弈**: 副本里不仅有怪，还有其他竞争者（敌对势力）。"人心比鬼神更可怕"。
*   **时限**: "遗迹将在三小时后崩塌"。倒计时能瞬间拉高紧张感。
*   **反转**: 以为是去寻宝，结果是去送祭品。副本本身是一个巨大的陷阱。

### 3.3 常见套路
*   **带妹下本**: 展示主角实力的机会。
*   **首杀**: 拿下全服唯一的荣誉。
*   **传承**: 副本主人留下的残魂选中了主角。

## 四、写作要点

### 节奏控制
*   **张弛有度**: 激烈的战斗后要有分赃/休整的环节。
*   **高潮叠加**: 打完小BOSS以为结束了，结果唤醒了真BOSS。

### 情绪设计
*   **贪婪**: 面对堆积如山的宝藏时的兴奋。
*   **恐惧**: 面对未知机关时的紧张。

### 陷阱规避
*   **避免迷路**: 描写迷宫时不要让读者也迷路。多用标志性参照物。
*   **避免收益递减**: 如果第二个副本的奖励和第一个差不多，读者就会疲劳。

## 五、检查清单
写完副本设定后自检：
- [ ] 副本是否有独特的环境特征（不仅是换个贴图）？
- [ ] 是否有至少一个有趣的机制（机关/规则）？
- [ ] 副本内的竞争者（人或怪）是否给主角制造了足够麻烦？
- [ ] 最终奖励是否配得上主角的付出？
