# Action Scene Guide - 战斗描写指南

> **用途**: 指导如何撰写高燃、流畅、画面感强的战斗场面。
> **适用流派**: 玄幻、仙侠、武侠、都市异能、科幻、网游
> **关键特征**: 节奏紧凑、招式清晰、打击感强、环境互动

---

## 一、核心要素

### 1.1 战斗节奏 (Pacing)
战斗不是流水账，要有张有弛。
*   **前奏 (Build-up)**: 气氛压抑，对峙，垃圾话（攻心），气势攀升。
*   **爆发 (Explosion)**: 瞬间交手，短句为主，动词密集。
*   **拉扯 (Exchange)**: 见招拆招，优劣势转换，心理博弈。
*   **高潮 (Climax)**: 底牌尽出，一击定胜负。
*   **余韵 (Aftermath)**: 战场惨状，赢家姿态，围观者反应。

### 1.2 画面感 (Visualization)
*   **动词为王**: 少用形容词，多用精准动词（砸、劈、挑、贯穿、撕裂）。
*   **空间方位**: 明确敌我位置，利用地形（墙角、树梢、悬崖）。
*   **感官细节**: 听觉（骨骼碎裂声）、触觉（拳风刮面）、视觉（血雾爆开）。

### 1.3 打击感 (Impact)
*   **受力反馈**: 打中之后，物体/人体的反应（倒飞、凹陷、喷血、地面龟裂）。
*   **侧面烘托**: 旁观者的惊呼、环境的破坏（气浪掀翻桌椅）。

## 二、流派特化技巧

### 2.1 玄幻/仙侠 (Xuanhuan/Xianxia)
*   **特效流**: 强调光影、声势、天地异象。
    *   *例*: "苍穹之上，巨掌如山岳般压下，空气被挤压得发出爆鸣。"
*   **意境流**: 强调道韵、法则、看似缓慢实则极快。
    *   *例*: "这一剑平平无奇，却仿佛切断了时光，让人避无可避。"

### 2.2 传统武侠 (Wuxia)
*   **招式拆解**: 具体的动作描写，攻防逻辑清晰。
    *   *例*: "他侧身避过刀锋，左手也就是一招‘顺手牵羊’，搭上对方手腕，借力一送。"
*   **兵器质感**: 金属碰撞、火花、兵器重量。

### 2.3 都市/科幻 (Urban/Sci-Fi)
*   **速度与火控**: 枪火、弹道、掩体、计算。
*   **暴力美学**: 拳拳到肉，骨折声，肾上腺素。

## 三、实战示例

### 3.1 示例：玄幻爆发
```markdown
萧炎眼眸猛然睁开，漆黑的眸子中，两团火苗骤然跳动。
“天火三玄变，第一变！”
心中一声低喝，周围原本狂暴的能量，仿佛受到了某种牵引，疯狂地对着萧炎体内灌注而去。
随着能量的涌入，萧炎的气息，也是在这一霎，以一种惊人的速度，节节攀升！
那股强悍的威压，令得不远处的纳兰嫣然，俏脸骤然苍白。
```

### 3.2 示例：武侠交锋
```markdown
刀光一闪，快得不可思议。
傅红雪的刀已出鞘，又归鞘。
没人看清他的动作，只听得一声清脆的金属撞击声。
那大汉保持着举刀劈砍的姿势，僵立不动。
片刻后，他手中的厚背大刀从中折断，“叮当”坠地。
紧接着，一道血线从他额头缓缓浮现。
```

## 四、常见误区
1.  **回合制**: 你打我一下，我打你一下，像在玩宝可梦。要写出**打断、连招、预判**。
2.  **解说过多**: 招式还没发出来，先心理活动解释三百字原理。打完再解释！
3.  **数值崩坏**: 前面说这一拳五万斤，后面打在木门上只留个印。破坏力要统一。

---

## 五、检查清单
- [ ] 动词是否足够有力？（避免“进行了攻击”）
- [ ] 是否有环境描写？（不仅是两个人打，还要破坏环境）
- [ ] 是否有身体反应？（痛感、麻木、耳鸣）
- [ ] 胜负逻辑是否合理？（避免机械降神）
