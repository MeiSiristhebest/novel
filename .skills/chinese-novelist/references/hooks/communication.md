# Communication - 通讯体系

> **用途**: 传递情报，控制信息差，摇人打群架，推动剧情发展。
> **适用流派**: 仙侠、谍战、科幻、奇幻
> **关键特征**: 延迟、距离限制、保密性、即时通讯带来的变革

---

## 一、核心特征分析

### 1.1 定义
通讯（Communication）解决了"我怎么知道那边发生了什么"的问题。
在古代/低魔背景下，**信息差**是智斗的核心；在现代/高魔背景下，**情报处理**是核心。
**核心公式**: 传讯手段的便捷程度 = 剧情推进的速度。

### 1.2 类型划分
1.  **物理传讯**: 信鸽、飞剑传书、人力驿站。（慢，容易丢，容易被截获）。
2.  **法术传讯**: 传音符、水镜术、纸鹤。（有距离限制）。
3.  **即时通讯**: 手机、玉简联网、神识网络。（无延迟，全球广播）。
4.  **心灵感应**: 主仆契约、灵魂链接。（点对点，无法窃听）。

### 1.3 常见误区
*   **如臂使指**: 在古代背景下，将军在千里之外居然能实时指挥前线，完全无视通讯延迟。
*   **全知全能**: 主角哪怕在深山老林里，也知道天下大事，除非随身带了收音机（老爷爷）。
*   **忽略屏蔽**: 既然有手机，为什么不求救？（必须要有"没信号/被屏蔽"的设定）。

## 二、Top作品解析

### 2.1 经典案例1: 《凡人修仙传》
**体系**: **万里符 / 传音符**。
**细节**:
- 传音符发出一道火光飞走。如果对方闭关/有阵法阻隔，就收不到。
- 这种**不确定性**（没收到、收晚了）制造了大量的悬念和误会。

### 2.2 经典案例2: 《修真聊天群》
**体系**: **QQ群 / 手机**。
**细节**:
- 现代修真，直接用社交通讯软件。
- 这种设定极大地加速了剧情节奏，也带来了独特的网聊幽默感（表情包斗图）。

### 2.3 经典案例3: 《雪中悍刀行》
**体系**: **谍报网（拂水房/赵勾） + 飞鸽**。
**细节**:
- 徐凤年养鹰，用来传递急报。
- 情报是有等级的，绝密情报用特殊的鸟，死士护送。

## 二点五、场景示例

### 【传音入密】紧急通讯

```
战场混乱。

林枫眼看队友即将中埋伏，却无法大喊——敌人就在附近。

他咬破舌尖，一滴血落在传音符上。

"三息后，往左侧撤！"

声音直接在队友耳边响起，外人听不到半点。

队友愣了一瞬，随即明白过来，脚下猛然一踏——

就在他离开原地的瞬间，三支弩箭破空而至，钉在了他刚才站立的位置！

"差点就死了..."队友心有余悸地回传，"多谢！"

林枫没有回答，传音符已经化作灰烬。

一次性道具，珍贵且致命。
```

**分析**:
- **即时性**: 传音符的及时性拯救了队友
- **代价感**: 咬破舌尖+一次性道具，不是随便用的
- **紧张感**: 通过通讯工具制造生死一线的节奏
- **可复用要素**: "战场通讯"的标准范式

### 【飞鸽传书】跨城情报

```
长安城，深夜。

一只灰羽信鸽落在窗台上，腿上绑着竹筒。

李寒取下竹筒，打开密信——

"京师三日后有变，速回。——父字"

短短十个字，却让他脸色骤变。

从江南到京师，八百里加急也要五天。

但这只信鸽...它是怎么两天就飞到的？

除非，不是普通的鸽子。

他仔细看向那只鸽子——眼睛是淡金色的，双爪有细微的符文。

"灵鸽...只有朝廷密探才用的传讯工具！"

父亲动用了这个，说明事态已经危急到极点。

李寒来不及多想，翻身上马，连夜启程。

信鸽扑棱着翅膀飞走了，消失在夜幕中。

它还要飞回去，向发信人汇报：信已送达。
```

**分析**:
- **世界观**: 通过"灵鸽"vs"普通鸽"体现修仙世界的通讯层级
- **悬念**: 简短的密信引发危机感
- **双向确认**: 鸽子飞回汇报，增加真实性
- **可复用要素**: "异常的通讯工具暗示事态严重"

## 三、实战技法

### 3.1 基础公式 (The Info Flow)
1.  **发送**: 需要消耗代价（法力、钱）。
2.  **传输**: 会被拦截、篡改、延迟。
3.  **接收**: 需要特定的解码方式（比如只有主角看得懂的暗号）。
4.  **广播**: 全服通告（如：天道榜单更新、由于大能陨落导致的天地同悲）。

### 3.2 进阶技巧
*   **信息茧房**: 反派通过控制舆论/通讯渠道，给主角泼脏水，主角百口莫辩。
*   **跨界通讯**: 如何联系上界的大佬？需要建立昂贵的祭坛，且只能说几句话（字字千金）。
*   **死前遗言**: 很多反派死前会发出一道神念标记主角，导致主角被全图追杀（Debuff）。

### 3.3 常见套路
*   **摇人**: 打不过了，捏碎玉牌，召唤老祖降临。
*   **直播**: 用留影石/水镜术直播主角的战斗，全场震惊（装逼最大化）。

## 四、写作要点

### 节奏控制
*   **滞后**: 坏消息通常传得很快，好消息传得很慢（或者反过来，为了制造反转）。
*   **时间差**: 利用通讯的时间差打歼灭战（在你援军收到消息赶来之前，把你灭了）。

### 情绪设计
*   **绝望**: 看着手机没信号，或者传音符发不出去，周围全是敌人。
*   **八卦**: 修真界的报纸/论坛是吐槽和调节气氛的好地方。

### 陷阱规避
*   **避免过于方便**: 如果无论何时何地都能联系上，主角就没有独立解决危机的压力了。
*   **加密意识**: 重要的情报不要明文发送，会被反派截获。
