# 都市异能风格范文 (Urban Supernatural Style Exemplar)

> **流派定位**：以现代都市为背景，将超自然能力融入日常生活，核心是"日常与异常的反差"、"隐秘世界的规则"、"普通人与异能者的身份张力"。

## 🎯 核心风格特征

### 1. 世界观架构类型

| 类型 | 设定特点 | 代表作 | 核心冲突 |
|-----|---------|-------|---------|
| **隐秘世界型** | 异能世界隐藏在普通社会之下 | 《天王》 | 身份暴露的风险 |
| **全民觉醒型** | 异能已公开，社会结构改变 | 《全球高武》 | 强者为尊的竞争 |
| **灵气复苏型** | 世界规则改变，异能涌现 | 《我师兄》 | 新旧秩序的碰撞 |
| **末世都市型** | 灾变后的都市废墟 | 《第一序列》 | 生存与人性 |
| **恐怖日常型** | 恐怖元素融入日常 | 《恐怖屋》 | 未知的恐惧 |

### 2. 日常锚点设计（必备）

```
【日常元素清单】
交通：地铁、公交、堵车、共享单车、滴滴打车
饮食：外卖、奶茶、火锅、烧烤、便利店
工作：996、加班、甲方、PPT、工位
社交：微信、朋友圈、抖音、B站、网络热梗
生活：房贷、相亲、催婚、养猫、熬夜

【使用方法】
每个异能场景前后，必须有至少一个日常锚点。
例：地铁里猎杀怪物 → 回家被老妈催吃饭
```

### 3. 异能设计原则

```
【COST原则】
C - Cooldown（冷却）：能力不能无限使用
O - Overdraft（透支）：过度使用有代价
S - Secret（保密）：能力暴露有风险
T - Trigger（触发）：需要特定条件才能使用

【示例】
能力：意念控物
冷却：每次使用后需休息30秒
透支：控制超过10公斤会头痛欲裂
保密：被普通人看到会被特殊部门带走
触发：需要肉眼能看到目标
```

### 4. 语言风格

| 场景 | 风格 | 示例 |
|-----|-----|------|
| **日常对话** | 现代网络语 | "我去，这也太离谱了吧？" |
| **能力释放** | 简洁有力 | "定。"（一字咒语） |
| **组织术语** | 现代+神秘 | "异能者管理条例第七条" |
| **战斗描写** | 快节奏短句 | "闪避。出拳。命中。一击毙命。" |
| **内心独白** | 吐槽+自嘲 | "所以我一个月薪三千的社畜，现在要去打怪？" |

## ✍️ 经典片段示范

### 【隐秘猎杀】日常与异常的反差

```
晚高峰的3号线，人挤人。

陈默把耳机音量调到最大，试图用周杰伦隔绝身边大妈的电话粥。

没用。

那位大妈中气十足，隔着三个人都能听清她在教训儿媳妇不会带孩子。

手机震动。

陈默低头，看到一条加密信息——

「目标确认。三号车厢，灰色卫衣，鸭舌帽。非人类。威胁等级：C。速处理。」

他面无表情地划掉消息，余光扫过车厢。

找到了。

靠门的位置，一个二十出头的年轻人，低着头刷抖音，看起来和普通乘客没什么区别。

但陈默能看到普通人看不到的东西——

那家伙的影子不对。

正常人的影子会随着灯光移动，而这个"人"的影子却像活物一样在地板上蠕动，偶尔伸出细长的触手，舔舐着周围乘客的脚踝。

没人察觉。

触手每舔过一个人，那人的脸色就会苍白一分。一个被舔了好几次的白领已经开始打哈欠，眼皮沉重得快睁不开。

「食梦者。」陈默在心里给目标分类，「低级货，但很麻烦。专门吸食精力，不会直接杀人，但被过度吸食的人会陷入嗜睡、抑郁……严重的会'睡死过去'。」

地铁减速。

「东直门站到了，请从左侧车门下车。」

陈默随着人流移动，不动声色地靠近目标。

灰卫衣男似乎察觉到了什么，猛地抬头。

两人的目光在空中相遇。

那一瞬间，灰卫衣男的瞳孔骤然收缩——他看到了陈默眼中一闪而过的金色光芒。

那是猎人的印记。

"操！"

他扭头就跑，撞开了好几个乘客，换来一串骂声。

陈默没有追。

他只是微微抬起右手，食指和中指并拢，轻轻一点——

"定。"

灰卫衣男的身体僵在原地，像被按下了暂停键。只有眼珠还能转动，里面满是惊恐。

周围的乘客从他身边鱼贯而出，没人觉得奇怪。在他们眼中，这只是一个在发呆等人的年轻人。

陈默走到他面前，从口袋里掏出一张黄纸。

"《都市异能者管理条例》第七条，"他面无表情地念道，"禁止在公共场所进行非授权能量汲取。你已违规387次。"

灰卫衣男的嘴唇艰难地蠕动："我……我可以解释……"

"第八条，"陈默继续念，"累计违规超过100次者——"

他把黄纸贴上了对方的额头。

"就地正法。"

刹那间，那个扭曲的影子发出无声的尖叫，像冰块遇到烈日一样迅速消融。灰卫衣男的身体也开始变得透明，最后化作一缕黑烟，消散在空调风中。

地铁门关闭。

「本次列车终点站为……」

陈默重新戴上耳机，靠着门边闭目养神。

手机震动。

「处理完了？」

他打了两个字：「嗯。」

「辛苦。顺便，你妈让我问你今晚回不回家吃饭。」

陈默沉默了三秒，回复：

「回。让她别放香菜。」

都市异能者的日常，大概就是这样吧。

杀完怪物，回家吃饭。
```

**风格要点**：
- 用地铁、大妈、刷抖音等日常元素开场
- 异常元素自然融入（影子不对）
- 能力使用要简洁帅气（一字"定"）
- 用日常细节收尾（不放香菜）

### 【国术实战】模仿《龙蛇演义》风格

```
擂台上，两个人影对峙。

王超穿着普通的运动服，站姿松散，看起来像个刚毕业的大学生。

对面的�的藤野�的一身白色道服，眼神锐利如鹰，浑身散发着杀气。

"中国人的功夫，不过是花拳绣腿。"藤野轻蔑地说，"今天，我要让全世界看看，什么才是真正的武道。"

观众席上，议论纷纷。

"那个中国小子看起来好弱啊……"

"据说是什么形意拳传人？没听说过。"

"别看了，肯定是来送的。"

王超没有说话。

他只是缓缓抬起双手，摆出一个架子——

三体式。

形意门的根基。

藤野眼睛一眯："有点意思。"

他不再废话，身形暴起，一记正蹬直踹王超面门——

极真空手道的必杀技，膝法！

速度快如闪电，角度刁钻至极。

普通人根本反应不过来。

但王超不是普通人。

他的身体微微一沉，左脚往前一垫，整个人像一支离弦的箭，迎着膝击——

撞了上去。

"找死！"藤野大喜。

硬接膝击？这是嫌命长吗？

然而下一秒，他的表情僵住了。

王超的右拳，在他的膝盖碰到自己之前，打在了他的小腹上。

崩拳。

形意五行拳之首。

拳经有云：崩拳如箭，一往无前。

这一拳没有任何花哨的动作，只是一个简单的直线——

但就是这条直线，打穿了藤野所有的防御。

砰！

藤野整个人飞了出去，在空中喷出一口鲜血，重重地砸在擂台边缘。

全场死寂。

三秒后——

"这……这什么拳法？一拳就KO了？！"

"看清楚了吗？我没看清楚！"

"卧槽，这他妈才多久？两秒？三秒？"

王超收拳，站直身体。

他的呼吸很稳，脸上没有任何得意之色。

"形意拳，"他淡淡开口，"只是中国功夫的冰山一角。"

"你想见识什么才是真正的武道？"

"很简单。"

"再来十个。"

解说席上，一位老人颤抖着站起来。

他是国术界的泰斗，已经八十高龄，见过太多所谓的"高手"。

但今天，他终于见到了——

传说中的，一击必杀。

"这才是……真正的形意啊……"老人喃喃自语，"当年王芗斋的崩拳，大概也就是这个样子吧……"

他的眼眶，已经湿润了。
```

**风格要点**：
- 武术描写要有"劲"的概念
- 一招制敌，体现国术的实战性
- 用旁观者的震惊烘托主角实力
- 引用真实武术家增加质感（王芗斋）

### 【能力觉醒】标准写法

```
林凡是在被卡车撞飞的那一刻觉醒的。

当时他正在过马路，手机刷着股票，完全没注意到闯红灯的泥头车。

等他反应过来的时候，车头已经距离他不到两米。

他甚至能看清挡风玻璃后面司机惊恐的脸。

完了。

这是他当时唯一的想法。

然后——

时间停止了。

不是那种比喻性的"时间仿佛停止"，而是真的、物理意义上的，停止了。

泥头车悬在半空中，轮子还在转，但车身纹丝不动。

空中的灰尘凝固成一颗颗微小的晶体。

一只正在飞过的麻雀，翅膀保持着上扬的姿势，像一个标本。

林凡愣住了。

他下意识地挥了挥手——空气很正常，没有变成果冻。

他往旁边走了两步——身体很正常，没有任何阻力。

这是……什么情况？

他走到泥头车旁边，绕着车转了一圈。驾驶座上的司机脸色惨白，嘴巴张成O型，显然正在尖叫——但没有声音传出来。

"我……我能控制时间？"

话音刚落，世界恢复正常。

泥头车从他身边呼啸而过，带起一阵狂风。

后面的行人发出惊呼——他们清楚地看到那辆车明明撞上了那个年轻人，但年轻人却突然"闪"到了旁边？

林凡站在原地，看着自己的双手。

它们在微微颤抖。

不是因为恐惧，而是因为……兴奋。

他拥有了超能力。

在这个异能者遍地走、觉醒者多如狗的时代，他终于不再是个普通人了。

"系统？金手指？老爷爷？"他小声嘀咕，"有没有谁来给我解释一下？"

没有回应。

看来没有系统。

纯天然觉醒。

"……也行吧。"林凡活动了一下脖子，"自己摸索就自己摸索。"

他看了看表，骂了一声。

"操，快迟到了。"

超能力什么的回头再说，先打卡要紧。

迟到一次扣五十，这个月已经扣了三次了。

林凡撒腿往公司跑去。

身后，几个目击者还在议论：

"刚才那个人是不是瞬移了？"

"你在看漫画吗？这是现实好吧。"

"可是我明明看到……"

"肯定是眼花。走了走了，上班要迟到了。"

人群散去。

没有人知道，这座城市又多了一个异能者。

而他此刻最担心的事情，是全勤奖还能不能保住。
```

### 【隐秘组织】世界观展示

```
"欢迎加入特异事务管理局。"

面试官是个三十多岁的中年人，西装革履，看起来就像个普通的HR。

但林凡知道，这个"普通"的HR，三分钟前徒手捏碎了一块铁板。

"我……我还没说同意呢。"

"你没有选择。"面试官微笑着，"根据《异能者管理条例》，所有新觉醒的异能者必须在72小时内到管理局报到。否则……"

他从抽屉里拿出一张表格推了过来。

"否则会被视为'野生异能者'，列入清除名单。"

林凡咽了口唾沫："清除？"

"字面意思。"

好家伙。

"那个……请问工资待遇怎么样？"

面试官愣了一下，似乎没想到他会问这个。

"基本工资8000，加上任务津贴、危险补助、异能等级补贴……入门级一个月大概两三万吧。"

"五险一金呢？"

"……六险二金。多一个异能险和一个抚恤金。"

林凡眼睛一亮："有年假吗？"

"有。但基本用不上。"

"加班有加班费吗？"

"有。而且是三倍。"

"……"面试官无奈地揉了揉太阳穴，"你到底是来面试的还是来查账的？"

"我就想确认一下。"林凡认真地说，"毕竟上一份工作996还不给加班费，我有心理阴影。"

"放心，管理局不会这样。"

"那就好。"林凡拿起笔，在表格上签下自己的名字，"我加入。"

"……你就不问问具体要做什么吗？"

"猎杀怪物、维护秩序、保护普通人对吧？我看过电影。"

面试官沉默了三秒。

"差不多。但现实比电影危险一万倍。"

"没关系。"林凡站起身，"只要不用996，一切都好说。"

面试官看着这个年轻人的背影，不知道该说什么。

入职理由千千万，他第一次见到因为"不想996"而加入猎魔组织的。

这一届年轻人，真的很难带啊。
```

## ⚠️ 常见雷区

| 雷区 | 错误示范 | 正确示范 |
|------|---------|---------| 
| 能力无限制 | 主角随便用能力，没有任何代价 | 用完能力头疼欲裂，需要休息恢复 |
| 太玄幻 | 在现代都市飞来飞去 | 能力要低调，避免暴露 |
| 脱离日常 | 全程打怪升级，没有生活气息 | 穿插外卖、房租、相亲等日常 |
| 组织太中二 | "吾乃黑暗骑士团第七序列" | "特异事务管理局外勤三组" |
| 普通人太蠢 | 异能者当众放技能没人发现 | 需要用障眼法、记忆消除等手段 |

## 🔧 都市异能世界观模板

```
【组织架构示例】
官方组织：
- 特异事务管理局（政府背景）
- 异能者协会（半官方）
- 各地分局/站点

民间组织：
- 猎人公会（赏金猎人）
- 各大异能家族
- 地下势力

威胁分类：
- F级：普通人可应对
- E-D级：需要初级异能者
- C-B级：需要中级异能者
- A级：需要高级异能者团队
- S级：需要顶级战力介入
- SS级：国家级威胁

【能力分类示例】
物理系：力量强化、速度强化、身体硬化
元素系：火、水、雷、风、土
精神系：读心、控制、幻术
特殊系：时间、空间、命运
诅咒系：献祭、契约、诅咒
```
