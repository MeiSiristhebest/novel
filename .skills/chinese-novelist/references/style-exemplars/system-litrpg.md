# 系统流/LitRPG风格范文 (System/LitRPG Style Exemplar)

> **流派定位**：以游戏化叙事为核心，通过数据面板、任务系统、升级反馈制造"叮"的爽感，是网文最具特色的本土创新流派之一。

---

## 📚 代表作品与作者

| 作品 | 作者 | 风格标签 | 核心卖点 |
|------|------|---------|---------|
| **《大王饶命》** | 会说话的肘子 | 吐槽流、负面情绪系统、日常搞笑 | 收集负面情绪值升级，骚话连篇 |
| **《全球高武》** | 老鹰吃小鸡 | 硬核数据、武道升级、热血 | 数据化武道体系，爽感十足 |

## 🎯 核心风格特征

### 1. 系统类型分类

| 类型 | 特点 | 代表作 | 核心爽点 |
|-----|-----|-------|---------|
| **属性面板型** | 力量/敏捷/智力等数值化 | 《全球高武》 | 加点升级的直观成长 |
| **任务驱动型** | 主线/支线/隐藏任务 | 《无限恐怖》 | 任务奖励的期待感 |
| **负面收集型** | 收集特殊资源升级 | 《大王饶命》 | 独特的升级方式 |
| **签到打卡型** | 每日签到获取奖励 | 各类签到流 | 稳定的成长预期 |
| **反向悖论型** | 想A却得到B | 《亏成首富》 | 反转的喜剧效果 |
| **恐怖诅咒型** | 系统本身是威胁 | 《从红月开始》 | 紧张感与成长并存 |

### 2. 面板设计黄金法则

```
【标准属性面板】
┌─────────────────────────────────┐
│  【角色属性】                    │
│  姓名：吕树                      │
│  等级：Lv.17 (3,240/5,000)       │
│  ─────────────────────────────  │
│  力量：24 (+3)    敏捷：31       │
│  体质：28         精神：22       │
│  ─────────────────────────────  │
│  负面情绪值：12,847              │
│  ─────────────────────────────  │
│  技能：                          │
│  【圣言术Lv.4】- 骚话伤害+40%    │
│  【大威天龙Lv.2】- 震慑效果      │
│  【贫穷被动Lv.MAX】- 永久生效    │
└─────────────────────────────────┘

【任务面板】
╔═══════════════════════════════════╗
║  【主线任务】调查学校异常事件     ║
║  难度：B级  时限：7天             ║
║  进度：[████████░░] 80%          ║
║  ─────────────────────────────── ║
║  奖励：经验×3000 | 技能点×2       ║
║  隐藏：？？？（完美完成解锁）     ║
╚═══════════════════════════════════╝

【系统提示】
┌──────────────────────────────────┐
│ 【叮！】                         │
│ 检测到来自"王小明"的负面情绪！   │
│ 情绪类型：愤怒+嫉妒              │
│ 收益：负面情绪值+147             │
│ 当前总计：12,847 (+147)          │
└──────────────────────────────────┘
```

**设计原则**：
- 面板要简洁，不能喧宾夺主
- 一章最多3-4次面板弹出
- 关键时刻才出现（升级、获得奖励、发现隐藏）
- 数字变化要"可感知"（不是+1，而是+147）

### 3. 系统人格化设计

| 系统性格 | 说话风格 | 适合题材 |
|---------|---------|---------|
| **毒舌型** | "宿主的智商令本系统担忧" | 搞笑、吐槽流 |
| **冷漠型** | "任务失败，宿主将被抹杀" | 恐怖、生存流 |
| **傲娇型** | "才、才不是特意帮你的！" | 轻松、后宫流 |
| **话痨型** | "恭喜恭喜恭喜！大喜事啊宿主！" | 日常、搞笑流 |
| **无感情型** | "检测完成。威胁等级：S。" | 硬核、数据流 |

## ✍️ 经典片段示范

### 【负面情绪收集】模仿《大王饶命》风格

```
"吕树，你给我站住！"

王小明气势汹汹地拦在吕树面前，身后还跟着几个狐朋狗友。

"你是不是在背后说我坏话？"

吕树一脸无辜："我？我什么时候说你坏话了？"

"少装蒜！李晓燕都告诉我了！"

吕树眨眨眼："哦，你说那个啊……"

"那个什么？！"

"我只是说……"吕树露出一个真诚的微笑，"你这种人，真的很少见。"

王小明愣了一下："……嗯？"

"我从来没见过，"吕树叹了口气，"有人能把愚蠢和自大结合得这么完美。"

"你说什么？！"

【叮！检测到来自王小明的负面情绪——愤怒】
【负面情绪值+324】

吕树的表情依然真诚："而且我也很佩服你的勇气。"

"勇气？"王小明警惕起来。

"对啊，明明长成这样，还敢出门。这份勇气，我真的自愧不如。"

【叮！检测到来自王小明的负面情绪——暴怒+羞耻】
【负面情绪值+512】

王小明的脸涨成了猪肝色："吕树！你给我去死！"

他挥拳打来——

然后，手腕被吕树轻轻握住。

"动手？"吕树歪了歪头，"你确定？"

他的眼神突然变了。

不再是那个嬉皮笑脸的吕树，而是一个……让人本能感到恐惧的存在。

王小明的拳头僵在半空中，冷汗瞬间湿透了后背。

三秒后，吕树松开手，笑容重新挂上脸。

"开玩笑的啦，同学之间打打闹闹很正常嘛。"

他拍拍王小明的肩膀，转身离去。

【叮！检测到来自王小明的负面情绪——恐惧+屈辱+愤怒（复合情绪）】
【负面情绪值+1024！】
【提示：复合情绪触发双倍加成！】

吕树嘴角微微上扬。

今天的收成，不错。
```

**风格要点**：
- 骚话要精准，直击痛点
- 系统提示配合情绪节奏
- 反差萌（嬉皮笑脸+突然认真）
- 数字递增制造爽感（324→512→1024）

### 【以弱胜强的系统战斗】标准写法

```
林凡看着面前的三阶异兽，咽了口唾沫。

【警告！检测到敌对生物】
┌────────────────────────┐
│ 名称：暗影豹            │
│ 等级：Lv.23（三阶巅峰） │
│ 威胁评估：极高          │
│ 建议：立即撤离          │
└────────────────────────┘

"撤离？"林凡苦笑，"它堵在我和出口之间。"

【提示：宿主当前等级Lv.15，与目标存在8级差距】
【战斗胜率：2.7%】
【建议：使用技能「观察」获取更多信息】

林凡深吸一口气，默念技能名。

眼前的世界变了。

暗影豹的身上浮现出淡淡的光点——大部分是红色，代表危险区域。但在它的左后腿位置，有一个微弱的黄色光点。

【「观察」触发！发现弱点！】
┌────────────────────────────────┐
│ 弱点：左后腿旧伤（未完全愈合） │
│ 效果：防御-40%，命中可致跛     │
│ 额外信息：目标处于饥饿状态     │
│          攻击欲望↑ 持久力↓    │
└────────────────────────────────┘

林凡眼睛一亮。

打不过，但可以耗。

暗影豹似乎感受到了什么，眸中凶光一闪，身形化作残影扑来——

【战斗开始！】

林凡侧身躲过第一击，刀光闪烁，在豹子腿上划出一道浅浅的伤口。

【造成伤害：17点】
【暗影豹剩余生命值：583/600】

"才17点……"他暗骂，但手上动作不停。

五分钟后。

林凡浑身是血，喘着粗气靠在树上。

【宿主剩余生命值：34/150】
【警告：生命值过低！】
【暗影豹剩余生命值：127/600】

暗影豹也好不到哪去，左后腿已经无法着地，正用仇恨的目光盯着他。

"就差一点……"

暗影豹低吼一声，准备做最后一击——

【紧急提示！】
【检测到宿主触发隐藏成就条件！】
【「以弱胜强」：在生命值低于25%的情况下，击杀比自己高5级以上的敌人】
【是否激活一次性技能「孤注一掷」？】
【效果：消耗50%当前生命，下一击伤害×500%】
【警告：使用后将进入虚弱状态，持续30分钟】

"确认！"

刹那间，一股力量涌入四肢百骸。

【生命值：34 → 17】
【「孤注一掷」激活！】
【下一击伤害×500%】

暗影豹扑来——

林凡没有躲。

他迎上去，手中的刀精准地刺入豹子左后腿的旧伤位置。

【暴击！弱点命中！】
【伤害计算：(45×5)×1.4(弱点加成) = 315点！】
【暗影豹剩余生命值：0/600】
【击杀成功！】

暗影豹的动作僵住了。

它难以置信地低头看了看胸口——刀刃贯穿了它的心脏。

下一秒，庞大的身躯轰然倒地。

【叮！恭喜宿主击杀三阶异兽「暗影豹」！】
【经验值+2400！】
【等级提升！Lv.15 → Lv.17！】
【获得技能点×3！】
【掉落物品：暗影豹核晶（稀有）×1】
【隐藏成就解锁：以弱胜强！】
【成就奖励：属性点+5，称号「越级杀手」】

林凡瘫倒在地，大口喘息。

【宿主剩余生命值：17/180（已更新）】
【提示：宿主当前处于虚弱状态，建议尽快寻找安全区域休息】

"系统……"

【宿主有何吩咐？】

"下次……能不能早点提醒我有这技能？"

【……建议宿主仔细阅读技能说明，而非临阵磨枪。】

"你这是在教训我？"

【陈述事实。】

林凡翻了个白眼，但嘴角还是忍不住上扬。

赢了。

狗系统虽然毒舌，但还挺靠谱的。
```

### 【升级仪式感】多巴胺轰炸写法

```
最后一只异兽倒下。

林凡站在尸体中间，浑身浴血，但眼中满是兴奋。

三天，整整三天没合眼。

但值了。

【叮！】

来了！

【恭喜宿主击杀二阶异兽「血蝠」×127！】
【累计经验值+15,240！】

【等级提升！】
【Lv.23 → Lv.24 → Lv.25 → Lv.26！】
【连升三级！】

金色的光芒从林凡体内迸发，伤口以肉眼可见的速度愈合。

【属性提升！】
【力量：47 → 56 (+9)】
【敏捷：52 → 61 (+9)】
【体质：44 → 53 (+9)】
【精神：38 → 44 (+6)】

【获得自由属性点×15！】
【获得技能点×6！】

【新技能解锁！】
【「血气回溯Lv.1」——击杀敌人可回复5%生命值】

【称号更新！】
【「血蝠克星」——对蝙蝠类异兽伤害+30%】

【隐藏奖励触发！】
【检测到宿主连续击杀100+同类异兽】
【解锁成就：屠夫！】
【奖励：「杀戮本能」被动技能（攻击附带5%流血效果）】

【物品结算中……】
【血蝠核晶×127】
【血蝠皮翼×89】
【血蝠毒牙×234】
【稀有掉落：血蝠心脏×3】

【总价值估算：约37,000积分】

林凡看着眼前密密麻麻的系统提示，忍不住仰天大笑：

"哈哈哈哈哈！"

【提示：建议宿主保持冷静，附近可能还有其他威胁。】

"让我爽一会儿怎么了？"

【……】

【宿主请自便。】
```

**风格要点**：
- 连续弹窗制造"多巴胺轰炸"
- 数字要大、要具体
- 升级要有仪式感（金光、伤口愈合）
- 多种奖励叠加（等级+属性+技能+成就+物品）

## 📝 模仿指令

> **给 AI 的指令**：
> 
> "请以《大王饶命》《全球高武》的笔触写作本章。注意以下要点：
> 1. **面板节制**：面板是调味品不是主菜，关键时刻才弹出
> 2. **数字爽感**：让数值变化可感知（+324而非+1）
> 3. **系统人格**：给系统一点性格（毒舌/冷漠/傲娇）
> 4. **升级仪式**：升级时配合一连串奖励轰炸
> 5. **困境先行**：先让主角数值劣势，再用技巧逆转
> 6. **吐槽互动**：主角与系统的对话要有梗"

## ⚠️ 常见雷区

| 雷区 | 错误示范 | 正确示范 |
|------|---------|---------| 
| 面板太多 | 每段都有系统提示 | 一章3-4次，关键节点才出现 |
| 数字太小 | "经验值+5" | "经验值+1,247" |
| 系统无性格 | "任务完成。" | "任务完成。宿主的表现勉强及格。" |
| 升级太随意 | "他升级了" | 金光+数据刷屏+成就解锁 |
| 无视等级差 | 主角轻松越级杀敌 | 越级需要付出代价（重伤/消耗底牌） |

## 🔧 系统设计模板

```
【系统基础框架】
1. 来源：（穿越赠送/意外获得/家族传承/神秘实体）
2. 功能：（属性面板/任务系统/商城/抽奖/签到）
3. 升级货币：（经验值/积分/负面情绪/气运值）
4. 限制：（每日次数/冷却时间/副作用）
5. 人格：（有无意识/性格类型/与宿主关系）

【任务设计公式】
任务 = 目标 + 时限 + 难度 + 基础奖励 + 隐藏奖励
例：
"调查废弃工厂（72小时/B级）
 基础：经验×2000
 隐藏：完美完成解锁SSR技能"
```
