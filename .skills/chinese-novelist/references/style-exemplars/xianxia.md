# 仙侠流风格范文 (Xianxia Style Exemplar)

> **流派定位**：东方仙侠，强调"修仙求道"的漫长旅程，比玄幻更注重**氛围**、**意境**和**道心**的刻画。

---

## 📚 代表作品与作者

| 作品 | 作者 | 风格标签 | 核心特色 |
|------|------|---------|---------|
| **《凡人修仙传》** | 忘语 | 凡人流开创者、谨慎算计、资源争夺 | 没有主角光环，只有步步为营 |
| **《仙逆》** | 耳根 | 古典仙侠、逆天改命、悟道 | "顺为凡，逆则仙" |
| **《一念永恒》** | 耳根 | 搞笑+热血、惜命、怕死 | 最怕死的主角，最热血的故事 |
| **《诛仙》** | 萧鼎 | 经典仙侠、悲剧爱情、正邪之辩 | "天地不仁，以万物为刍狗" |
| **《遮天》** | 辰东 | 帝路争锋、宿命轮回、荒古传说 | 成帝之路的孤独与悲壮 |
| **《我师兄实在太稳健了》** | 言归正传 | 稳健流、苟道、布局千年 | "能苟就苟，苟到最后" |

## 🎯 核心风格特征

### 1. 境界体系（仙侠标配）
仙侠流的境界体系强调"修心"与"悟道"，不只是战力提升。

**常见模板**：
```
练气 → 筑基 → 金丹 → 元婴 → 化神 → 炼虚 → 合体 → 大乘 → 渡劫 → 仙人
                                                              ↓
                                              真仙 → 金仙 → 大罗 → 混元
```

**凡人流特色**：每个境界都需要"资源"（灵石、丹药、功法），突破失败会倒退甚至陨落。

### 2. 凡人流核心公式
```
【匮乏】资源不足 + 天赋平庸 + 敌人觊觎
    ↓
【算计】主角谋划 + 借力打力 + 隐藏底牌
    ↓
【收获】险中求胜 + 获取资源 + 敌人陨落（但代价惨重）
```

### 3. 语言特点
- **冷静克制**：少用感叹号，多用陈述句
- **内心独白**：大量心理活动，展现算计过程
- **古风用词**：适度使用文言词汇（"此人""莫非""竟是"）
- **留白艺术**：不把话说满，给读者想象空间

## ✍️ 经典片段示范

### 【凡人流生存】模仿《凡人修仙传》风格

```
韩立看着手中仅剩的三颗筑基丹，眉头紧锁。

按照他的估算，以他中等偏下的灵根资质，突破筑基期至少需要五颗。而黄枫谷的丹药配给，每年只有一颗。

也就是说，他还需要等两年。

两年后，他已经四十三岁了。

四十三岁的练气期修士，在修仙界几乎等同于废人。没有宗门愿意培养一个即将寿终的老头。

"必须另想办法。"

韩立闭上眼睛，开始回忆近三年来收集的情报。

乱星海有一处秘境，据说产出天灵草……但那里是化神期老怪的地盘。

天南有一场交易会，偶尔会有筑基丹出售……但价格是他全部身家的三倍。

还有一个办法——

杀人夺宝。

这个念头一闪而过，韩立并没有立刻否定。

在修仙界，杀人夺宝是再正常不过的事。问题在于，他现在的实力，杀不了任何一个值得杀的人。

"先苟着。"

韩立睁开眼睛，目光恢复了平静。

他还有两年时间。两年之内，机会总会出现的。

如果没有，那就创造机会。
```

**风格要点**：
- 用数字量化困境（"三颗""五颗""两年""四十三岁"）
- 展现主角的冷静思考和算计过程
- 不回避阴暗面（"杀人夺宝"）但也不美化
- 结尾留有余地，体现"苟"的精神

### 【悟道场景】模仿《仙逆》风格

```
王林盘坐于枯井之底，已是第七日。

七日前，他亲眼看着师门被灭，同门尽死，自己却只能躲在井底苟延残喘。

"我太弱了。"

这个念头像毒蛇一样啃噬着他的心。

修仙界没有对错，只有强弱。弱者，就是原罪。

他闭上眼睛，试图参悟那部残缺的功法。但心中的愤怒和悲伤如潮水般涌来，让他根本无法静心。

"为什么？"

"凭什么？"

"天道不公！"

他猛然睁眼，眼中满是血丝。

就在这一刻，一道闪电划过他的脑海。

不是天道不公。

是他太弱。

弱者求天，强者逆天。

"顺为凡……"

他喃喃自语，声音沙哑。

"逆则仙。"

枯井之中，一道微弱的灵光亮起。那是属于他的道。

从今日起，王林不再怨天。

他要逆天。
```

**风格要点**：
- 用极端困境逼迫主角"悟道"
- 心理挣扎要真实（愤怒、悲伤、自我怀疑）
- 悟道的瞬间要有"顿悟感"（闪电、灵光）
- 用一句话总结"道"（"顺为凡，逆则仙"）

### 【搞笑+热血】模仿《一念永恒》风格

```
"长生……长生……"

白小纯蹲在角落里，瑟瑟发抖，嘴里念念有词。

他刚刚亲眼看到一个师兄在修炼时走火入魔，当场爆体而亡。那血肉横飞的场面，让他连续三天吃不下饭。

"修仙太可怕了！"

"动不动就要死人！"

"我不修了！我要回村子种地！"

然而，当他想起村子里那个追着他要嫁给他的胖丫头，白小纯又缩了回来。

"还是修仙吧……"

他咬了咬牙，拿起功法继续修炼。

就在这时，一声惊雷在他耳边炸响。

"白小纯！你偷的鸡呢？"

白小纯吓得一个激灵，功法在手中一抖——

轰！

一股灵气从他体内爆发，将面前的木门直接轰成了碎片。

"我……我突破了？"

白小纯愣在原地，一脸懵逼。

门外，被碎片糊了一脸的大师兄，表情比他还懵逼。

"……你小子是故意的吧？"

"不是！真不是！我发誓！"

白小纯疯狂摆手，眼泪都快出来了。

他真的只是被吓到了啊！

谁能想到被吓一下就能突破？

这算什么？受惊突破法？
```

**风格要点**：
- 主角要"怂"但不能"废"
- 用反差制造笑点（怕死却偏偏要修仙）
- 突破要有戏剧性（意外、搞笑）
- 对话要口语化，节奏要快

## 📝 模仿指令

> **给 AI 的指令**：
> 
> "请以《凡人修仙传》《仙逆》的笔触写作本章。注意以下要点：
> 1. **冷静克制**：少用感叹号，用陈述句
> 2. **算计过程**：展现主角的思考和权衡
> 3. **资源意识**：灵石、丹药、功法都是稀缺资源
> 4. **生存压力**：随时可能死，没有主角光环
> 5. **古风用词**：适度使用文言词汇，但不要过度"

> **给 AI 的指令（搞笑向）**：
> 
> "请以《一念永恒》的笔触写作本章。注意以下要点：
> 1. **主角要怂**：怕死、惜命、能苟就苟
> 2. **反差笑点**：越怕死越能活，越想低调越高调
> 3. **对话口语化**：像现代人说话
> 4. **热血时刻**：关键时刻要燃起来，反差拉满"

## ⚠️ 仙侠 vs 玄幻 对比

| 维度 | 仙侠流 | 玄幻流 |
|------|-------|-------|
| 节奏 | 慢热，铺垫长 | 快节奏，三章一小高潮 |
| 主角 | 谨慎、算计、步步为营 | 热血、霸气、一路碾压 |
| 战斗 | 重策略、重底牌 | 重场面、重震撼 |
| 境界 | 突破困难，需要机缘 | 突破相对容易，主要靠资源 |
| 情绪 | 压抑、隐忍、厚积薄发 | 爽快、直接、即时反馈 |
| 氛围 | 修仙界残酷无情 | 热血燃向、正邪分明 |
