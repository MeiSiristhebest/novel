# 历史/穿越风格范文 (Historical/Time Travel Style Exemplar)

> **流派定位**：以历史为舞台，现代灵魂为主角，通过知识降维、权谋博弈、蝴蝶效应展现"改变历史"的爽感与思辨。

## 🎯 核心风格特征

### 1. 朝代选择与特色

| 朝代 | 核心冲突 | 常见金手指 | 代表作 |
|-----|---------|----------|-------|
| **秦** | 统一vs六国、法治vs人性 | 军功爵制、兵法 | 《秦吏》 |
| **汉** | 外戚vs皇权、匈奴边患 | 造纸术、马镫 | 《覆汉》 |
| **唐** | 藩镇割据、胡汉融合 | 炒钢法、火药 | 《唐砖》 |
| **宋** | 变法之争、靖康之耻 | 经济学、军事改革 | 《新宋》《宰执天下》 |
| **明** | 党争、倭寇、后金崛起 | 商业、海贸、火器 | 《回明》《官居一品》 |
| **清** | 闭关锁国、列强叩关 | 工业革命、维新 | 《篡清》 |

### 2. 穿越适应期写法

```
【标准开局公式】
穿越瞬间（懵逼+身体不适）
    ↓
确认身份（翻记忆/照镜子/问仆人）
    ↓
了解处境（什么朝代/什么身份/有什么危机）
    ↓
制定计划（先活下来/再谋发展）
    ↓
第一次装逼（用现代知识解决眼前危机）
```

### 3. 语言风格分层

| 场景 | 语言风格 | 示例 |
|-----|---------|------|
| **旁白叙述** | 现代白话 | "他知道，这是自己改变历史的第一步。" |
| **主角内心** | 现代吐槽 | "我去，这不是嘉靖皇帝吗？历史书上的暴君啊！" |
| **主角对话** | 半文半白 | "学生不才，愿为大人分忧。" |
| **古人对话** | 符合身份 | 皇帝："朕意已决。" 百姓："官爷饶命啊！" |
| **奏折文书** | 文言文 | "臣惶恐，伏惟圣裁。" |

## ✍️ 经典片段示范

### 【穿越开局】标准写法

```
李牧睁开眼睛的时候，第一反应是——头疼。

不是普通的疼，是那种宿醉后被人用锤子敲了一夜的疼。

他试图抬起手揉太阳穴，却发现自己的手……不太对劲。

太细了。

而且，没有那道三年前骑摩托车摔的疤。

他猛地坐起身，四下打量。

这是哪？

昏暗的光线，粗糙的土墙，空气中弥漫着一股潮湿和霉味。身下是一张硬得硌骨头的木板床，身上盖着一床破旧的棉被。

"这是……"

"少爷醒了！"

一个扎着双髻的小丫头跑了进来，手里端着一碗黑乎乎的药汤。

少爷？丫鬟？

李牧的大脑飞速运转。

"我……这是哪里？"

小丫鬟一脸担忧："少爷，您昏迷三天了，可把老爷夫人急坏了。大夫说您是落水受了惊吓……少爷，您没事吧？"

落水？受惊？

一股陌生的记忆涌入脑海——

他叫李牧，不对，现在应该叫……李文远。

建文四年，靖难之役的最后一年。

他爹是建文帝的忠臣，方孝孺的学生。

而再过三个月，燕王朱棣就要攻破南京城。

他爹……会被诛十族。

李牧——不，李文远——感觉那碗药汤突然变得烫手。

完蛋了。

穿越到明朝不可怕，可怕的是穿越到一个即将被灭门的家族里。

而且还是方孝孺的学生家。

十族啊！历史上唯一一个被诛十族的！

"少爷？少爷您脸色怎么这么白？"

李文远挤出一个比哭还难看的笑容："没事……就是有点饿。"

他得想办法活下去。

至少，得在三个月之内，把全家人弄出南京城。
```

**风格要点**：
- 先写身体感受（头疼、手不对）
- 通过丫鬟/仆人确认身份
- 记忆涌入的设定
- 立刻点明历史危机（靖难之役、诛十族）
- 主角内心吐槽增加代入感

### 【朝堂博弈】权谋名场面

```
奉天殿内，死一般的寂静。

嘉靖皇帝端坐在龙椅上，手里把玩着一串沉香念珠，目光似笑非笑地扫过群臣。

"严阁老，这江南织造局的账目，你怎么看？"

严嵩出列，颤巍巍地跪下。他今年七十有三，在这朝堂上浮沉了半辈子，早已修炼得喜怒不形于色。

"回陛下，老臣以为……此事还需详查。"

"详查？"嘉靖冷笑，"查了三年，从南京查到苏州，从苏州查到杭州。三百万两银子，就这么不翼而飞了？"

站在武官队列里的徐渭，低着头，心跳如鼓。

他知道，这是严党和清流的又一次交锋。

而他手里的那份证据——如果拿出来，就是滔天大案。

但如果不拿出来……

"陛下，"一个声音突然响起，"臣有本奏。"

是徐阶。

内阁次辅，严嵩最大的政敌。

徐阶从袖中取出一份奏折，双手呈上："臣偶然查得，这三百万两银子的去向，与……东南倭寇有关。"

朝堂上顿时响起一阵嗡嗡的议论声。

严嵩的眼睛眯了起来。

徐渭心中一凛。

倭寇？这个帽子扣得太大了。一旦坐实，就是通敌叛国的死罪。

"证据呢？"嘉靖问。

"臣已命人查抄了织造局主事王文远的府邸，"徐阶不卑不亢，"在他的书房密室中，找到了与倭寇首领汪直的通信。"

严嵩的脸色终于变了。

王文远，是他的人。

"陛下，"严嵩急忙开口，"此事蹊跷，恐有人构陷……"

"构陷？"嘉靖打断他，声音陡然冷了下来，"严阁老的意思是，朕的锦衣卫，会构陷朕的臣子？"

严嵩浑身一震，立刻伏地叩首："老臣不敢！老臣绝无此意！"

嘉靖没有说话，只是静静地看着他。

那目光，像蛇一样阴冷。

良久。

"此事，交由三法司会审。"嘉靖站起身，"严阁老年事已高，就不必操劳了。"

他拂袖而去，留下满殿惊恐的群臣。

严嵩跪在地上，久久没有起身。

他知道，这一局，他输了。

而这，仅仅是开始。

徐渭悄悄攥紧了袖中的那份证据。

看来，还不到他出手的时候。

这场棋局，才刚刚开始。
```

**风格要点**：
- 皇帝要有威压，话不多但字字诛心
- 两派博弈要有来有回
- 主角先观察，不要急于出手
- 结尾留悬念

### 【技术降维】现代知识的正确用法

```
"陈公子，你说的这个……水泥，当真能比夯土还坚固？"

工部主事张大人一脸不信，盯着面前那滩灰白色的糊状物。

陈安微微一笑："大人若不信，可亲眼验证。"

他让工匠将水泥浆倒入木模，又在旁边用传统的夯土法筑了一块同样大小的土墙。

"三日后，大人再来看。"

三日后。

张大人带着一群工部官员，浩浩荡荡地来到工地。

陈安递给他一把锤子："大人请。"

张大人接过锤子，先对着那块夯土墙砸了下去。

砰！

一个拳头大的坑。

"还行。"张大人点点头，转向那块水泥墙。

他举起锤子，用力砸下。

当！

火星四溅。

张大人虎口发麻，锤子差点脱手。

而那块水泥墙——

纹丝不动。

连个白印都没有。

"这……这是何方神物？！"张大人瞪大了眼睛。

陈安不紧不慢地解释："此物名曰水泥，以石灰石、黏土烧制而成。凝固后坚硬如石，可用于筑城、修路、建桥。"

"最重要的是——"

他指了指那些原材料："这些东西，到处都有。成本不到糯米浇筑的十分之一。"

张大人沉默了。

他做了二十年的工部主事，太清楚这意味着什么了。

边关的城墙，年年被鞑子攻破，年年修，年年塌。

若有了这水泥……

"陈公子，"张大人一把抓住陈安的手，声音都在颤抖，"这水泥的方子……"

"大人放心，"陈安笑道，"学生既然拿出来，就没想过藏私。"

"只是……"

他话锋一转，目光变得意味深长：

"学生有个不情之请。"

张大人愣了一下，随即恍然大悟。

"陈公子但说无妨！"

官场不就是这样吗？

有来有往，各取所需。

这位年轻人，看来是深谙此道啊。
```

### 【古人智慧】不要小看古人

```
沈默以为自己稳了。

他拿出的是五百年后的经济学理论——供需关系、通货膨胀、货币政策。

在这个连纸币都没有的朝代，他就是神。

然而……

"沈公子的意思，老夫听明白了。"

王安石放下茶盏，目光平静得可怕。

"你说的青苗法改良，核心是降低利息，扩大放贷范围，对否？"

"对。"沈默自信满满，"这样可以打击豪强的高利贷，让百姓有喘息之机。"

"想法是好的。"王安石点点头，"但你忽略了一件事。"

"什么？"

"执行者。"

王安石站起身，走到窗前，望着窗外的汴京城。

"你设计的利息再低，到了地方官手里，会变成什么样？"

"为了完成朝廷的放贷指标，他们会强行摊派。不需要借的也要借，借了还不起的就抄家。"

"本来是救命的钱，最后变成催命的刀。"

沈默张了张嘴，却发现自己无话可说。

他想到了经济模型，想到了激励机制，却唯独忘了——

这是封建社会。

官员的KPI不是百姓的幸福指数，而是上级的满意程度。

"年轻人，"王安石转过身，目光复杂，"你的学问，老夫自愧不如。"

"但老夫在这官场上混了三十年，看过太多好经被和尚念歪。"

"任何政策，设计只占三分，执行才是七分。"

"而执行……"

他叹了口气：

"靠的是人。贪婪的、自私的、趋利避害的人。"

沈默沉默了。

他第一次意识到，穿越者最大的敌人，不是历史惯性，不是技术壁垒。

而是人性。

五百年前的人性，和五百年后，并没有什么不同。
```

## 📝 模仿指令

> **给 AI 的指令**：
> 
> "请以《庆余年》《秦吏》的笔触写作本章。注意以下要点：
> 1. **称谓规范**：对皇帝称'臣'，对上官称'卑职'，对平辈称'兄台'
> 2. **语言分层**：旁白现代，内心吐槽，对话半文半白
> 3. **装逼铺垫**：先写古人的质疑或困境，再反转打脸
> 4. **尊重古人**：不要把古人当傻子，论权谋他们是祖宗
> 5. **环境细节**：烛火、更漏、熏香、马蹄声等古风意象
> 6. **引经据典**：适当引用诗词典故，但要服务于剧情"

## ⚠️ 常见雷区

| 雷区 | 错误示范 | 正确示范 |
|------|---------|---------| 
| 现代词汇乱入 | "OK，这个proposal我同意" | "好，此事就依先生所言" |
| 技术开挂 | 穿越第一天造出蒸汽机 | 花三年时间培养工匠、积累材料，才造出初代蒸汽机 |
| 古人太蠢 | 用小学知识碾压丞相 | 现代知识有优势，但古人的政治智慧让主角吃亏 |
| 历史常识错误 | 唐朝吃辣椒（辣椒明朝才传入） | 查证时代背景，注意食物/器物的时代性 |
| 没有代价 | 主角随便改变历史，毫无阻力 | 每次改变都有反噬，蝴蝶效应难以控制 |
