# 武侠风格范文 (Wuxia Style Exemplar)

> **流派定位**：江湖恩怨，刀光剑影，侠义精神，快意恩仇。以"侠"为核心，武功为载体，书写江湖儿女的爱恨情仇。

## 🎯 核心风格特征

### 1. 江湖世界观（必备框架）

```
【庙堂】朝廷、皇权、锦衣卫、东厂
    ↓↑ 对抗/合作
【江湖】门派、帮派、世家、散人
    
【正道】少林、武当、峨眉、名门正派
    vs
【邪道】魔教、邪派、杀手组织
```

**门派势力设定模板**：
```
门派名 + 所在地 + 绝学 + 特点 + 代表人物
例：华山派 + 华山 + 紫霞神功 + 君子剑/剑宗气宗之争 + 令狐冲
```

### 2. 武功体系（核心设定）

**内功层次**：
```
入门 → 小成 → 大成 → 登峰造极 → 返璞归真
```

**武功类型**：
| 类型 | 特点 | 经典例子 |
|------|------|---------|
| 内功心法 | 根基，决定上限 | 九阳神功、易筋经 |
| 剑法 | 最常见，变化多端 | 独孤九剑、辟邪剑谱 |
| 刀法 | 大开大合，霸道刚猛 | 降龙十八掌（掌法） |
| 轻功 | 身法，逃命/追击 | 凌波微步、梯云纵 |
| 暗器 | 防不胜防 | 满天花雨、暴雨梨花针 |
| 奇门 | 毒功、蛊术、幻术 | 五毒神功、化功大法 |

### 3. 侠义精神（哲学内核）

**金庸式**："侠之大者，为国为民"
- 郭靖：守襄阳，死守到底
- 萧峰：为两国和平，自尽于雁门关

**古龙式**："人在江湖，身不由己"
- 李寻欢：明知不可为而为之
- 陆小凤：浪子本色，重情重义

**新派式**：侠义与现实的冲突
- 徐凤年：用手中刀，护住想护的人
- 陈平安：道理是道理，规矩是规矩

### 4. 语言特点

| 风格 | 特点 | 代表 |
|------|------|------|
| **金庸体** | 典雅厚重，史诗感 | 长句，引经据典 |
| **古龙体** | 短句，留白，诗意 | 一句一段，意境深远 |
| **新派** | 热血激昂，节奏快 | 对白犀利，场面宏大 |

## ✍️ 经典片段示范

### 【金庸风】大战场面

```
萧峰双掌齐出，降龙十八掌中的"亢龙有悔"轰然击出。

掌风所至，落叶纷飞，劲气激荡，方圆三丈内无人能近。

慕容复长剑一抖，剑身化作万点寒星，正是姑苏慕容家的绝学"参合指"化入剑招。以彼之道，还施彼身——他竟要以剑气硬接萧峰的掌力！

"铮！"

剑气与掌力相撞，发出惊天动地的巨响。

慕容复连退七步，每一步都在青石板上踏出一个深深的脚印。他虎口震裂，鲜血顺着剑柄滴落。

萧峰却只退了一步。

"慕容公子，"萧峰沉声道，"萧某敬你是条汉子。今日之事，到此为止。你我各走各路，如何？"

慕容复惨笑一声。

"萧峰，你可知我慕容家为何要复国？"他缓缓举起长剑，"我家数代先祖，无一日不思故土。纵然身死，也要完成遗愿！"

"今日——不是你死，就是我亡！"

说罢，慕容复身形暴涨，长剑幻出千百道剑影，如暴风骤雨般向萧峰席卷而去。

萧峰长叹一声。

他双掌缓缓抬起，体内真气运转到极致，丹田处如有雷鸣滚动。

降龙十八掌，第十八掌——

"神龙摆尾！"
```

**风格要点**：
- 武功名称明确（降龙十八掌、参合指）
- 招式对决，有来有往
- 掌力/剑气可视化描写
- 对白揭示人物背景与动机
- 最后一招用于收尾，制造高潮

### 【古龙风】高手对决

```
月色很好。

刀光也很好。

傅红雪站在长街的这一头，刀还没有出鞘。

他的手在抖。

这是一种病，一种无法治愈的病。每当他紧张的时候，他的手就会抖。

但他依然是天下第一快刀。

因为他出刀的时候，手不会抖。

对面站着的人叫叶开。

叶开在笑。

他的笑容很温暖，像三月的春风。没有人会相信，这个笑起来如此温暖的人，手中的那把刀，曾经杀过三十七个人。

"你不该来的。"傅红雪说。

"我知道。"叶开说。

"但你还是来了。"

"因为她在这里。"

傅红雪沉默了。

他当然知道叶开说的"她"是谁。

"你以为你能带走她？"

"我不知道。"叶开的笑容淡了一些，"但我想试试。"

风吹过长街，卷起一片枯叶。

枯叶落地的瞬间——

刀光亮了。

没有人看清楚谁先出的刀。只有两道白芒，在月光下交织，碰撞，然后分开。

傅红雪的刀，停在叶开的咽喉前三寸。

叶开的刀，停在傅红雪的心口前两寸。

"你变快了。"傅红雪说。

"你变慢了。"叶开说。

他们同时收刀。

"下次再来。"叶开转身走进黑暗中。

傅红雪没有追。

他低头看了看自己的手。

依然在抖。
```

**风格要点**：
- 短句、碎片化叙事
- 大量留白，让读者想象
- 人物特征鲜明（傅红雪的抖，叶开的笑）
- 点到为止，不写透
- 意境>动作，氛围>细节

### 【新派风】热血豪情

```
徐凤年提刀上马，回首望向身后的五千白马义从。

这些人，有前朝遗民，有边关孤儿，有落魄游侠，有铁匠猎户。

他们来自五湖四海。

但此刻，他们只有一个名字——

北凉！

"弟兄们！"徐凤年的声音在风中回荡，"今日北莽三十万铁骑压境，朝廷见死不救，天下人都在看我们笑话！"

"他们说，北凉完了！"

"他们说，徐家完了！"

"他们说——咱们都得死在这里！"

五千白马义从沉默不语，但每个人的眼中都燃烧着火焰。

徐凤年一字一顿道："那我就让他们看看，什么叫——北凉铁骑！"

他高高举起手中的刀，那是一把普普通通的刀，没有名字，没有来历。

但在这一刻，它就是天下最锋利的刀。

"北凉——"

"万岁！"五千人齐声怒吼。

"杀！"

徐凤年一夹马腹，单人独骑，向着三十万大军冲去。

身后，五千白马义从紧随其后。

五千对三十万。

这是一场必败的战争。

但北凉人，从不知道"败"字怎么写。

马蹄声如雷。

刀光照亮了整片天空。
```

**风格要点**：
- 战前动员，热血沸腾
- 以少敌多，凸显悲壮
- 家国情怀，北凉vs朝廷
- 节奏快，情绪激昂
- 收尾有气势，画面感强

## 📝 模仿指令

> **给 AI 的指令**：
> 
> "请以武侠风格写作本章。注意以下要点：
> 1. **风格选择**：金庸体（典雅大气） / 古龙体（短句留白） / 新派（热血豪情）
> 2. **武功描写**：内功外功有层次，招式有名称，对决有来有往
> 3. **侠义精神**：侠之大者or人在江湖，根据故事主题选择
> 4. **江湖气息**：门派、帮派、恩怨、情仇
> 5. **情感克制**：不滥情，用行动表达，留白给读者"

## ⚠️ 常见雷区

| 雷区 | 错误示范 | 正确示范 |
|------|---------|---------|
| 武功无层次 | 主角随便一招秒杀高手 | 高手对决，各有来往 |
| 招式无美感 | "他一剑刺过去" | "剑走偏锋，剑尖如毒蛇吐信" |
| 侠义空洞 | 主角喊口号"我要行侠仗义" | 用行动证明，不用嘴说 |
| 对白太白 | "你竟然敢杀我爹！" | "二十年前的血债，今日该清了" |
| 忽视江湖 | 只有主角和反派 | 门派、帮派、朝廷、散人多方博弈 |
