# 种田/领主流范文指南 (Farming/Kingdom Building)

> **流派定义**：主角在异世界/古代/末世建设领地、发展势力、种田经营的故事

---

## 📊 流派特征

### 核心卖点

| 要素 | 说明 | 读者期待 |
|------|------|----------|
| **建设成就感** | 从无到有，逐步发展 | 看着势力壮大的满足 |
| **种田日常** | 农业、手工、科技发展 | 慢节奏、治愈感 |
| **领主经营** | 管理领地、发展经济 | 模拟经营的快感 |
| **势力扩张** | 从村庄到帝国 | 争霸爽感 |

### 细分类型

| 类型 | 特点 | 代表作品 |
|------|------|----------|
| **纯种田** | 专注农业发展 | 《重生之农门娇女》 |
| **领主建设** | 经营领地势力 | 《领主时代》 |
| **基建流** | 科技/工业发展 | 《工业霸主》 |
| **末世种田** | 末世背景生存 | 《末世之种田》 |
| **穿越种田** | 古代背景发展 | 《穿越之种田记》 |
| **星际种田** | 星际背景开拓 | 《星际种田记》 |

## 🏰 领地设定

### 领地发展阶段

```
第一阶段：生存期（1-50章）
├── 初始条件：荒地/破村/废弃领地
├── 核心任务：活下去
├── 发展重点：基础农业、住所
├── 人口规模：几十人
└── 冲突来源：野兽、饥荒、小土匪

第二阶段：发展期（51-150章）
├── 领地状态：初具规模的村镇
├── 核心任务：稳定发展
├── 发展重点：手工业、商业、防御
├── 人口规模：几百到几千人
└── 冲突来源：周边势力、商业竞争

第三阶段：扩张期（151-300章）
├── 领地状态：繁荣的城市/多个村镇
├── 核心任务：扩大影响力
├── 发展重点：军事、政治、科技
├── 人口规模：数万人
└── 冲突来源：领主争霸、政治斗争

第四阶段：争霸期（301-500章）
├── 领地状态：王国/诸侯
├── 核心任务：称王称霸
├── 发展重点：战争、外交、统一
├── 人口规模：数十万到百万
└── 冲突来源：列国争霸、终极敌人

第五阶段：帝国期（501章+）
├── 领地状态：帝国
├── 核心任务：长治久安/开拓新疆
├── 发展重点：文化、传承、扩张
├── 人口规模：百万以上
└── 冲突来源：内忧外患、终极挑战
```

### 领地系统模板

```yaml
领地名称: 青山领
领地等级: Lv.1 荒村 → Lv.10 帝国

核心数据:
  人口: 当前/上限
  粮食: 产量/消耗/储备
  金币: 收入/支出/储备
  军事: 兵力/士气/装备
  民心: 满意度/忠诚度
  
建筑系统:
  农业类: 农田、水车、谷仓...
  工业类: 铁匠铺、木工坊、纺织厂...
  商业类: 市场、商会、钱庄...
  军事类: 兵营、城墙、箭塔...
  民生类: 民居、医馆、学堂...
  特殊类: 领主府、祭坛、奇迹...

科技树:
  农业科技: 轮作、灌溉、育种...
  工业科技: 冶炼、机械、蒸汽...
  军事科技: 武器、战术、兵种...
  文化科技: 教育、法律、外交...
```

## 🌾 种田元素

### 农业发展线

```
初级农业:
├── 开荒种地
├── 基础作物（小麦、水稻）
├── 靠天吃饭
└── 自给自足

中级农业:
├── 水利灌溉
├── 优良品种
├── 肥料使用
├── 产量提升
└── 开始有余粮

高级农业:
├── 精耕细作
├── 多样化种植
├── 畜牧养殖
├── 农产品加工
└── 商业化农业

顶级农业:
├── 机械化/魔法化
├── 大规模农场
├── 农业帝国
└── 粮食控制权
```

### 经典种田桥段

```yaml
开局艰难:
  - 一穷二白，连吃饭都成问题
  - 第一块田的收获
  - 第一笔收入
  - 第一批追随者

发明创造:
  - 引入新作物（土豆、玉米）
  - 改良农具
  - 发明水车/风车
  - 发现矿藏

商业发展:
  - 第一个爆款产品
  - 建立商队
  - 垄断某种资源
  - 富甲一方

人才招募:
  - 发现能人
  - 说服人才加入
  - 建立班底
  - 人才济济
```

## ✍️ 语言风格

### 种田文风

```
数据化描写:
"【系统提示】
农田Lv.1升级为农田Lv.2
粮食产量+20%
当前粮食产量：120石/月"

或者不用系统的写法:
"秋收结束，账房先生呈上了今年的收成报告。
'大人，今年共收粮三千石，比去年增产两成！'
李青满意地点点头，这两年推行的新耕作法终于见效了。"

建设成就感:
"站在城墙上，李青望着眼前的景象，心中涌起一股自豪。
三年前，这里还是一片荒地。
三年后，这里已是千人大镇，商队往来，百业兴旺。
而这一切，都是他一手建立的。"

规划描写:
"'第一年，我们要开垦一百亩农田，解决温饱问题。'
李青在地图上画了一个圈，'第二年，建立铁匠铺和木工坊。
第三年，招兵买马，建立正规军队。
五年内，我要让这里成为方圆百里最繁华的城镇！'"
```

### 发展爽感

```
从无到有:
"【一年前】
破败的茅草屋，十几个衣不蔽体的流民。
【一年后】
整齐的砖瓦房，繁忙的集市，来往的商队。
曾经嘲笑他的人，现在都巴不得成为他的子民。"

碾压差距:
"邻郡的刘太守终于亲自来了。
他原本是来'指导'这个穷乡僻壤的，
然而当他看到整齐的街道、繁华的商业、
装备精良的士兵时，脸色变得精彩极了。
'这...这怎么可能？'"
```

## 📝 范文示例

### 开篇示例

```
【第一章 破落领地】

李青站在所谓的"领主府"前，沉默了。

这是一座破旧的木屋，屋顶的茅草已经枯烂，
墙壁上的裂缝大到能伸进去一只手。

这就是他穿越到异世界后，继承的全部家业。

"少爷..."老管家颤颤巍巍地走过来，
"这是领地的全部账册...您看看吧。"

李青接过那本薄薄的册子，翻开一看：

人口：43人（老弱居多）
可耕地：50亩（荒废30亩）
粮食储备：够吃两个月
金币：5枚
军事力量：无

他深吸一口气，强迫自己冷静下来。

"老管家，领地周围的情况怎么样？"

"东边是荒山，有野兽出没；西边是一条小河；
南边是林子；北边...北边有一伙山贼，有四五十人..."

四五十个山贼，他这边连能打的壮丁都凑不出十个。

李青看着眼前的残破景象，嘴角却慢慢扬起。

困难？这才有意思。

等着吧，他会让这片荒凉的土地，变成人人向往的乐土！
```

### 发展示例

```
【某章 第一次丰收】

秋天到了。

李青站在田边，看着金黄的麦浪，眼眶有些湿润。

三个月前，他亲自带人开荒、翻地、播种。
没有耕牛，就用人拉犁。
没有水源，就挖渠引河水。
没有肥料，就收集各种有机物堆肥。

无数个日夜的辛苦，终于换来了眼前这一片金色。

"大人，可以收割了！"

"好！所有人，收割！"

整个领地都沸腾了。
老人、女人、孩子，所有能动的人都涌向农田。
他们的脸上带着久违的笑容——
这是希望的笑容。

三天后，所有粮食入仓。

"大人，总共收获粮食八百石！"账房激动地报告。

八百石！足够领地所有人吃两年！

李青看着堆满粮食的仓库，终于松了一口气。
活下去的问题，解决了。
接下来，是活得更好。

他转身看向远处的荒山，眼中闪过一丝精光。
据说那里有铁矿...
```

## 📝 模仿指令

> **给 AI 的指令**：
> 
> "请以《领主时代》《工业霸主》的笔触写作本章。注意以下要点：
> 1. **数据可视化**：领地人口、粮食、金币等要有具体数字
> 2. **阶段感**：明确当前处于发展的哪个阶段（生存期/发展期/扩张期）
> 3. **成就爽感**：用对比手法展示发展成果（一年前vs一年后）
> 4. **规划蓝图**：主角要有清晰的发展计划，给读者期待感
> 5. **困难先行**：每个阶段都要有相应的困难和敌人
> 6. **人才招募**：定期引入新人才，丰富阵容"

## ⚠️ 常见问题

| 问题 | 表现 | 解决方案 |
|------|------|----------|
| **发展太快** | 三章就称霸 | 设置合理的困难和时间 |
| **发展太慢** | 五十章还在种地 | 加快节奏，加入冲突 |
| **太流水账** | 今天种地明天盖房 | 聚焦关键节点，跳过日常 |
| **缺乏冲突** | 一路顺风顺水 | 增加外敌、内忧 |
| **金手指太强** | 系统太万能 | 设置限制和代价 |
