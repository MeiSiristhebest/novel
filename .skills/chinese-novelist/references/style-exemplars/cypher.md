# 赛博朋克风格范文 (Cyberpunk Style Exemplar)

> **流派定位**：高科技低生活，霓虹灯下的阴暗角落，人机融合的身份困境，大公司vs底层民众的阶级对立。

## 🎯 核心风格特征

### 1. 视觉美学（必备元素）
```
霓虹灯 + 雨夜 + 全息广告 + 拥挤街道 + 破旧与高科技并存
```

**环境描写公式**：
```
高处：巨型企业大厦、全息广告、无人机
低处：积水小巷、地下黑市、改装义体店
光影：霓虹闪烁、雨水反光、烟雾缭绕
```

### 2. 阶级对立（核心冲突）
```
【上层】大企业（megacorp）、高管、精英区
    vs
【底层】街头混混、黑客、雇佣兵、贫民窟
```

### 3. 身份危机（哲学内核）
- 义体改造后，"我"还是"我"吗？
- 记忆可以被篡改，什么才是真实？
- 人工智能有灵魂吗？

### 4. 语言特点
- **科技术语**：义体、神经接口、ICE（入侵对抗电子）、黑冰
- **俚语黑话**：choom（兄弟）、delta（滚开）、gonk（蠢货）
- **冷硬叙事**：短句、动作导向、不煽情
- **品牌植入**：虚构的大企业名称、武器型号

## ✍️ 经典片段示范

### 【街头场景】霓虹雨夜

```
雨水顺着我的合成皮肤往下淌，神经末梢传来的触感和真皮没什么两样——荒坂的高端货，花了我三个月的佣金。

北山区的夜晚永远是这副德行。

头顶是三十米高的全息广告，一个巨大的日本女人在推销最新款的脑舞芯片，笑容甜得发腻。脚下是积满污水的小巷，一个吸多了的瘾君子靠在垃圾箱旁边抽搐。

这就是夜之城。

上面是天堂，下面是地狱。我们这些人，永远在中间那层夹缝里讨生活。

"V，货到了。"

耳机里传来杰克的声音，我点了点头——虽然他看不见。

"多少人？"

"三个保镖，标准企业配置。维克多科技的二流货色，别担心。"

我摸了摸腰间的统一制式手枪。这把枪跟了我五年，杀过的人比我交过的朋友还多。

"三分钟后动手。"

"收到，choom。"

我深吸一口气，尼古丁混合着雨水的味道。

然后我走进了那片霓虹灯。
```

**风格要点**：
- 开篇即点明义体（合成皮肤）
- 上/下对比（全息广告 vs 瘾君子）
- 黑话自然融入（choom）
- 冷硬、克制、不煽情

### 【义体改造】身份危机

```
医生把镜子递给我。

镜子里的人有一双银色的眼睛。

"克比光学的顶级产品，"医生的声音里带着一丝得意，"四倍变焦，夜视模式，还能直连你的神经接口。"

我盯着那双眼睛。

它们很漂亮，冷冰冰的，像两颗金属球。

"我原来的眼睛呢？"

"已经处理掉了。"医生耸耸肩，"你又不是第一次换件儿了，V。有什么好纠结的？"

我没回答。

他说得对。这不是我第一次换件儿了。

左手是液压驱动的机械臂，能捏碎钢管。右腿装了弹跳强化，三层楼一跃而上。脊柱里埋着神经加速器，反应速度是普通人的两倍。

我低头看了看自己的身体。

还有多少是原装的？

大脑？心脏？还是只剩下那些无法被替换的……记忆？

"记忆也能换，"医生像是看穿了我的想法，"桃井科技正在测试新产品。听说能把别人的记忆直接灌进你脑子里。"

"那还是'我'吗？"

医生笑了笑，收起了手术刀。

"谁知道呢。这个问题，问了几十年也没人能回答。"

我沉默了。

镜子里，那双银色的眼睛毫无表情地看着我。

我不认识它们。

但从今天起，它们就是"我"了。
```

**风格要点**：
- 通过义体清单暗示改造程度
- 核心追问："我"还是"我"吗？
- 医生的冷漠态度代表社会常态
- 结尾留白，不给答案

### 【黑客入侵】矩阵空间

```
我闭上眼睛，激活神经接口。

现实世界消失了。

取而代之的是一片无尽的数据海洋。

在这里，建筑不再是钢筋水泥，而是一座座发光的数据堡垒。每一道防火墙都化作实体的城墙，每一个ICE程序都变成巡逻的守卫。

荒坂塔的虚拟映像就在前方，一座通天的黑色尖塔，周身缠绕着血红色的数据流。

"探测到入侵者。"

冰冷的机械音响起，六个黑影从塔中飞出，朝我扑来。

黑冰。致命级别的反入侵程序。

在现实世界，它们只是一串代码。但在这里，它们是真正的杀手。如果我的虚拟形象被它们撕碎，我的大脑也会跟着被烧穿。

"来吧。"

我举起手中的破解程序——在这里，它看起来像是一把燃烧着蓝色火焰的长刀。

第一个黑冰扑到面前。

我侧身，挥刀，刀锋切开它的数据结构，无数0和1如同鲜血一样喷涌而出。

还有五个。

计时器在我的视野角落跳动。

120秒。

我只有120秒。

超过这个时间，荒坂的追踪程序就会锁定我的物理位置。然后，会有一整队死亡小队破门而入。

"专注，V。"

我咬紧牙关，冲向下一个黑冰。

在数据的海洋里，我是一头孤狼。

要么吃掉猎物，要么被猎物吃掉。
```

**风格要点**：
- 虚拟空间可视化（数据→建筑/城墙）
- 程序→实体化（黑冰→杀手）
- 时间压力制造紧张感
- 现实后果（大脑被烧穿）提高stakes

## ⚠️ 常见雷区

| 雷区 | 错误示范 | 正确示范 |
|------|---------|---------|
| 科技堆砌 | 一段话出现10个术语 | 术语自然融入，不解释 |
| 太光明 | 主角是正义的英雄 | 主角是灰色地带的求生者 |
| 忽视底层 | 只写高科技 | 高科技+低生活并存 |
| 情绪外放 | "他感到绝望" | "他点燃一根烟，沉默了" |
