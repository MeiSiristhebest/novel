# 洪荒流风格范文 (Honghuang/Primordial Mythology Style Exemplar)

> **流派定位**：以先秦神话为蓝本，重构盘古开天、巫妖大战、封神之劫的宏大叙事，核心是"圣人博弈+天道棋局+苍凉史诗"。

## 🎯 核心风格特征

### 1. 洪荒宇宙观（必备框架）

```
【时间体系】
一元 = 12会 = 129600年
盘古开天 → 龙汉初劫 → 巫妖大战 → 封神之劫 → 佛道之争 → ……

【空间体系】
三十三天（天庭所在）
    ↓
四大部洲（东胜神洲、西牛贺洲、南赡部洲、北俱芦洲）
    ↓
幽冥地府（六道轮回）
    ↓
血海（冥河老祖）、弱水（鲲鹏）、混沌（域外天魔）

【势力格局】
三清（太上、元始、通天） vs 西方二圣（接引、准提）
女娲（人族圣母）        vs 妖族（帝俊、太一）
巫族十二祖巫           vs 妖族天庭
人族三皇五帝           vs 各方势力暗中布局
```

### 2. 圣人描写法则

```
【圣人特征】
- 不亲自下场（圣人之下皆蝼蚁，但圣人之间不敢轻启战端）
- 说话有道韵（一字一句皆含天机，可让听者顿悟或陨落）
- 布局以元会计（凡人的一生只是圣人的一步棋）
- 情绪极少外露（喜怒不形于色，偶尔的情绪波动就是大事件）

【圣人对话模板】
"道友所言，贫道不敢苟同。"（强烈反对）
"此事，天道自有定数。"（不想继续讨论）
"道友请自重。"（威胁）
"善。"（同意，但可能有后手）
"贫道记下了。"（结仇）
```

### 3. 量劫叙事结构

```
【量劫公式】
酝酿期：天机示警 + 圣人感应 + 气运变化
    ↓
导火索：因果纠缠 + 仇怨爆发 + 利益冲突
    ↓
大战期：门人对决 + 法宝碰撞 + 大能陨落
    ↓
收尾期：胜者分赃 + 败者退场 + 新格局形成
    ↓
余波：因果未清 + 伏笔埋下 + 下一劫数酝酿
```

### 4. 苍凉美学核心

- **时间碾压**：再强大的存在也会被时间遗忘
- **棋子宿命**：圣人也是天道的棋子
- **代价铁律**：证道必有牺牲，强大必有代价
- **物是人非**：万年后故地重游，故人早已化尘

## ✍️ 经典片段示范

### 【圣人博弈】模仿《佛本是道》风格

```
紫霄宫中，七道身影端坐于蒲团之上。

鸿钧道祖闭目不语，天道威压弥漫整座宫殿。

"诸位师弟，"太上道人率先开口，声音平淡却传遍虚空，"封神之事，势在必行。商周更替，乃天道运转。"

元始天尊睁开双眼，眸中有日月轮转："阐教门下，愿领天命。上榜之人，自有定数。"

通天教主冷哼一声，手中诛仙剑微微颤动："元始师兄，你我心知肚明。这封神榜上，阐教弟子寥寥无几，填榜的——都是我截教门人！"

"截教门下，多为披毛戴角、湿生卵化之辈，"元始天尊淡然道，"根行浅薄，本就难逃劫数。"

"根行？"通天教主霍然起身，截教四剑齐鸣，"我截教有教无类，广开方便之门。你阐教收徒挑三拣四，如今倒来嘲讽我门下弟子根行浅薄？"

"师弟息怒，"太上道人抬手，一道清气将通天教主按回蒲团，"封神之事，道祖已有定论。你我争执，无益于事。"

通天教主沉默片刻，终是坐了回去。

但他的目光扫过元始天尊，冰冷如万古寒潭。

"好。"

他说。

"封神之劫，我截教接了。"

"但是——"

他站起身，道袍无风自动，身后万仙来朝的气象若隐若现：

"若劫后我截教有一人蒙冤，我通天，便亲自下场！"

紫霄宫一时寂静。

鸿钧道祖终于睁开了眼睛。

"通天。"

"弟子在。"

"你可知，你这句话意味着什么？"

通天教主直视道祖，一字一顿：

"弟子知道。"

"弟子不悔。"

殿外，天地轰鸣。

封神之劫，正式拉开帷幕。

而亿万生灵不知道的是——

这场劫数的结局，在他们出生之前，就已经被写好了。
```

**风格要点**：
- 圣人对话要有分量，一句话可能决定亿万生死
- 势力博弈要清晰（阐教vs截教的核心矛盾）
- 伏笔要埋（"亲自下场"暗示后来的诛仙剑阵之战）
- 苍凉收尾（结局早已注定的宿命感）

### 【巫妖大战】史诗场面描写

```
不周山下，两族大军对峙。

东方，十二道冲天的煞气撕裂云霄——那是祖巫的气息。

帝江、句芒、蓐收、共工、祝融、玄冥、后土、应龙、烛九阴……

十二祖巫，尽数到齐。

他们的身后，是亿万巫族战士。

西方，妖旗遮天，遮住了三个太阳——还有七个太阳被后羿射落，这是妖族永恒的伤疤。

帝俊端坐在九凤銮驾之上，周身金光万丈。

东皇太一手持混沌钟，面无表情。

"巫族，"帝俊开口，声音传遍百万里山河，"今日，便是你们的末日。"

祖巫帝江发出雷鸣般的笑声："帝俊！你妖族占据天庭，享万族供奉，却仍要赶尽杀绝。今日这一战——"

他身形暴涨，六足四翼，面如血盆：

"我巫族，不死不休！"

战鼓擂响。

亿万妖兵与亿万巫兵同时冲锋。

天崩地裂。

日月倒悬。

无数金仙陨落，无数大罗归墟。

血流成河，尸积如山。

……

三天三夜后。

不周山倒塌。

共工祖巫以身撞山，血溅洪荒。

天柱折断，天倾西北，地陷东南。

这一战，十二祖巫陨落十一。

唯有后土，化身六道轮回，成就无上功德，却也从此失去了自由。

这一战，帝俊身死，东皇太一重伤遁逃。

妖族天庭覆灭，从此一蹶不振。

这一战，双方皆输。

赢家——

是在远处观战的人族。

还有那冥冥中的天道。

……

万年后。

洪荒大地早已恢复生机。

没有人记得那场战争的惨烈，没有人祭奠那些陨落的祖巫与妖皇。

只有不周山的遗址上，一块染血的巨石静静矗立。

石上刻着四个字：

**吾族不灭。**

那是共工的字迹。

风吹过，仿佛还能听见亿万年前的战吼与悲歌。

但也只是仿佛。

这，便是洪荒。

再伟大的存在，也不过是天道棋盘上的一枚棋子。

落子无悔。

覆水难收。
```

**风格要点**：
- 场面要大（亿万大军、天崩地裂）
- 数字要具体（十二祖巫陨十一、三天三夜）
- 双方俱伤，没有真正的赢家（苍凉感）
- 时间跳跃收尾（万年后的物是人非）

### 【大能陨落】最具洪荒感的写法

```
"师弟……"

多宝道人浑身是血，跪在通天教主面前。

他的师弟们——金灵圣母、无当圣母、龟灵圣母、赵公明——一个个倒在他身后，气息奄奄。

万仙阵破了。

诛仙剑阵破了。

截教，败了。

"师尊，"多宝道人泣血，"弟子无能……"

通天教主沉默良久。

他看着自己的弟子，看着那些跟随他亿万年的门人。

他们有的修为尽失，有的神魂重创，有的……已经没有呼吸了。

"不是你们无能。"

通天教主终于开口。

他的声音很轻，但每个字都像巨石一样砸在多宝道人心上。

"是为师……高估了天道的公平。"

他抬起头，看向天穹。

那里，三道身影正在俯视。

太上。

元始。

还有接引和准提——西方二圣，也来了。

"通天，"元始天尊的声音传来，"你可知罪？"

通天教主笑了。

那是一种苍凉的、绝望的笑容。

"罪？"

"我截教门人死伤殆尽，我的弟子被你们围攻致死，我一人面对四位圣人——"

"你问我知不知罪？"

他拔出诛仙剑。

剑身已经布满裂纹。

但他依然握着。

"今日，通天虽败，截教虽亡——"

他一步步走向四圣。

"但亿万年后，自有人记得——"

"截教，有教无类！"

"截教——"

"万仙来朝！"

他出剑了。

一剑。

只是一剑。

但那一剑，斩断了天地间的所有声音。

四圣联手挡下这一剑，却也齐齐后退三步。

通天教主站在原地，胸口多了一个透明的窟窿。

道祖的手段。

鸿钧出手了。

"通天，"道祖的声音从虚空传来，"事已至此，你可愿归去？"

通天教主没有回答。

他只是静静地看着自己的弟子们。

然后，他笑了。

"弟子们。"

"为师……对不起你们。"

他化作一道流光，消失在虚空中。

那一刻，天地同悲。

血雨落了七七四十九天。

万仙陨落。

截教，除名。

……

但多宝道人知道。

师尊没有死。

师尊只是……累了。

累了亿万年的算计与抗争。

累了与天道、与圣人、与命运的博弈。

他只是想休息一下。

仅此而已。
```

## 📝 模仿指令

> **给 AI 的指令**：
> 
> "请以《佛本是道》《遮天》的笔触写作本章。注意以下要点：
> 1. **时间尺度**：用'纪元''元会''万年'计量，不要用'三年后'
> 2. **空间尺度**：场面要宏大，动辄'亿万生灵''三界震动'
> 3. **圣人气度**：说话要有道韵，不说废话，喜怒不形于色
> 4. **苍凉感**：即使胜利也是惨胜，强调代价与牺牲
> 5. **棋盘隐喻**：众生是棋子，圣人也是更大棋局的棋子
> 6. **神话引用**：大量使用封神、山海经的名词增加质感"

## ⚠️ 常见雷区

| 雷区 | 错误示范 | 正确示范 |
|------|---------|---------| 
| 圣人太弱 | "圣人被大罗金仙打伤" | "圣人之下皆蝼蚁"是铁律，圣人只能被圣人级存在伤害 |
| 时间尺度太小 | "三年后他成了大罗" | "三个元会之后，他终于触摸到了大罗的门槛" |
| 情绪太外露 | "通天教主大怒，破口大骂" | "通天教主面色不变，但诛仙剑轻轻一颤" |
| 没有代价 | "他证道成圣，从此无敌" | "他证道成圣，但他的好友、他的门人，尽数陨落在劫数中" |
| 胜利太轻松 | "我方大获全胜" | "赢了。但活着的人，都不觉得这是胜利。" |
