# 克苏鲁/诡秘流风格范文 (Cthulhu/Lovecraftian Style Exemplar)

> **流派定位**：以未知恐惧、理智崩溃、神秘学体系为核心，融合维多利亚时代美学与超自然元素。

## 🎯 核心风格特征

### 1. 序列/途径体系（《诡秘之主》标杆）

```
【非凡者序列示例】
序列9（入门）→ 序列7（中阶）→ 序列4（圣者）→ 序列1（天使）→ 序列0（真神）

途径特色：
- 占卜家途径：命运、预言、灵视
- 黑夜途径：隐匿、暗杀、梦境
- 读心者途径：精神、操控、疯狂
- 猎人途径：追踪、诅咒、献祭
```

**核心规则**：
- 序列越高，越接近非人
- 扮演法则：你在扮演怪物，怪物也在扮演你
- 污染与失控：获取力量必然付出代价

### 2. 恐惧公式（未知 > 已知）

```
【恐惧递进公式】
暗示（奇怪的细节）
    ↓
不安（生理反应：头痛、恶心、幻听）
    ↓
窥见（模糊的、不完整的真相）
    ↓
崩溃（理智值归零，或被迫接受不可名状的真实）
```

**关键原则**：永远不要完全展示怪物。留白比描写更恐怖。

### 3. 维多利亚时代美学

| 元素 | 描写方式 |
|-----|---------|
| **服饰** | 燕尾服、高礼帽、束腰裙、怀表链、蕾丝手套 |
| **建筑** | 哥特式尖顶、煤气灯、雾气弥漫的街道、阴暗的下水道 |
| **交通** | 蒸汽火车、四轮马车、公共马车、雾中的汽笛声 |
| **氛围** | 工业革命的煤灰、阶级分明、神秘学复兴、连环杀手 |

### 4. 感官描写法则

```
视觉：扭曲、非欧几何、不可能的角度、边缘模糊的阴影
听觉：低语、呓语、金属摩擦、来自深处的回响
嗅觉：腐烂、海腥味、焚烧的羊皮纸、甜腻的血腥
触觉：粘腻、冰冷、不该存在的质感、皮肤下的蠕动
味觉：金属味（血）、苦涩（毒素）、甜腻（腐败）
```

## ✍️ 经典片段示范

### 【理智动摇】模仿《诡秘之主》风格

```
克莱恩按住太阳穴，感觉有什么东西正试图从颅骨里钻出来。

那不是普通的头痛。那是一种……被窥视的感觉。

他强迫自己看向窗外。月光惨白，照在贝克兰德的屋顶上，一切似乎都很正常。但他的余光捕捉到了什么——

街角的煤气灯下，站着一个人影。

不，不是人影。

那个轮廓的比例是错误的。手臂太长，几乎垂到膝盖。头部的角度不对，像是被拧过一样。最可怕的是，它没有影子。

煤气灯的光芒明明照在它身上，但地面上空无一物。

克莱恩眨了眨眼。

人影消失了。

但他的耳边响起了一阵细微的、如同指甲刮擦黑板的声音：

*"看见我了。"*

不是疑问句。是陈述句。

克莱恩猛地拉上窗帘，后背抵住墙壁，大口喘息。他能感觉到冷汗正沿着脊椎往下淌。

他不确定那个声音是来自窗外，还是……来自他的脑子里。
```

**风格要点**：
- 先写生理反应（头痛、冷汗）
- 细节错位制造不安（比例错误、没有影子）
- 留白恐惧（它是什么？不解释）
- 结尾的不确定性（外部还是内部？）

---

### 【仪式与污染】模仿《诡秘之主》风格

```
蜡烛的火焰是深红色的，像凝固的血。

达斯特用银刀在羊皮纸上刻下最后一个符文。符文的形状让他的眼睛刺痛——那些线条似乎在蠕动，像活物一样试图爬出纸面。

"我以真实造物主之名……"

他的声音在地下室回荡。每一个音节都带着奇怪的重量，像是有另一个声音在与他同步——更低沉，更古老，来自某个不该存在的地方。

"……召唤你，来自深渊的使者。"

蜡烛的火焰同时熄灭。

黑暗中，他听到了呼吸声。

不是他自己的。

"成功了？"达斯特颤声问道。

没有回答。但他感觉到有什么东西在接近。空气变得粘稠，像是浸满了看不见的蜂蜜。他的皮肤开始发痒——那种痒不是在表面，而是在肌肉下面，在骨头缝隙里。

然后，他看见了。

黑暗中有一双眼睛。不，不是眼睛。是……孔洞。两个完美的、不反射任何光线的圆形虚空。它们正在注视着他。

*"你召唤了我。"*

那个声音直接出现在他的脑海里，像一根生锈的钉子钉入柔软的大脑灰质。

*"现在，你属于我了。"*

达斯特想尖叫，但他的喉咙发不出任何声音。

他低头，看见自己的手正在变化——皮肤下有什么东西在蠕动，指尖正在变黑，像是被墨水浸透。

这就是代价。

他早该知道的。

召唤从来不是免费的。
```

**风格要点**：
- 仪式细节要具体（银刀、羊皮纸、符文）
- 感官叠加（视觉+听觉+触觉）
- 污染的可视化（皮肤变化、身体异变）
- 代价永远存在——这是克苏鲁流的铁律

### 【信息污染】SAN值狂掉的描写

```
那本书的封面是用人皮装订的。

这不是比喻。艾伦能清晰地看见皮革表面的毛孔，以及一小块残留的……纹身。一朵已经变色的玫瑰。

他应该把书放回去。他知道自己应该这么做。

但他的手不听使唤，翻开了第一页。

文字是用深褐色的墨水书写的。他不想知道那是什么墨水。第一句话很简单：

*"阅读即契约。"*

艾伦皱眉。什么意思？

他继续往下看。第二句话：

*"你已经无法停止阅读了。"*

一阵寒意窜上脊椎。艾伦试图合上书本——

他做不到。

他的手指像是被粘在纸页上。眼睛不受控制地继续扫视文字。那些句子开始变化，不再是完整的语句，而是碎片——

*——深渊回视——*
*——你的名字已被记录——*
*——七日后——*
*——门会打开——*

艾伦的头痛欲裂。鼻血开始往下滴，滴在书页上，被文字吸收。

最后一行字浮现出来，用的是他自己的血：

*"现在，你知道了。"*

书本自动合上。

艾伦跪倒在地，浑身颤抖。他不知道自己读到了什么，但有一件事是确定的——

那些知识现在在他脑子里。

而那些知识，正在改变他。
```

## 📝 模仿指令

> **给 AI 的指令**：
> 
> "请以《诡秘之主》《深海余烬》的笔触写作本章。注意以下要点：
> 1. **感官先行**：每个恐怖场景先写生理反应（头痛、恶心、冷汗）
> 2. **细节错位**：用微小的"不对劲"制造不安（比例错误、颜色异常）
> 3. **永远留白**：不要完整描写怪物外貌，暗示比明说更恐怖
> 4. **代价铁律**：获取力量/知识必须付出代价（理智、身体、灵魂）
> 5. **维多利亚美学**：服饰、建筑、语言风格要有时代感
> 6. **信息即污染**：某些知识本身就是危险的，知道越多越疯狂"

## ⚠️ 常见雷区

| 雷区 | 错误示范 | 正确示范 |
|------|---------|---------| 
| 怪物全露 | "那是一只八米高的触手怪物，有十二只眼睛……" | "黑暗中有什么东西在蠕动。他只看见了一个轮廓——那个形状不该存在于任何已知的生物学分类中。" |
| 缺乏代价 | "他学会了禁咒，变得更强了" | "他学会了禁咒。代价是他再也无法直视镜子——镜中的自己已经不是人类的面孔了。" |
| 恐惧直给 | "他非常害怕" | "他的手在发抖，牙齿在打颤，一种原始的、来自基因深处的本能正在尖叫：逃。" |
| 理智太稳定 | 主角目睹恐怖场景后继续正常行动 | 主角目睹后需要时间恢复，可能出现PTSD症状、噩梦、幻觉 |
| 世界观太明确 | 把所有神话体系解释得清清楚楚 | 保持迷雾，连主角也只知道冰山一角 |

## 🔧 序列能力设计参考

| 途径 | 低序列能力 | 高序列能力 | 代价/污染 |
|-----|----------|----------|----------|
| **占卜家** | 灵视、简单预知 | 命运编织、因果律操控 | 窥见越多真相，越难区分现实与幻象 |
| **黑夜** | 隐匿、制造黑暗 | 概念级隐身、梦境操控 | 逐渐厌恶光明，与人类情感疏离 |
| **死神** | 通灵、操控尸体 | 收割灵魂、死亡不可逆 | 活人的温度让你不适，尸体让你安心 |
| **猎人** | 追踪、陷阱 | 诅咒锁定、必杀猎杀 | 越来越享受狩猎的快感，包括猎杀人类 |
| **读心者** | 读取表层思想 | 人格覆写、精神支配 | 自我边界模糊，分不清哪些思想是自己的 |
