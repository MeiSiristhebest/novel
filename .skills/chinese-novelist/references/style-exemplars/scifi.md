# 科幻/星际流风格范文 (Sci-Fi Style Exemplar)

> **流派定位**：中国科幻网文，融合"硬科幻设定+修炼体系+星际冒险"，是近年来增长最快的流派之一。2023年银河奖、启明星奖的加持，让科幻网文从小众走向主流。

## 🎯 核心风格特征

### 1. 双体系融合（科幻网文特色）
科幻网文通常将"科技"与"修炼/异能"融合，创造独特的战力体系。

**常见模式**：
```
模式A：科技为主 + 修炼为辅
  例：《吞噬星空》— 武者修炼体内宇宙，使用科技武器

模式B：修炼为主 + 科技为载体
  例：《星域四万年》— 修真者驾驶灵能机甲，穿越星际

模式C：科技即魔法
  例：《深海余烬》— 蒸汽机械与神秘力量共存
```

### 2. 宇宙观构建
科幻网文需要构建多层次的宇宙观：

```
【个人层】→ 战斗力等级（武者/宇宙级/域主级）
    ↓
【文明层】→ 星球/星系/宇宙帝国
    ↓
【宇宙层】→ 高维生命/宇宙意志/多元宇宙
```

### 3. 语言特点
- **科技感术语**：量子、空间跃迁、暗能量、纳米机器人
- **数字精准**：坐标、距离、能量等级都要有具体数值
- **冷峻叙事**：星际探索的孤独感，宇宙的冷漠
- **热血燃点**：人类vs宇宙的渺小与伟大

## ✍️ 经典片段示范

### 【星际战斗】模仿《吞噬星空》风格

```
虚空之中，罗峰驾驭着暗金色的飞船，如一道流光穿梭于星域之间。

"前方检测到域主级生命体反应，数量：三。"

智脑的声音冷静而机械。

罗峰眯起眼睛，手指在控制台上轻轻敲击。

三个域主级？

放在三年前，他连逃跑的资格都没有。

但现在……

"解除隐匿模式。"

"命令已执行。"

暗金色的飞船表面，浮现出一层淡蓝色的能量护罩。那是S9级文明的尖端科技——空间护盾。

三道身影出现在前方虚空。

是虫族！

三只体型如山岳的巨型虫族，复眼中满是贪婪与凶残。

"人类？"领头的虫族发出刺耳的精神波动，"这里是我们的领地，入侵者，死！"

罗峰没有回答。

他从飞船中缓步走出，站在虚空之中，周身没有任何能量波动。

"就一个人？"

虫族们发出嘲讽的嘶鸣。

下一秒，它们的嘲讽声戛然而止。

因为罗峰的身后，出现了一片星海。

那不是真正的星海，而是无数把飞剑汇聚而成的剑阵！每一把飞剑都蕴含着毁灭一颗行星的力量！

"我说了吗？"罗峰的声音平静，"我只有一个人？"

轰——！

万剑齐发。

三只域主级虫族，连惨叫都来不及发出，就被剑光淹没。

战斗结束。

从开始到结束，不过三秒。

罗峰收起飞剑，转身回到飞船。

"继续前进。"

"命令已执行。"

暗金色的飞船再次化作流光，消失在无尽的星空之中。

在这片宇宙，人类，从来不是弱者。
```

**风格要点**：
- 科技元素要具体（S9级文明、智脑、空间护盾）
- 战力对比要清晰（域主级、万剑齐发）
- 节奏要快，战斗要干脆利落
- 结尾要升华（人类的骄傲）

### 【深海探索】模仿《深海余烬》风格

```
邓肯站在"无归号"的甲板上，凝视着前方那片永恒的迷雾。

蒸汽引擎发出低沉的轰鸣，铜质的管道中流淌着炽热的蒸汽。这是人类文明最后的造物，也是他们穿越无尽之海的唯一依凭。

"船长，前方探测到异常信号。"

大副的声音中带着一丝颤抖。

邓肯没有回答，只是将手中的望远镜举得更高。

迷雾之中，隐约有什么东西在移动。

那不是船。

那是……一座城市。

一座倒悬在海面之上的城市。

尖塔朝下，城墙朝上，无数的窗户中透出幽绿色的光芒。那光芒像是无数只眼睛，正在注视着他们。

"我的神……"有水手发出惊恐的低呼。

邓肯却笑了。

这种景象，他已经见过太多次了。

无尽之海不欢迎清醒的人。每一次航行，都是与疯狂的赛跑。

"全速前进。"

"船长？那座城市……"

"那不是城市。"邓肯打断他，声音平静得像是在谈论天气，"那是诱饵。"

"诱饵？"

"是的。真正的危险，在我们身后。"

大副猛然回头。

在"无归号"的航迹上，海水正在缓缓升起。

那不是浪涛，而是一只巨大的眼睛。

正在睁开。

邓肯点燃了一根烟，深吸一口。

"欢迎来到无尽之海。"

他的声音中带着一丝疯狂的笑意。

"在这里，醒着的人才是怪物。"
```

**风格要点**：
- 蒸汽朋克元素（铜质管道、蒸汽引擎）
- 克苏鲁式恐怖（不可名状、巨大眼睛）
- 氛围感要强（迷雾、幽绿光芒）
- 主角要有"疯狂"的气质

### 【机甲热血】模仿《星域四万年》风格

```
李耀的机甲在爆炸的火光中穿行，浑身上下已经残破不堪。

左臂被打断了，推进器损坏了三分之二，能量储备只剩下7%。

"警告，能量不足。建议立即撤离。"

机甲的AI发出冰冷的提示。

李耀没有理会。

他的目光锁定在前方那尊高达百米的巨型机甲上。那是魔族的旗舰机——"深渊领主"。

"撤离？"李耀笑了，嘴角渗出血丝，"我要是撤了，身后那三百万平民怎么办？"

"根据计算，以您目前的战力，与'深渊领主'交战的存活概率为：0.03%。"

"够了。"

李耀握紧操纵杆。

"0.03%，就意味着还有机会。"

他启动了最后的能量，机甲周身亮起炽白色的光芒——那是自爆系统的启动信号。

"李耀！你疯了！"通讯频道中传来战友的吼声。

"我没疯。"李耀的声音异常平静，"我只是……不想让他们失望。"

他想起了那三百万平民的眼神。

恐惧，但也有希望。

那些希望，是给他的。

"修真者的使命，是守护。"

李耀深吸一口气，将能量输出拉到最大。

"那么，就让我来守护吧。"

轰——！

一道炽白色的光芒冲向"深渊领主"。

那是人类，在星际时代的黎明中，留下的最灿烂的光。
```

**风格要点**：
- 机甲要有具体数据（能量7%、0.03%存活率）
- 热血要有牺牲精神（守护、自爆）
- 对比要强（渺小人类 vs 巨型机甲）
- 结尾要升华（人类的光芒）

## 📝 模仿指令

> **给 AI 的指令（硬科幻向）**：
> 
> "请以《吞噬星空》《深海余烬》的笔触写作本章。注意以下要点：
> 1. **科技术语**：空间跃迁、能量护盾、量子通讯等
> 2. **数字精准**：距离、能量、战力等级都要有具体数值
> 3. **宇宙冷感**：星际探索的孤独与渺小
> 4. **文明层次**：个人→星球→星系→宇宙"

> **给 AI 的指令（热血向）**：
> 
> "请以《星域四万年》的笔触写作本章。注意以下要点：
> 1. **机甲描写**：型号、武器、能量状态
> 2. **战斗激烈**：爆炸、火光、残破
> 3. **牺牲精神**：守护、使命、人类之光
> 4. **修真+科技**：灵能机甲、剑气炮、符文护盾"

## ⚠️ 科幻网文常见设定表

| 设定类型 | 常见元素 | 示例 |
|---------|---------|------|
| **战力体系** | 武者等级/机甲等级/异能等级 | 行星级→恒星级→宇宙级 |

| **种族设定** | 人类/虫族/机械族/高维生命 | 虫族入侵、机械叛乱 |
| **空间概念** | 亚空间/高维空间/平行宇宙 | 空间跃迁、维度裂缝 |
| **能量体系** | 暗能量/灵能/量子能 | 灵能机甲、暗能武器 |
