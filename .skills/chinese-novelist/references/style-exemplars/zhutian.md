# 诸天流范文指南 (Multiverse/Zhutian)

> **流派定义**：主角穿越到不同的影视剧/小说/动漫世界，利用先知优势或金手指在各个世界冒险成长

---

## 📊 流派特征

### 核心卖点

| 要素 | 说明 | 读者期待 |
|------|------|----------|
| **世界跳转** | 穿越多个熟悉的IP世界 | 新鲜感、情怀 |
| **先知优势** | 知道剧情走向 | 看主角如何操作 |
| **收集系统** | 各世界获得能力/物品 | 成长感、收集癖 |
| **蝴蝶效应** | 改变原著剧情 | 同人创作快感 |

### 经典作品参考

```
代表作品：
- 《从斗罗开始的万界旅程》
- 《诸天万界神龙系统》
- 《从遮天开始的诸天万界》
- 《诸天投影》
- 《带着仓库到大明》

常见穿越世界：
影视剧：亮剑、雪豹、琅琊榜、庆余年
武侠：射雕、天龙、笑傲、倚天
玄幻：斗破、斗罗、完美、遮天
动漫：火影、海贼、龙珠、死神
```

## 🎯 系统设定

### 系统类型

| 类型 | 特点 | 优势 | 劣势 |
|------|------|------|------|
| **任务型** | 完成任务获得奖励 | 剧情推动力强 | 被动 |
| **抽奖型** | 随机获得能力 | 惊喜感 | 不可控 |
| **签到型** | 每天/每世界签到 | 稳定成长 | 可能无聊 |
| **商城型** | 用积分兑换 | 选择自由 | 需要设计经济 |
| **融合型** | 以上多种结合 | 丰富 | 复杂 |

### 系统设定模板

```yaml
系统名称: 诸天万界系统

核心功能:
  - 世界穿越
  - 能力保留
  - 任务发布
  - 商城兑换

货币体系:
  - 世界积分（完成世界任务获得）
  - 功德点（善行获得）
  - 气运值（改变命运获得）

穿越规则:
  - 每个世界停留时间：完成主线任务
  - 能力保留：100%保留
  - 物品保留：空间类物品可带
  - 身份：随机/选择

成长路径:
  世界1（新手村）→ 世界2 → 世界3 → ... → 终极世界
```

## 🌍 世界选择策略

### 世界难度分级

| 难度 | 世界类型 | 示例 | 主角等级要求 |
|------|----------|------|-------------|
| **D级** | 普通武侠 | 射雕、鹿鼎记 | 新手 |
| **C级** | 高武武侠 | 天龙、风云 | 入门 |
| **B级** | 仙侠 | 仙剑、诛仙 | 筑基 |
| **A级** | 玄幻 | 斗破、斗罗 | 金丹 |
| **S级** | 洪荒/遮天 | 洪荒、遮天 | 元婴+ |
| **SS级** | 漫威/龙珠 | 漫威、龙珠 | 渡劫+ |

### 世界顺序设计

```
推荐路线（由易到难）：

路线1：武侠成长线
射雕 → 天龙 → 风云 → 仙剑 → 斗破 → 遮天

路线2：影视流
亮剑 → 雪豹 → 琅琊榜 → 庆余年 → 择天记

路线3：动漫流
火影 → 死神 → 海贼 → 龙珠 → 漫威

设计原则：
1. 前期世界让主角快速成长
2. 中期世界获得关键能力
3. 后期世界挑战极限
```

## 📖 剧情模板

### 单世界标准流程

```
【进入世界】
├── 系统提示世界信息
├── 确认身份和时间点
├── 明确主线任务
└── 评估当前实力差距

【前期（10-30章）】
├── 适应世界规则
├── 利用先知优势布局
├── 结交重要人物
├── 获得第一桶金/第一份机缘
└── 展示初步实力

【中期（30-60章）】
├── 参与原著重要剧情
├── 改变关键节点
├── 与原著主角互动
├── 获得核心能力/物品
└── 建立势力/关系

【后期（60-80章）】
├── 完成主线任务
├── 处理遗留问题
├── 收割剩余资源
├── 安排后事
└── 准备离开

【离开世界】
├── 系统结算奖励
├── 能力整合
├── 下一世界预告
└── 过渡章节
```

### 经典桥段

```yaml
先知碾压:
  - 提前到达机缘地点
  - 预知敌人计划并反制
  - 点化原著主角
  - 避开原著死亡陷阱

蝴蝶效应:
  - 救下原本该死的角色
  - 提前击杀大反派
  - 改变原著CP走向
  - 创造新的历史

收集环节:
  - 获得特殊功法/武学
  - 带走神器/宝物
  - 吸收特殊能力
  - 收服原著强者（召唤类）

装逼打脸:
  - 被原著反派小看后反杀
  - 展示跨世界的能力
  - 一句话点评原著剧情
  - "你知道你以后会死在谁手里吗？"
```

## ✍️ 语言风格

### 世界切换的写法

```
离开旧世界:
"【叮！恭喜宿主完成射雕世界主线任务】
【获得奖励：九阴真经（完整版）、降龙十八掌、世界积分×5000】
【是否返回主神空间？】"

林凡看了一眼远处的桃花岛，心中有些不舍。
"确认。"
白光一闪，他的身影消失在这个世界。

进入新世界:
意识恢复时，林凡发现自己站在一片悬崖边。
远处，一座巨大的学院若隐若现。
空气中弥漫着淡淡的药香。

"【检测到宿主进入新世界】
【当前世界：斗破苍穹】
【身份：萧家旁系子弟】
【主线任务：获得异火三枚】"

林凡嘴角微扬。
斗破世界...有意思。
```

### 先知视角的描写

```
林凡看着眼前意气风发的少年，眼神复杂。

萧炎，萧家的天才，纳戒中藏着一个老头子，
三年后会被未婚妻退婚，然后开始逆袭之路。

而现在，他还什么都不知道。

"你就是萧炎？"林凡开口。

"你是谁？"萧炎皱眉，语气带着几分傲气。

林凡笑了笑，没有回答。
这份傲气，等三年后再看看还剩多少吧。

不过...他看向萧炎的纳戒，
药老啊药老，咱们打个商量怎么样？
```

## 📝 范文示例

### 开篇示例

```
【第一章 诸天系统】

林凡睁开眼，发现自己躺在一片草地上。

头痛欲裂，大量的记忆涌入脑海——
他穿越了，附身在一个刚被狼咬死的猎户儿子身上。

更离谱的是...

【叮！检测到宿主符合条件，诸天万界系统绑定中...】
【绑定成功！】
【欢迎使用诸天万界系统，祝宿主旅途愉快】

林凡愣了一下，随即狂喜。

诸天系统！这是网文标配啊！

他连忙查看系统面板：

【宿主：林凡】
【当前世界：射雕英雄传（开局前三年）】
【实力：普通人】
【技能：无】
【积分：0】
【主线任务：获得九阴真经，成为当世一流高手】
【奖励：C级世界通行证，5000积分】

射雕？开局前三年？

林凡快速回忆原著剧情，眼睛越来越亮。

三年时间，足够他做很多事了。

桃花岛的武学、全真教的功法、欧阳锋的蛤蟆功...
还有那本人人争抢的九阴真经。

"系统，最近的城镇在哪？"

【东南方向，三十里外，牛家村】

牛家村？那不是郭靖杨康出生的地方吗？

林凡嘴角上扬，起身向东南走去。

他的诸天之旅，就从这里开始！
```

## ✍️ 经典片段示范

### 【世界切换】标准写法

```
白光消散，林凡睁开眼睛。

入目是一片雾蒙蒙的山林，空气中弥漫着淡淡的灵气。

【叮！检测到宿主进入新世界】
【当前世界：斗破苍穹】
【时间节点：主线开始前两年】
【身份：萧家旁系子弟，萧远】
【主线任务：获得三枚异火，达到斗皇境界】
【奖励：B级世界通行证，异火融合术】

斗破苍穹？

林凡眼睛一亮。

这可是他最熟悉的世界之一——异火、斗技、药老、萧炎……

还有那句经典的"三十年河东三十年河西，莫欺少年穷"。

他低头看了看自己的身体，感受着体内微弱的斗气。

【宿主当前实力：斗者三段】

才三段？

不过没关系。他知道这个世界所有的机缘在哪里。

"系统，萧炎现在在哪？"

【萧炎当前位置：萧家，正处于天才期，斗之气九段】

两年后，纳兰嫣然就会上门退婚。

林凡嘴角微扬。

两年时间，足够他做很多事了。

青莲地心火、骨灵冷火、落星阁的藏书……

这个世界的资源，他要好好收割一番。
```

### 【先知碾压】预知优势的正确用法

```
迦南学院，内院选拔赛。

林凡站在角斗场中央，看着对面气势汹汹的对手。

"萧远，你一个旁系子弟，也敢参加内院选拔？"对面的年轻人冷笑，"今天我让你知道什么叫天差地别！"

林凡没有说话。

他只是静静地看着对手，眼中闪过一丝玩味。

这个人叫刘青，斗灵巅峰，擅长火属性斗技"烈焰掌"。

原著中，他在内院混了三年，最后死在了魔兽山脉的一头四阶魔兽口中。

一个连原著配角都算不上的路人甲。

"动手吧。"林凡淡淡道。

刘青大怒，双掌燃起火焰，直扑而来——

"烈焰掌！"

林凡微微侧身，轻松躲过。

他的动作很慢，但恰好避开了刘青的攻击轨迹。

因为他知道，刘青这招烈焰掌有一个破绽——第三掌会出现0.3秒的僵直。

一掌、两掌、三掌——

林凡出手了。

他的拳头精准地打在刘青的胸口，内劲透体而入。

噗！

刘青喷出一口鲜血，整个人飞出擂台。

全场死寂。

"这……一招？"

"他什么时候这么强了？"

"不是说他才斗灵初期吗？怎么能一招打败巅峰？"

林凡拍了拍手上并不存在的灰尘，转身离开。

先知的优势，不在于力量碾压。

而在于——他知道敌人所有的弱点。
```

### 【跨世界能力整合】

```
"你究竟是什么人？"

魂殿长老浑身颤抖，看着眼前的年轻人。

这个年轻人明明只是斗宗境界，但他刚才展现的手段……

那一剑，快得连他都看不清轨迹。

林凡收起手中的剑，微微一笑。

"我是谁不重要。"

他慢慢走向魂殿长老，每一步都带着压迫感。

"重要的是，你们魂殿，惹了不该惹的人。"

魂殿长老咬牙，双手结印——

"魂灭！"

一道黑色的光芒直奔林凡面门。

林凡没有躲。

他只是轻轻伸出手，一股金色的真气包裹掌心——

那是九阳神功。

来自射雕世界的顶级内功。

金光与黑芒相撞，黑芒瞬间消融。

"不可能！"魂殿长老瞳孔剧震，"你的斗气怎么能克制魂力？！"

林凡笑了。

斗气？

不，这是真气。是内功。

是来自另一个世界的力量。

"给你个建议，"林凡走到魂殿长老面前，居高临下地看着他，"回去告诉魂天帝，这个世界——"

他的眼中闪过一道精光：

"已经有人预定了。"
```

## 📝 模仿指令

> **给 AI 的指令**：
> 
> "请以诸天流的笔触写作本章。注意以下要点：
> 1. **世界信息**：进入新世界时要明确时间节点、身份、任务
> 2. **先知视角**：展示主角对原著剧情的了解，用内心独白点评
> 3. **能力整合**：战斗时可以使用其他世界获得的能力
> 4. **系统互动**：适时弹出系统提示，但不要太频繁
> 5. **蝴蝶效应**：主角的行为要对原著剧情产生影响
> 6. **世界尊重**：熟悉原著设定，不要出现常识性错误"

## ⚠️ 常见问题

| 问题 | 表现 | 解决方案 |
|------|------|----------|
| **主角太弱鸡** | 每个世界都被打 | 合理升级，前期给足金手指 |
| **主角太无敌** | 每个世界都碾压 | 选择高难度世界，设置限制 |
| **世界太短** | 一个世界10章就完了 | 重要世界至少50章 |
| **能力太杂** | 收集太多用不上 | 整合能力，删繁就简 |
| **与原著脱节** | 完全不参与原著剧情 | 深度参与，改变关键点 |
