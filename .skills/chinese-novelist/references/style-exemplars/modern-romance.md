# 现代言情范文指南 (Modern Romance)

> **流派定义**：以现代都市为背景，聚焦男女主角的感情发展，满足读者对理想爱情的向往

---

## 📊 流派特征

### 核心卖点

| 要素 | 说明 | 读者期待 |
|------|------|----------|
| **甜蜜互动** | 男女主的日常撒糖 | 嗑糖快乐 |
| **人设苏感** | 男主完美/女主可爱 | 代入心动 |
| **情感共鸣** | 现实中能触及的感情 | 情绪价值 |
| **HE保证** | 一定是圆满结局 | 安全感 |

### 细分类型

| 类型 | 特点 | 代表作品 |
|------|------|----------|
| **总裁文** | 霸道总裁+普通女主 | 《杉杉来了》 |
| **校园文** | 青涩纯爱 | 《致我们单纯的小美好》 |
| **职场文** | 办公室恋情 | 《何以笙箫默》 |
| **娱乐圈** | 明星恋爱 | 《微微一笑很倾城》 |
| **破镜重圆** | 分手后重逢 | 《原来你还在这里》 |
| **先婚后爱** | 契约/闪婚 | 《亲爱的，热爱的》 |
| **暗恋成真** | 长期暗恋终成眷属 | 《你好，旧时光》 |

## 🎭 人物设定

### 女主类型

| 类型 | 特点 | 适合 | 读者群 |
|------|------|------|--------|
| **软萌型** | 可爱、单纯、治愈 | 总裁文、宠文 | 喜欢被宠的感觉 |
| **独立型** | 有事业、有主见 | 职场文、双强 | 喜欢平等关系 |
| **学霸型** | 聪明、高冷 | 校园、电竞 | 喜欢智慧女主 |
| **元气型** | 活泼、搞笑 | 轻松向、日常 | 喜欢欢乐氛围 |

### 女主人设模板

```yaml
基础信息:
  姓名: 苏念念
  年龄: 22-28岁
  职业: 设计师/医生/程序员/编辑...
  
性格特点:
  外在: 温柔/活泼/高冷
  内在: 善良、有原则、努力
  萌点: 小迷糊/吃货/社恐/工作狂
  
与男主的关系起点:
  - 偶遇/相亲/同事/旧识
  - 误会开场/欢喜冤家/一见钟情
```

### 男主类型

| 类型 | 特点 | 经典标签 |
|------|------|----------|
| **霸道总裁** | 有钱有权，独占欲强 | 冷面热心、宠妻狂魔 |
| **温润学长** | 温柔体贴，默默守护 | 暖男、邻家哥哥 |
| **高冷医生** | 理性克制，反差萌 | 禁欲系、医者仁心 |
| **天才程序员** | 闷骚、专一、宅 | 技术大神、只对你话多 |
| **娱乐圈顶流** | 表面风光，内心孤独 | 只对你露出真面目 |

### 男主人设模板

```yaml
基础信息:
  姓名: 陆景深
  年龄: 28-35岁
  身份: 公司总裁/医生/律师/电竞选手...
  
外在形象:
  外貌: 高大英俊，气场强大
  穿着: 精致讲究
  气质: 冷峻/温润/痞帅
  
性格特点:
  对外: 冷淡/高傲/不近女色
  对女主: 独宠/包容/话多/孩子气
  反差萌: 这是苏点来源！

追妻技能:
  - 默默付出型
  - 直球告白型
  - 制造偶遇型
  - 欲擒故纵型
```

## 📖 剧情模板

### 感情线发展模型

```
阶段1：相遇（1-20章）
├── 特别的第一印象
├── 偶然的再次相遇
├── 被迫产生交集
└── 男主开始注意女主

阶段2：心动（21-60章）
├── 频繁接触
├── 了解彼此
├── 心动瞬间（单方或双方）
├── 暧昧升温
└── 有人先察觉感情

阶段3：确认（61-100章）
├── 告白/表明心意
├── 确定关系
├── 甜蜜日常
└── 第一次危机（小误会）

阶段4：考验（101-150章）
├── 外部阻力（家人反对/情敌）
├── 内部矛盾（价值观冲突）
├── 短暂分离
└── 重新理解对方

阶段5：圆满（151-200章）
├── 解决所有问题
├── 双方成长
├── 求婚/结婚
└── HE番外
```

### 经典桥段（言情必备）

```yaml
心动时刻:
  - 雨中送伞/共撑一把伞
  - 生病照顾
  - 醉酒说真话
  - 意外同居/同床
  - 英雄救美
  - 吃醋发狂

甜蜜互动:
  - 壁咚/头咚/床咚
  - 公主抱
  - 偷偷看对方
  - 牵手/拥抱/亲吻
  - 给对方做饭
  - 接机/送机

虐心时刻:（慎用）
  - 误会分手
  - 前任/情敌出现
  - 家人反对
  - 异地/出国
  
高光时刻:
  - 当众宣誓主权
  - 霸气护妻
  - 深情告白
  - 求婚
```

## ✍️ 语言风格

### 甜文文风

```
对话自然亲密：
"在想什么？"他从背后环住她的腰。
"在想你什么时候回来。"
"想我了？"
"嗯...有一点点。"她心虚地缩了缩脖子。
男人低低地笑了："只有一点点？"

内心戏细腻：
苏念念的心跳漏了一拍。
他离得太近了，近到她能闻到他身上清冽的香水味，
近到她能看清他睫毛的弧度。
她慌张地垂下眼，不敢和他对视。

小动作描写：
他伸手替她理了理被风吹乱的头发，
动作轻柔，仿佛她是什么易碎的珍宝。
苏念念愣住了，一时间忘记了躲开。
```

### 苏感制造

```
男主的宠：
"不想吃就不吃，想吃什么我让人做。"
"冷了？"他皱眉，直接把外套披在她身上。
"以后不准加班到这么晚，知道吗？"

男主的醋：
"那个男的是谁？"
"看来我来的不是时候。"语气冷了下来。
他一言不发地拉着她离开，力道大得让她吃痛。

男主的情话：
"从看到你的第一眼，就知道完了。"
"我这辈子，只你一个。"
"你可以不喜欢我，但不能喜欢别人。"
```

## 📝 范文示例

### 开篇示例

```
【第一章 初遇】

苏念念觉得今天真是倒霉透了。

先是上班迟到被扣了半天工资，
然后咖啡洒在了重要的设计稿上，
现在，她被困在电梯里，已经快二十分钟了。

"喂？物业吗？我被困在电梯里了...什么？要等一个小时？"

她绝望地蹲在角落里，开始怀疑人生。

就在这时，电梯门突然开了一条缝。

她猛地抬头，看到一双修长有力的手正用力扒着门。
门缝越来越大，一个高大的身影出现在面前。

逆光下，她看不清他的脸，只能看到一双深邃的眼睛。

"还愣着干什么？"低沉的嗓音响起，"出来。"

苏念念手忙脚乱地站起来，刚迈出一步就脚软，
整个人朝前栽去——

她闭上眼，预想中的疼痛没有来。

一只有力的手臂揽住了她的腰。

"这么着急投怀送抱？"

苏念念猛地睁眼，撞进一双带着笑意的眼睛里。

完了。

她的脸，瞬间红透了。
```

### 暧昧升温示例

```
【某章 心动时刻】

"你冷吗？"

陆景深突然问道。

苏念念愣了一下，低头看看自己的薄外套，
刚想说"还好"，就感觉肩膀一沉。

他的外套落在她身上，还带着他身上的温度。

"我不——"

"穿着。"他打断她，语气不容拒绝。

苏念念抿了抿唇，没再说什么。

其实她真的不冷，可是...
她偷偷把脸埋进领口，贪婪地嗅着上面的气息。

好像...有点上头。

陆景深余光瞥见她的小动作，嘴角微微上扬。

这只小东西，什么时候能对他坦诚一点呢？

不过...看她这副样子，好像也快了。
```

## ✍️ 经典片段示范

### 【心动瞬间】标准甜文写法

```
苏念念没想到会在这里遇见他。

公司楼下的便利店，凌晨两点，她蓬头垢面地来买泡面，
而陆景深穿着笔挺的西装，手里拿着一瓶矿泉水。

四目相对。

"……"

苏念念下意识想逃，却被他一把拉住。

"这么晚还没回家？"

他的声音带着一丝责备，眉头微微皱起。

"加班……"她小声说，努力把乱糟糟的头发往后拢了拢。

陆景深看着她手里的泡面，眉头皱得更紧了。

"就吃这个？"

"方便嘛……"

他没说话，直接拿过她手里的泡面放回货架，然后拉着她往外走。

"干什么？"苏念念慌了。

"带你吃饭。"

"可是现在凌晨两点——"

"我知道一家店，二十四小时营业。"

他的手很大，很温暖，牢牢地包裹着她的手腕。

苏念念的心跳漏了一拍。

她偷偷抬头看他的侧脸，路灯的光打在他的轮廓上，
棱角分明，像是漫画里走出来的人物。

"看什么？"

他突然偏头，目光正好与她撞上。

苏念念慌忙移开视线，耳根却悄悄红了。

"没……没什么。"

陆景深看着她通红的耳朵，嘴角微微上扬。

这只小东西，什么时候才能对他诚实一点呢？
```

### 【吃醋名场面】男主占有欲

```
"那个男的是谁？"

陆景深的声音很平静，平静得有些吓人。

苏念念愣了一下："什么男的？"

"中午和你吃饭的那个。"

"哦，你说李浩啊，我大学同学——"

"他看你的眼神不对。"

苏念念哭笑不得："哪里不对了？我们就是普通朋友……"

话没说完，她就被他按在了墙上。

陆景深一只手撑在她耳边，低头看她，眼神幽深。

"普通朋友会盯着你看那么久？"

"他只是——"

"普通朋友会把手机壁纸设成你的照片？"

苏念念愣住了。"他的壁纸是我？我不知道……"

"现在知道了。"陆景深的声音更低了，"以后不许单独和他吃饭。"

"陆景深你这是吃醋吗？"苏念念鼓起勇气问。

他没有回答，只是低头在她唇上落下一个霸道的吻。

良久，他松开她，拇指擦过她红肿的嘴唇。

"记住了吗？"

苏念念大脑一片空白，只能傻傻地点头。

陆景深满意地勾起嘴角。

吃醋？

他是在宣示主权。
```

### 【告白时刻】深情直球

```
"苏念念。"

他突然叫她的名字，语气郑重得不像平时。

苏念念抬头，对上他认真的眼神，心跳骤然加速。

"怎、怎么了？"

陆景深深吸一口气，仿佛下定了什么决心。

"我喜欢你。"

四个字，简单直接，却像是一颗石子投入平静的湖面，
激起了巨大的涟漪。

苏念念愣在原地，大脑一片空白。

"从第一次见你开始，"他继续说，声音低沉而坚定，
"电梯里你摔进我怀里的那一刻，我就知道完了。"

"这一年，我一直在等。等你能看到我，等你能接受我。"

"如果你现在还没准备好，我可以再等。"

他看着她，眼里是她从未见过的温柔和忐忑。

"但是苏念念，你要知道——"

"我喜欢你，是认真的。"

苏念念的眼眶突然有些发酸。

原来不是她一个人在心动。

原来，他也是。

"我……"她张了张嘴，声音小得像蚊子，"我也喜欢你。"

陆景深愣了一秒，然后笑了。

那是她第一次见他笑得那么开心，像个得到糖的孩子。

"你说什么？再说一遍。"

"不说了！"苏念念羞得捂住脸。

他轻轻拉下她的手，在她额头落下一个吻。

"没关系，"他的声音带着笑意，"以后有的是时间让你说。"
```

## 📝 模仿指令

> **给 AI 的指令**：
> 
> "请以《何以笙箫默》《微微一笑很倾城》的笔触写作本章。注意以下要点：
> 1. **细节甜蜜**：用小动作和眼神描写制造心动感
> 2. **内心戏细腻**：多写女主的心理活动和小纠结
> 3. **男主苏感**：对别人冷淡，只对女主例外
> 4. **对话自然**：不要太文艺，要像真实情侣说话
> 5. **反差萌**：霸道总裁的小可爱瞬间
> 6. **甜虐平衡**：如果要虐，后面一定要有糖"

## ⚠️ 常见问题

| 问题 | 表现 | 解决方案 |
|------|------|----------|
| **太假太苏** | 男主完美得不像人 | 给一点小缺点（如吃醋） |
| **节奏太慢** | 50章还没确定关系 | 加快感情进度 |
| **虐过头** | 分手N次，误会不断 | 甜虐比例7:3 |
| **女主太作** | 无理取闹让人烦 | 保持可爱，有理有据 |
| **配角抢戏** | 闺蜜/情敌戏份太多 | 聚焦主CP |
