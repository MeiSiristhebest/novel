# 古代言情·宫斗流范文指南 (Palace Drama Romance)

> **流派定义**：以古代宫廷/后宅为背景，女主在复杂的人际斗争中生存、成长、获得爱情的故事

---

## 📊 流派特征

### 核心卖点

| 要素 | 说明 | 读者期待 |
|------|------|----------|
| **权谋博弈** | 后宫/宅斗的智力对决 | 看女主如何智斗对手 |
| **情感纠葛** | 与男主/其他角色的感情 | 甜虐交织，HE收尾 |
| **逆袭成长** | 从底层到巅峰 | 爽感、成就感 |
| **服化道美学** | 华美的古代背景 | 审美享受 |

### 经典作品参考

```
宫斗代表：
- 《甄嬛传》- 经典宫斗，步步为营
- 《如懿传》- 虐心向，情感深度
- 《延禧攻略》- 爽文向，快节奏打脸

宅斗代表：
- 《知否知否》- 宅斗+言情
- 《锦心似玉》- 继室逆袭
- 《庶女攻略》- 庶女上位
```

## 🎭 人物设定

### 女主类型

| 类型 | 特点 | 代表 | 适合 |
|------|------|------|------|
| **隐忍型** | 前期忍辱负重，后期爆发 | 甄嬛 | 长篇、虐文 |
| **爽利型** | 直接反击，快意恩仇 | 魏璎珞 | 爽文、短篇 |
| **智慧型** | 以智取胜，运筹帷幄 | 明兰 | 宅斗、权谋 |
| **佛系型** | 不争不抢，被动卷入 | 沈眉庄 | 配角向 |

### 女主人设模板

```yaml
基础信息:
  姓名: 沈清浅
  身份: 嫡女/庶女/继室/新入宫嫔妃
  出身: 书香门第/将门之后/没落世家
  
性格特点:
  表面: 温婉贤淑/天真烂漫
  内里: 聪慧果断/心有丘壑
  成长: 从单纯到成熟
  
核心能力:
  - 识人之明
  - 隐忍之功
  - 布局之智
  - 人心之术
  
软肋:
  - 在意的人（家人、男主）
  - 不忍的底线（不害无辜）
```

### 男主类型

| 类型 | 特点 | 读者反应 |
|------|------|----------|
| **帝王** | 权力顶峰，多疑深沉 | 又爱又恨 |
| **王爷** | 风流不羁或冷面冰山 | 苏点满满 |
| **将军** | 铁血柔情，战场英雄 | 安全感 |
| **世家公子** | 温润如玉或腹黑深沉 | 少年感 |

### 反派设计

```yaml
主要反派（大boss）:
  身份: 皇后/正室/嫡母/宠妃
  动机: 地位威胁/争宠/家族利益
  手段: 高明、阴狠、有章法
  结局: 最终败北，但要有足够篇幅

次要反派（阶段性boss）:
  身份: 其他嫔妃/庶女/侧室
  动机: 嫉妒/利用
  手段: 相对低级，给女主练手
  结局: 逐一被击败
```

## 📖 剧情模板

### 开局模式

```
模式1：新人入场
"大婚/选秀/入府第一天，所有人都在看新人笑话"
→ 女主初来乍到，被各种刁难，展示智慧

模式2：重生开局
"上一世惨死，这一世绝不重蹈覆辙"
→ 带着记忆重来，规避上辈子的坑

模式3：穿越开局
"一睁眼成了炮灰/书中人物"
→ 知道剧情，利用先知优势

模式4：复仇开局
"家族覆灭/被害惨死，誓要讨回一切"
→ 目标明确，步步为营
```

### 经典剧情线

```
第一阶段：立足（1-100章）
├── 初入场，被刁难
├── 小试牛刀，初步立足
├── 获得第一个盟友
├── 击败第一个对手
└── 引起男主注意

第二阶段：上位（101-300章）
├── 与主要反派正面交锋
├── 感情线发展
├── 遭遇重大危机
├── 绝地反击
└── 地位上升

第三阶段：争锋（301-500章）
├── 成为有力竞争者
├── 与反派多次交手
├── 感情确定
├── 后宫/宅斗白热化
└── 阶段性胜利

第四阶段：巅峰（501-700章）
├── 击败主要对手
├── 登上高位
├── 最终考验（家族/权力/情感）
├── HE结局
└── 番外
```

### 经典桥段

```yaml
必备桥段:
  - 初见惊艳（男主第一次注意女主）
  - 被陷害（反派设局，女主化解）
  - 宴会出彩（重要场合展示才华）
  - 生死相依（共患难，感情升温）
  - 误会虐心（短暂的分离/冷战）
  - 大仇得报（对主要反派的反击）
  - 真相大白（身世/阴谋揭示）
  - 并肩作战（男女主共同对敌）
  
加分桥段:
  - 服饰/妆容碾压
  - 才艺展示（琴棋书画）
  - 怀孕/子嗣线
  - 娘家助力/拖后腿
  - 闺蜜线（真假姐妹）
```

## ✍️ 语言风格

### 文风特点

```
用词典雅:
❌ "她很漂亮"
✅ "她生得花容月貌，顾盼间自有一番风流"

对话古意:
❌ "你干嘛？"
✅ "姐姐这是何意？"

心理细腻:
❌ "她很生气"
✅ "她面上仍是淡淡的笑，指尖却不自觉地攥紧了帕子"

环境渲染:
"廊下挂着的八角宫灯被风吹得轻晃，
映得她的面容忽明忽暗，看不清神色。"
```

### 宫斗专用表达

```
表面恭维实则内涵:
"妹妹真是好福气，这样的恩宠，可是旁人求都求不来的。"
（暗讽：你不过是靠恩宠上位）

笑里藏刀:
"姐姐说的是，妹妹年轻不懂事，日后还要姐姐多提点。"
（实际：你等着，来日方长）

绵里藏针:
"娘娘这簪子好生眼熟，仿佛在哪里见过..."
（暗示：这是别人的东西）
```

## 📝 范文示例

### 开篇示例

```
【第一章 入府】

沈清浅在马车里颠簸了三天，终于看到了顾府那对威严的石狮子。

这是她往后余生的家了。

作为继室嫁入国公府，她知道等待自己的不会是什么好日子。
果然，马车刚停稳，就有一个尖利的声音传来：

"哟，这就是新夫人？怎么就一辆马车，嫁妆呢？"

沈清浅掀开帘子，看到一个穿着桃红比甲的嬷嬷站在那里，
脸上的笑容里满是打量。

她微微一笑，不紧不慢地说道：
"嫁妆不多，三十二抬而已，嬷嬷若不嫌弃，随我一同清点？"

那嬷嬷脸上的笑僵了一瞬——
三十二抬，可不是小数目。

沈清浅提起裙摆，缓步下车，仿佛没看到对方的脸色。

入府的第一步，她不能输。
```

### 宫斗场景示例

```
【请安场景】

坤宁宫内，各位嫔妃齐聚一堂。

皇后端坐主位，目光从众人脸上一一扫过，
落在角落里安静坐着的沈贵人身上。

"沈贵人，听说你昨日得了陛下赏赐的东珠？"

满室皆静。东珠本是皇后专属之物。

沈清浅起身福了福，面色从容：
"回娘娘的话，确有此事。陛下说臣妾侍疾有功，赐了两颗。
臣妾受宠若惊，本想着该孝敬娘娘，又怕僭越，
正不知如何是好，娘娘便提起了。"

说着，她从袖中取出一只锦盒，双手奉上：
"臣妾斗胆，将这两颗东珠献给娘娘，
也算臣妾的一点心意。"

皇后愣了一瞬。

她本想借此给沈贵人难堪，却不料对方主动献上。
收也不是，不收也不是。

"你倒是有心。"皇后干笑了一声，"既是陛下赐的，你便留着吧。"

沈清浅低头行礼，嘴角勾起一抹不易察觉的弧度。

这一局，她赢了。
```

## ✍️ 经典片段示范

### 【绵里藏针】宫斗对话标准写法

```
永寿宫内，茶香袅袅。

惠贵妃端着茶盏，似笑非笑地看着对面的沈贵人。

"妹妹这身衣裳倒是好看，湖绿色，衬得妹妹愈发清丽了。"

沈清浅微微欠身："多谢姐姐夸奖，不过是内务府新送来的料子，
哪里比得上姐姐这身蜀锦，听说是贡品，整个后宫也只有姐姐一匹呢。"

惠贵妃的笑容僵了一瞬。

蜀锦是她托家里人弄来的，本不该张扬，这沈氏如何知道？

"妹妹倒是消息灵通。"她放下茶盏，语气淡了几分。

"姐姐说笑了，"沈清浅低眉顺眼，"臣妾不过是听宫人们闲聊几句。
姐姐家世显赫，这些东西自然是唾手可得，
不像臣妾，连身像样的衣裳都要等着内务府发放。"

"只是……"她抬起头，眼中闪过一丝无辜，"臣妾斗胆提醒姐姐一句，
这蜀锦虽好，却是贡品。若是被太后知道流落到外头……"

惠贵妃脸色骤变。

"妹妹这是在威胁本宫？"

"臣妾不敢。"沈清浅起身福了福，"臣妾只是担心姐姐，
毕竟这后宫里，什么话都有人传，什么事都有人看。
姐姐万金之躯，可不能被这些小事绊住了脚。"

说罢，她盈盈一拜，缓步退出。

留下惠贵妃一个人坐在殿中，脸色阴晴不定。

这个沈氏……绝不是什么省油的灯。
```

### 【请安交锋】群戏标准写法

```
坤宁宫，辰时。

各宫嫔妃依次落座，皇后高坐主位，目光扫过众人。

"今日倒是齐整。"皇后笑道，"难得见沈贵人也来得这样早。"

沈清浅起身行礼："臣妾不敢误了给娘娘请安。"

"坐吧。"

她刚落座，便听到旁边传来一声轻笑。

是淑妃。

"沈贵人可是忙得很呢，听说昨日在御花园逛了整整两个时辰？"

此言一出，众人的目光都聚了过来。

沈清浅心中一凛，面上却不动声色。

"回淑妃娘娘，臣妾昨日是去御花园写生，陛下说那里的芙蓉开得正好。"

"哦？是陛下说的？"淑妃挑眉，"那倒是本宫多虑了。
只是听人说，贵人昨日似乎见了什么人……"

"淑妃是在怀疑本宫？"沈清浅不等她说完，直接打断。

她站起身，目光平静地与淑妃对视：

"臣妾一介贵人，在后宫无依无靠，若是做了什么出格之事，
只怕第二天就被拖出去杖毙了。
淑妃娘娘位分尊贵，若是有什么疑虑，大可派人去查。"

"只是……"她话锋一转，嘴角微微上扬，
"臣妾倒是好奇，是谁在娘娘耳边嚼舌根呢？
后宫之中最忌的就是谣言，若是有人故意挑拨……"

她没有继续说下去，但意思已经很明显了。

淑妃的脸色有些难看。

"你——"

"好了。"皇后终于开口，语气淡淡的，"都是自家姐妹，
说话注意分寸。沈贵人的事，本宫会让人去查，
若是清白，本宫自会为你做主。"

沈清浅盈盈拜下："多谢娘娘。"

皇后点点头，目光意味深长地在她身上停留了一瞬。

这个沈氏……倒是有几分手段。
```

### 【帝妃情深】感情线高光

```
养心殿，深夜。

沈清浅跪在殿外，已经两个时辰了。

膝盖已经麻木，脊背却依然挺直。

她知道，皇上在生气。

因为她擅自放走了那个本该被处死的宫女——
那宫女是被人陷害的，她查到了证据，却来不及呈上。

"沈氏，你可知罪？"

殿门终于打开，皇上的声音从里面传来。

沈清浅没有抬头，只是平静地说：

"臣妾知罪。但臣妾不悔。"

"不悔？"皇上走到她面前，居高临下地看着她，
"你可知抗旨是什么罪？就算朕宠你，也保不住你！"

"臣妾知道。"她终于抬起头，眼中没有惧色，只有坦然，
"但那宫女是无辜的。臣妾若是眼睁睁看着她死，
以后还有何面目侍奉陛下？"

皇上沉默了。

良久，他叹了口气，弯腰将她扶起。

"你这性子……"他摇头，语气里却没有多少怒意，
"早晚要吃亏。"

"臣妾……"

"行了，"他打断她，"朕已经派人去查了。
若真如你所说，那宫女自然无罪。"

他看着她红肿的膝盖，眉头皱起：

"下次若再有这种事，先来告诉朕。
别动不动就跪，朕不喜欢。"

沈清浅心中一暖，低头掩去眼中的湿意。

"臣妾遵旨。"

皇上伸手将她打横抱起，大步走进殿内。

"陛下！"沈清浅惊呼。

"别动。"他低头看她，眼中带着一丝笑意，
"膝盖伤成这样，还想自己走？"

沈清浅红着脸，不敢再说话。

只是悄悄地，把头埋进了他的胸口。

这个人，明明那么威严，对她却总是这样温柔。
```

## 📝 模仿指令

> **给 AI 的指令**：
> 
> "请以《甄嬛传》《知否知否》的笔触写作本章。注意以下要点：
> 1. **对话要有机锋**：表面客气，内里藏刀，每句话都有潜台词
> 2. **用词典雅**：避免现代词汇，多用古风表达
> 3. **心理细腻**：少用形容词，多用动作神态暗示情绪
> 4. **群戏层次**：多人场景要有主次，各怀心思
> 5. **智斗为主**：减少直接冲突，增加智力博弈
> 6. **感情线细腻**：帝妃/夫妻感情要有张力和深度"

## ⚠️ 常见问题

| 问题 | 表现 | 解决方案 |
|------|------|----------|
| **女主太圣母** | 一直原谅敌人 | 给女主底线，触碰必反击 |
| **斗争太低级** | 互扇巴掌 | 增加智斗元素 |
| **男主工具人** | 只在关键时刻出场 | 加强感情线戏份 |
| **虐太过分** | 读者弃文 | 虐完必须有糖 |
| **节奏太慢** | 铺垫太长 | 前10章必须有爽点 |
