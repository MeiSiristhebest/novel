# 凡人流风格范文 (Fanren Style Exemplar)

> **流派定位**：以《凡人修仙传》为开创者，强调"没有主角光环"的修仙生存法则。主角资质平庸，依靠谨慎、算计、机缘步步为营。

---

## 🎯 核心风格特征

### 1. 资源意识（核心逻辑）

凡人流的世界观建立在"资源稀缺"之上：

```
【修仙资源】
灵石 → 功法 → 丹药 → 法器 → 洞府 → 机缘
   ↓
全部都要抢！
```

**资源公式**：
```
资质差 + 资源少 = 必须算计
资质好 + 资源多 = 才能挥霍
```

### 2. 生存法则（行为准则）

| 法则 | 说明 | 例子 |
|------|------|------|
| **不露富** | 财不外露，底牌藏好 | 韩立从不展示全部实力 |
| **不冒险** | 七成把握才出手 | 没把握就跑，绝不硬刚 |
| **不树敌** | 能不得罪就不得罪 | 杀人必灭口，斩草除根 |
| **不心软** | 修仙界没有善恶 | 对敌人仁慈就是对自己残忍 |
| **不贪婪** | 吃得下才拿 | 机缘再好，命没了也白搭 |

### 3. 修仙残酷（世界观基调）

```
【凡人流真相】
- 99%的修士死在筑基前
- 金丹期就是一方霸主
- 元婴期已是传说级别
- 每一次突破都是生死关
- 没有主角光环，死了就是死了
```

### 4. 语言特点

- **冷静克制**：少用感叹号，多用陈述句
- **内心算计**：大段心理活动，权衡利弊
- **数字精确**：灵石数量、突破概率、寿命年限
- **不煽情**：杀人不眨眼，死人不悲伤

## ✍️ 经典片段示范

### 【资源困境】修仙的代价

```
韩立看着手中仅剩的三颗筑基丹，眉头紧锁。

按照宗门典籍的记载，以他中下等的灵根资质，突破筑基期的成功率约在三成左右。服用筑基丹可以将成功率提升到五成，但若失败，修为将倒退一个小境界，且三年内无法再次尝试。

三颗筑基丹。

也就是说，他只有三次机会。

如果三次都失败了……

韩立闭上眼睛，默默计算。

他今年三十七岁。练气期修士的寿命极限是一百二十年。扣除修炼所需的时间，他最多还能尝试突破十五次。

但筑基丹的价格是三百块下品灵石一颗。

黄枫谷的俸禄，每年只有五十块。

也就是说，就算他不吃不喝，六年才能攒一颗筑基丹。

十五次机会？

他最多只有两次。

"必须另想办法。"

韩立睁开眼睛，目光落在桌上那份悬赏令上。

乱星海，万妖海域。

据说那里有一处上古修士的洞府遗迹。消息已经走漏，各方势力蠢蠢欲动。

九死一生。

但如果能从中获取资源，他的困境就能迎刃而解。

韩立沉默了很久。

最终，他将筑基丹收入储物袋，拿起了那份悬赏令。

修仙界从来没有稳妥的路。

要么冒险，要么等死。

他选择冒险。
```

**风格要点**：
- 数字量化困境（三颗、三成、三十七岁、一百二十年）
- 冷静的利弊分析
- 没有热血口号，只有理性抉择
- 结尾点明"冒险vs等死"的核心矛盾

### 【算计博弈】杀人夺宝

```
韩立跟踪这个练气九层的修士已经三天了。

对方叫王鸿，散修，手中有一株他急需的千年灵药。

三天来，韩立摸清了对方的实力、习惯和路线。

王鸿的实力在练气九层中属于中等偏上，擅长火系法术，有一件中品法器——赤焰刀。性格谨慎，每次休息都会布置警戒法阵。

正面交手，韩立只有四成胜算。

不够。

他需要至少七成。

第四天，机会来了。

王鸿在穿越一处峡谷时，遭遇了一头三阶妖兽的袭击。双方缠斗了半个时辰，王鸿虽然斩杀了妖兽，但也身受重伤，法力消耗了七成。

韩立就藏在三百丈外的一块巨石后面。

他没有动。

因为王鸿还有三成法力，加上那件中品法器，依然不容小觑。

又过了两个时辰。

王鸿服下丹药，恢复了一成法力，开始收取妖兽的妖丹。

就是现在。

韩立祭出十二口青竹蜂云剑，无声无息地向王鸿飞去。

王鸿察觉到危险的瞬间，十二道剑光已经将他团团围住。

"你——！"

他想要说什么，但韩立没有给他机会。

剑光暴涨，瞬间将王鸿绞成了碎片。

韩立落到地面，面无表情地收起储物袋。

他没有任何愧疚。

在修仙界，弱肉强食是唯一的法则。

今天他杀了王鸿，明天可能就有人来杀他。

这就是修仙。
```

**风格要点**：
- 耐心等待最佳时机（三天跟踪+两个时辰等待）
- 详细的实力评估（四成→七成）
- 出手果断，不废话
- 杀人后没有内疚，只有冷静
- 结尾点题：弱肉强食

### 【突破生死】筑基关口

```
洞府深处，韩立盘膝而坐。

面前摆着三样东西：一颗筑基丹，一枚聚灵阵盘，一份遗书。

遗书是写给墨大夫的。

如果他死在这里，至少要让人知道他的尸体在哪。韩家还有几个晚辈，或许能从他的遗物中得到一些好处。

这是他最后的算计。

做完这一切，韩立深吸一口气，将筑基丹送入口中。

丹药入腹，一股狂暴的灵力在他体内炸开。

疼痛如潮水般涌来。每一条经脉都像是被火烧灼，每一寸肌肤都在颤抖。他的意识开始模糊，仿佛有什么东西正在将他的灵魂撕成碎片。

"稳住……稳住……"

他在心中默念，拼命引导着狂暴的灵力冲击那道看不见的屏障。

一次。

失败。

灵力回涌，他喷出一口鲜血。

二次。

失败。

丹田传来撕裂般的剧痛。

三次。

四次。

……

第七次。

咔嚓。

那道屏障，终于出现了一丝裂缝。

韩立双眼猛然睁开，燃烧起血红色的光芒。他将所有的灵力、精神、甚至生命力，全部灌注到这一击——

轰！

屏障碎裂。

灵力如洪水般涌入他的丹田，一颗米粒大小的金丹雏形，缓缓成型。

韩立瘫倒在地，浑身被汗水和鲜血浸透。

他成功了。

筑基。

从今日起，他不再是那个随时可能死去的练气期散修。

他是筑基期修士了。

韩立躺在地上，嘴角缓缓勾起一抹笑容。

这是他三十年来，第一次笑得如此畅快。
```

**风格要点**：
- 准备遗书，凸显生死未卜
- 突破过程艰难（七次冲击）
- 痛苦描写真实，不轻描淡写
- 成功后的喜悦克制但真实
- "三十年来第一次"强化积累感
