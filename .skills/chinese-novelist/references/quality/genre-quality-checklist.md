# 流派专项质检清单 (Genre-Specific Quality Checklist)

> **维护者**：Chinese Novelist Pro Team
> **适用范围**：流派专项质检
> **状态**：Active
> **最后校验**：2026-03-01

> 本清单是对《质量检查清单》的补充，针对不同流派的特殊要求

---

## 一、宫斗流专项检查

### 核心要素检查

- [ ] **权力层级明确**
  - 后宫位份清晰（皇后→贵妃→妃→嫔→贵人→常在→答应）
  - 人物间权力关系明确
  - 宫规礼仪描写准确

- [ ] **绵里藏针的对话**
  - 每场重要对话有表面含义和潜台词
  - 避免直接骂战，要"笑里藏刀"
  - 对话推动势力博弈

- [ ] **势力分布清晰**
  - 主要势力阵营明确（皇后党、贵妃党等）
  - 女主的位置和策略清晰
  - 配角立场有迹可循

### 情节检查

- [ ] **以退为进的智斗**
  - 女主不是莽撞对抗
  - 有隐忍、蛰伏、反击的节奏
  - 每次胜利都有代价或风险

- [ ] **感情线与权谋线交织**
  - 男主/感情线不喧宾夺主
  - 感情发展服务于生存/上位
  - 情感抉择有利益考量

- [ ] **危机层层升级**
  - 从小陷害到大阴谋
  - 敌人级别逐步提升
  - 女主实力同步增长

### 常见错误检查

| 错误 | 正确做法 |
|-----|---------|
| ❌ 女主开挂无敌 | ✅ 有失败有挫折，险中求胜 |
| ❌ 男主一见钟情无脑宠 | ✅ 感情发展有过程有波折 |
| ❌ 反派智商下线 | ✅ 反派有能力有手段 |
| ❌ 宫斗靠打架 | ✅ 宫斗靠心计和布局 |

## 二、历史穿越专项检查

### 核心要素检查

- [ ] **历史节点明确**
  - 穿越的时间点清晰
  - 未来走向（主角知道的）有交代
  - 时间紧迫感有体现

- [ ] **现代知识合理运用**
  - 使用的技术/知识在当时可实现
  - 材料和工具符合时代
  - 不是一穿越就开挂

- [ ] **古人智商在线**
  - 古人不是傻子
  - 主角的"先知"会被质疑
  - 要有说服/证明的过程

### 情节检查

- [ ] **蝴蝶效应意识**
  - 主角行动会改变历史
  - 改变带来新的变数
  - 不是照着历史走一遍

- [ ] **资源积累有过程**
  - 第一桶金怎么来
  - 人才如何招揽
  - 势力如何发展

- [ ] **身份掩护合理**
  - 主角的现代思维如何解释
  - 不合时宜的言行如何遮掩
  - 被怀疑时如何应对

### 常见错误检查

| 错误 | 正确做法 |
|-----|---------|
| ❌ 第一天就造蒸汽机 | ✅ 用当时能找到的材料 |
| ❌ 古人完全听不懂 | ✅ 换一种方式表达 |
| ❌ 历史人物是工具人 | ✅ 有自己的想法和选择 |
| ❌ 用现代网络用语 | ✅ 半文半白，符合时代 |

## 三、仙侠凡人流专项检查

### 核心要素检查

- [ ] **资质设定明确**
  - 灵根品级清晰（天/地/玄/黄）
  - 资质带来的劣势具体化
  - 吸收效率有数字锚定

- [ ] **资源系统量化**
  - 灵石作为基础货币
  - 收入与支出有对比
  - 每个选择都是资源博弈

- [ ] **修炼体系清晰**
  - 境界划分明确
  - 突破条件/瓶颈有描写
  - 功法/丹药/法器有价格

### 情节检查

- [ ] **以智取胜**
  - 不是蛮干硬拼
  - 有观察、分析、规划
  - 用脑子弥补天赋

- [ ] **投入产出意识**
  - 每个决策都有成本考量
  - 风险与收益并行
  - 不浪费任何资源

- [ ] **步步为营**
  - 小目标积累大成就
  - 有长期规划
  - 不贪功冒进

### 常见错误检查

| 错误 | 正确做法 |
|-----|---------|
| ❌ 下品灵根随便突破 | ✅ 每次突破都很艰难 |
| ❌ 资源随便获取 | ✅ 每颗灵石都有来源 |
| ❌ 敌人送装备送功法 | ✅ 机缘要自己争取 |
| ❌ 主角无脑冲动 | ✅ 谨慎算计步步为营 |

## 四、都市异能专项检查

### 核心要素检查

- [ ] **异能设定合理**
  - 异能有明确边界
  - 有弱点或限制条件
  - 不是无敌挂

- [ ] **都市背景真实**
  - 现代社会规则存在
  - 主角还要正常生活
  - 隐藏身份的必要性

- [ ] **势力分布**
  - 官方（异能管理机构）
  - 民间（异能者组织）
  - 敌对（反派势力）

### 常见错误检查

| 错误 | 正确做法 |
|-----|---------|
| ❌ 当街放技能无后果 | ✅ 要隐藏身份 |
| ❌ 警察是摆设 | ✅ 官方有约束力 |
| ❌ 异能无消耗 | ✅ 有体力/精神消耗 |

## 五、玄幻爽文专项检查

### 核心要素检查

- [ ] **打脸节奏明确**
  - 蔑视→对决→反转→打脸
  - 旁观者反应到位
  - 打脸后有余韵

- [ ] **等级压制清晰**
  - 高一级=碾压
  - 越级战斗有代价
  - 等级差距有数字感

- [ ] **金手指逐步升级**
  - 不是一开始就无敌
  - 金手指有成长
  - 代价与收获平衡

### 常见错误检查

| 错误 | 正确做法 |
|-----|---------|
| ❌ 反派太弱无挑战 | ✅ 反派实力逐步提升 |
| ❌ 打脸太多审美疲劳 | ✅ 打脸方式要有变化 |
| ❌ 女主角都是花瓶 | ✅ 女性角色有自己的故事线 |

## 六、系统流/LitRPG专项检查

### 核心要素检查

- [ ] **系统规则明确**
  - 任务类型清晰（主线/支线/日常）
  - 奖惩机制明确（经验/装备/属性点）
  - 面板数据可视化（数据不要太水）

- [ ] **成长反馈及时**
  - 付出努力立刻有回报
  - 升级/装备更换节奏紧凑
  - 越级挑战的收益要高

- [ ] **游戏与现实的互动**
  - 游戏能力如何影响现实（如有）
  - 玩家群体反应真实（论坛/公会）
  - 经济系统逻辑自洽

### 常见错误检查

| 错误 | 正确做法 |
|-----|---------|
| ❌ 系统无所不能 | ✅ 系统有边界和BUG，可被利用 |
| ❌ 数据通货膨胀 | ✅ 前期数值要克制，有锚定 |
| ❌ 只有打怪没有剧情 | ✅ 任务要驱动剧情发展 |

---

## 七、克苏鲁/诡秘流专项检查

### 核心要素检查

- [ ] **氛围营造**
  - 五感描写到位（气味、触感、声音）
  - 环境压抑、灰暗、神秘
  - 未知恐惧的侧面烘托

- [ ] **San值（理智）系统**
  - 接触神秘必有代价（精神污染）
  - 疯狂与理智的拉锯战
  - 锚点的设定与作用

- [ ] **神秘学规则严谨**
  - 仪式流程、魔药配方详细
  - 等价交换原则
  - 力量体系层级分明（序列/位阶）

### 常见错误检查

| 错误 | 正确做法 |
|-----|---------|
| ❌ 像普通玄幻一样打架 | ✅ 战斗靠信息差、准备和诡计 |
| ❌ 直视神灵无后果 | ✅ 不可直视神，看一眼就疯 |
| ❌ 主角无敌不疯 | ✅ 主角时刻在疯狂边缘试探 |

---

## 八、洪荒流专项检查

### 核心要素检查

- [ ] **宏大叙事感**
  - 时间跨度大（元会/量劫）
  - 人物跟脚深厚（先天神圣）
  - 宝物等级明确（先天/后天/至宝）

- [ ] **因果与算计**
  - 任何行动都讲因果
  - 圣人算计深远（布局未来）
  - 抢夺气运/功德的逻辑清晰

- [ ] **尊重原著设定**
  - 三清、女娲、西方二圣性格符合传统
  - 重大事件（紫霄宫讲道、巫妖大战）节点准确
  - 逆天改命的合理性（变数）

### 常见错误检查

| 错误 | 正确做法 |
|-----|---------|
| ❌ 说话像现代流氓 | ✅ 语言半文半白，有道韵 |
| ❌ 随便杀圣人 | ✅ 圣人不死不灭，只能封印/落面皮 |
| ❌ 忽视天道大势 | ✅ 顺势而为，小势可改 |

## 九、民俗志怪恐怖专项检查

### 核心要素检查

- [ ] **禁忌规则可复述**
  - 至少一条规则可被角色完整复述
  - 规则来源明确（祖训/碑文/口供）
  - 规则不是临时编造

- [ ] **仪式步骤可执行**
  - 步骤顺序清晰，不可随意替换
  - 每一步有可感知细节（声/味/触）
  - 失败后果可回溯到具体步骤

- [ ] **现实层与诡异层并存**
  - 有现实惩罚（人情/利益/舆论）
  - 有超自然偏差（错位/失声/倒影异常）
  - 两层互相推动，而非各写各的

### 常见错误检查

| 错误 | 正确做法 |
|-----|---------|
| ❌ 只靠鬼脸惊吓 | ✅ 先立规矩，再让破戒后果分批到账 |
| ❌ 主角无脑破戒 | ✅ 给出现实动机（救人/求财/查案） |
| ❌ 一次性解释所有真相 | ✅ 保留可追查线索，分卷揭示 |

## 使用指南

1. **先完成基础质检清单**（质量检查清单.md）
2. **再完成流派专项清单**（本文档）
3. **两份评分相加**，基础满分80 + 专项满分40 = 总分120
4. **交付标准**：总分 > 90 可交付，> 100 优秀
