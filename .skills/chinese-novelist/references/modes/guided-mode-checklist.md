# 引导模式检查清单 (Guided Mode Checklist)

> **触发词**：`我是新手` / `带我入门` / `引导模式` / `开新书`
> **核心行为**：5 轮对话收集信息 → 渐进式生成 8 个文件

---

## 📋 访谈检查清单

AI **必须**逐项完成以下访谈，才能开始生成文件：

### Phase 1: 信息收集

- [ ] **Q1: 核心概念**
  - 询问："你想写一个什么样的故事？用一句话描述。"
  - 收集：一句话梗概 (Logline)

- [ ] **Q2: 流派与风格**
  - 询问："这是什么类型的故事？（玄幻/都市/言情/悬疑...）"
  - 收集：流派、目标读者、节奏偏好

- [ ] **Q3: 主角设定**
  - 询问："你的主角叫什么名字？TA 最想要什么？TA 有什么致命缺陷？"
  - 收集：名字、终极目标、幽灵/创伤

- [ ] **Q4: 世界观**
  - 询问："故事发生在什么样的世界？有什么力量体系？使用力量有什么代价？"
  - 收集：世界类型、力量体系、代价法则

- [ ] **Q5: 确认**
  - 输出：汇总所有收集的信息
  - 询问："以上信息正确吗？确认后我将开始生成项目文件。"
  - 等待用户确认

### Phase 2: 渐进式生成

**所有访谈完成后**，按以下顺序逐个生成：

```
[1/8] 生成 00-策划书.md → 确认
[2/8] 生成 01-世界观圣经.md → 确认
[3/8] 生成 02-人物深度档案.md → 确认
[4/8] 生成 03-分章细纲.md → 确认
[5/8] 生成 04-伏笔追踪表.md → 确认
[6/8] 生成 05-时间线.md → 确认
[7/8] 生成 06-素材碎片.md → 确认
[8/8] 生成 07-项目状态.md → 完成
```

---

## ⚠️ 禁止行为

1.  **禁止跳过访谈**：不得在收集完信息前生成任何文件。
2.  **禁止批量生成**：除非用户明确说"全部生成"，否则必须逐个确认。
3.  **禁止假设**：如果用户没有回答某个问题，必须追问，不得自行填充。
