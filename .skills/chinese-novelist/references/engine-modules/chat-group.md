# 高流量引擎包：聊天群流 (Chat Group System)

> **核心逻辑**：跨界交流 -> 信息差/资源置换 -> 影响力扩张。群友都是大佬，主角是群主/管理员/或者装作大佬的萌新。

---

## 1. 结构发动机 (Structure Engine)

### 核心循环 (The Loop)
1.  **入群**: 主角意外加入一个全是神仙/历史名人的群。
2.  **伪装/立威**: 主角利用现代知识或特殊物品（如辣条、可乐）震惊群友，确立"大能"形象。
3.  **交易**: 用低价值物品换取群友的高价值红包（功法、丹药、运气）。
4.  **奔现**: 群友穿越过来，或者主角去群友的世界，解决危机。

### 黄金三章 (The Golden Three)
1.  **Chapter 1: 误入神群**
    *   **事件**：手机突然震动，加入"修真聊天群"或"万界交流群"。
    *   **冲突**：以为是中二病患者在Cosplay，准备退群。
    *   **钩子**：抢到一个红包，发现提取出来的是真的修仙物品（雷击木/筑基丹）。

2.  **Chapter 2: 忽悠瘸了**
    *   **事件**：群里有人遇到困难（如：炼丹炸炉/国家治理）。
    *   **行动**：主角随口说了几句现代科学/历史知识（"火药配方"、"水法炼丹"）。
    *   **反馈**：群友大受震撼，奉为真理，疯狂发红包感谢。

3.  **Chapter 3: 现实反馈**
    *   **事件**：主角在现实中遇到麻烦（校花被欺负/小混混找茬）。
    *   **高潮**：使用群里抢到的能力/物品，轻松解决麻烦，震惊路人。
    *   **收尾**：群里再次艾特主角，又有新任务/新成员入群。

## 2. 爽点回收表 (Payoff Checklist)

| 铺垫 (Setup) | 回收 (Payoff) |
| :--- | :--- |
| 主角随手发一张自拍（背景是现代都市） | 群友以为是"末法时代的上古遗迹"或"仙界"，顶礼膜拜 |
| 群友A说自己是秦始皇，缺钱 | 主角打过去100块，秦始皇送来一颗长生不老药 |
| 主角被反派包围 | 摇人，群里的孙悟空/通天教主跨界投影一巴掌拍死反派 |
| 群友争论谁更强 | 主角作为群主/裁判，制定规则，让双方都不敢造次 |

---

## 3. 毒点黑名单 (Toxic Tropes to Avoid)

*   ❌ **即使发现了也不信**: 抢了红包拿了实物，还坚持认为是特效/魔术，强行降智。
*   ❌ **群友太弱**: 既然是金手指，群友必须是各自世界的顶尖战力，不能是杂鱼。
*   ❌ **跪舔群友**: 主角作为连接点，地位应该是平等的甚至超然的，不能变成群友的跑腿小弟。
*   ❌ **掉马太快**: 必须维持"大佬"的人设，直到主角真实实力追上来。
