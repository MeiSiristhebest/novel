# 高流量引擎包：赛博边缘 (Cyberpunk Runner)

> **核心逻辑**：高科技低生活 -> 义体改造 -> 对抗公司。在极度压抑的霓虹城市中，为了自由或生存而战。

---

## 1. 结构发动机 (Structure Engine)

### 核心循环 (The Loop)
1.  **接单**: 中间人发布委托（偷取数据/刺杀高管/护送货物）。
2.  **准备**: 更换义体，准备黑客程序，加载武器模组。
3.  **潜入/突袭**: 骇入安保系统，开启斯安威斯坦（时间减缓），暴力突破。
4.  **撤离与代价**: 完成任务，但可能被通缉，或者义体过热导致赛博精神病发作。

### 黄金三章 (The Golden Three)
1.  **Chapter 1: 阴沟里的老鼠**
    *   **环境**：霓虹灯下的贫民窟，酸雨，全息广告牌。
    *   **主角**：一个非法义体医生，或者一个欠了高利贷的雇佣兵。
    *   **危机**：帮派分子上门催债，或者为了救妹妹急需一百万欧金。

2.  **Chapter 2: 唯一的出路**
    *   **事件**：接到一个"自杀式"任务，窃取荒坂/公司的新型原型机。
    *   **装备**：主角安装了一个未经测试的军用级义体（本来是死人的遗物）。
    *   **表现**：义体启动，世界变慢了，主角感受到了力量的代价（剧痛/幻听）。

3.  **Chapter 3: 第一滴血**
    *   **高潮**：任务执行中。原本必死的局面，主角靠着新义体大开杀戒。
    *   **反转**：拿到了原型机，但发现那是一个人（意识芯片）。
    *   **收尾**：主角站在摩天大楼顶端，看着脚下的城市，"既然活下来了，就要把这城市烧成灰"。

## 2. 爽点回收表 (Payoff Checklist)

| 铺垫 (Setup) | 回收 (Payoff) |
| :--- | :--- |
| 公司狗高高在上，视人命如草芥 | 主角骇入他们的脑机接口，让他们在会议室里集体跳舞/自爆 |
| 普通人面对改造人毫无还手之力 | 主角展现出"肉体苦弱，机械飞升"的极致暴力美学 |
| 传说中的赛博精神病无人能治 | 主角靠着特殊的意志力（或金手指），驾驭了疯狂，成为完美兵器 |
| 只有黑客能对抗黑客 | 网域空间（赛博空间）里的史诗级数据对决 |

## 3. 毒点黑名单 (Toxic Tropes to Avoid)

*   ❌ **名词堆砌**: 太多生造词会让读者晕。用通俗的描述（如：电子脑、机械臂）。
*   ❌ **过早推翻公司**: 公司是庞然大物，主角前期只能是"老鼠"，只能在夹缝中生存，不能一上来就灭了公司。
*   ❌ **失去人性**: 虽然是非人改造，但内核必须是人。如果完全变成了机器，就没有共情了。
*   ❌ **黑客无敌**: 物理断网必须能克制黑客，否则黑客坐在家里就无敌了，打斗没法写。
