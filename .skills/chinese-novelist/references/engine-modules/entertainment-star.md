# 高流量引擎包：文娱巨星 (Entertainment Star)

> **核心逻辑**：文抄公 -> 降维打击 -> 文化输出。利用地球的文化瑰宝（周杰伦、金庸、好莱坞）征服异世界/平行世界。

---

## 1. 结构发动机 (Structure Engine)

### 核心循环 (The Loop)
1.  **困境**: 没钱/过气/被雪藏，急需一个作品翻身。
2.  **搬运**: 从记忆中搜索一首/部经典作品（《消愁》、《射雕》、《泰坦尼克号》）。
3.  **发布**: 在质疑声中发布作品。
4.  **轰动**: 销量/票房/点击率爆炸，打脸原来的质疑者。

### 黄金三章 (The Golden Three)
1.  **Chapter 1: 烂摊子**
    *   **事件**：穿越成被公司解约的小鲜肉，或者离婚带娃的落魄大叔。
    *   **冲突**：前经纪人/前妻嘲讽："你已经完了，这辈子都别想红。"
    *   **机会**：一个选秀节目/征文比赛正在报名。

2.  **Chapter 2: 惊艳全场**
    *   **事件**：登上舞台，并没有唱现在的口水歌。
    *   **行动**：拿起吉他，弹唱一首民谣（如《成都》或《曾经的你》）。
    *   **反馈**：评委惊呆，观众流泪，直播间弹幕爆炸。

3.  **Chapter 3: 发酵与打脸**
    *   **事件**：歌曲一夜爆红，冲上热搜第一。
    *   **高潮**：之前的公司想蹭热度/求复合，主角冷冷拒绝。
    *   **收尾**：主角在此刻意识到，自己背后站着整个地球的文娱库。

## 2. 爽点回收表 (Payoff Checklist)

| 铺垫 (Setup) | 回收 (Payoff) |
| :--- | :--- |
| 这个世界的武侠小说只有打打杀杀 | 主角写出《射雕英雄传》，提出"侠之大者，为国为民"，文学界地震 |
| 大家都说华语乐坛已死 | 主角抛出"中国风"歌曲（青花瓷），开创一个时代 |
| 外国电影霸占票房 | 主角拍出《流浪地球》或《哪吒》，开启国产电影元年 |
| 某著名毒舌乐评人准备开喷 | 听完歌后痛哭流涕，当场转粉写千字长评 |

## 3. 毒点黑名单 (Toxic Tropes to Avoid)

*   ❌ **记忆模糊**: 主角必须记得很清楚，或者有系统帮忙复刻，不能出现"记不清歌词"这种低级失误。
*   ❌ **魔改原著**: 除非是为了适应异世界背景，否则不要随意修改经典作品的核心，由于读者认可的是原版。
*   ❌ **全能太快**: 今天唱歌，明天拍电影，后天写诗。要有一个主线（如先称霸乐坛，再进军影坛）。
*   ❌ **现实人物影射**: 尽量用架空人名，避免法律风险和粉丝撕逼。
