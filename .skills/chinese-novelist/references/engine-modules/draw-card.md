# 高流量引擎包：抽卡流 (Gacha/Draw Card System)

> **核心逻辑**：随机性 -> 赌徒心理 -> 资源管理的极致。一切能力全靠抽，欧皇与非酋的二象性。

---

## 1. 结构发动机 (Structure Engine)

### 核心循环 (The Loop)
1.  **积攒资源**: 打怪/做任务获得抽卡资源（钻石/源石/寿命）。
2.  **抽卡环节**: 极具仪式感的抽卡描写（金光！彩光！）。
3.  **获得能力**: 抽到了一张看起来很废但其实很强的卡（R卡），或者直接神卡（SSR）。
4.  **实战应用**: 用卡牌的能力组合解决难题（卡组构建）。

### 黄金三章 (The Golden Three)
1.  **Chapter 1: 绝境首抽**
    *   **事件**：主角穿越到危险世界，手无缚鸡之力，即将被杀。
    *   **机制**：由于是新手，系统赠送一发十连或者是单抽。
    *   **钩子**：按下抽卡键，天空出现异象。

2.  **Chapter 2: 出货了！**
    *   **事件**：只有一张SSR，其他的全是垃圾N卡（如：马桶搋子、清洁剂）。
    *   **能力**：这张SSR是一个强力人物召唤卡，或者一个规则级道具。
    *   **反杀**：利用刚抽到的东西，以一种荒诞或炫酷的方式反杀敌人。

3.  **Chapter 3: 卡组规划**
    *   **事件**：危机暂时解除，主角研究系统卡池。
    *   **期待**：卡池里还有"盘古斧"、"歼星舰"，主角定下目标——搞钱抽卡。
    *   **收尾**：使用了那张看起来没用的N卡（清洁剂），意外发现它能清除诅咒。

## 2. 爽点回收表 (Payoff Checklist)

| 铺垫 (Setup) | 回收 (Payoff) |
| :--- | :--- |
| 只有最后一发单抽的机会 | 逆天改命，单抽出奇迹（如果不中就死，极限拉扯） |
| 敌人嘲笑主角也是一种战术 | 主角抽出"被动反伤卡"，站着不动弹死敌人 |
| 抽到一张说明不明的废卡 | 在特定环境（如水下）这张卡变成了神卡 |
| 敌人以为主角是战士 | 主角抽出一把法杖，我其实是法师哒！ |

---

## 3. 毒点黑名单 (Toxic Tropes to Avoid)

*   ❌ **全图鉴太快**: 不要让主角太容易抽到所有好东西，要有"非酋"时刻，增加趣味性。
*   ❌ **卡牌强度失衡**: 抽出一个无敌卡，后面就不用写了。卡牌要有冷却、限制或消耗。
*   ❌ **不仅抽不中还倒扣**: 所谓"谢谢惠顾"。在生死关头抽到谢谢惠顾是会劝退读者的，至少给个安慰奖。
*   ❌ **复杂的TCG规则**: 不要照搬万智牌/游戏王的复杂规则，网文读者看不懂。规则要简单粗暴（比大小/属性克制）。
