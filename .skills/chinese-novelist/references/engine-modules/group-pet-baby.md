# 高流量引擎包：团宠萌宝 (Group Pet/Baby)

> **核心逻辑**：凄惨开局 -> 被大佬收养 -> 治愈全家。女主负责可爱/懂事，全家负责宠/解决麻烦。

---

## 1. 结构发动机 (Structure Engine)

### 核心循环 (The Loop)
1.  **遭遇**: 萌宝遇到小麻烦（想要糖/被坏人欺负）。
2.  **召唤**: 哥哥/爸爸/叔叔们闻讯赶来（全员大佬）。
3.  **解决**: 用夸张的方式解决问题（为了买糖买下糖果厂）。
4.  **治愈**: 萌宝的一个亲亲/抱抱，化解了家庭内部的矛盾或大佬的暴躁。

### 黄金三章 (The Golden Three)
1.  **Chapter 1: 垃圾桶里的小可怜**
    *   **事件**：三岁半的女主被反派后妈/人贩子丢弃在雪地里。
    *   **相遇**：首富爸爸/霸总哥哥路过，本该冷漠离开，却鬼使神差地停下了车。
    *   **互动**：女主伸出脏兮兮的小手，喊了一声"粑粑/哥哥"。大佬的心化了。

2.  **Chapter 2: 入住豪门**
    *   **事件**：带回家。家里其他成员（二哥、三哥...）表示反对，讨厌小孩。
    *   **冲突**：女佣/管家试图欺负女主，被女主无意中化解或被发现。
    *   **转折**：女主主动关心那个最凶的哥哥（"哥哥不痛"），哥哥沦陷。

3.  **Chapter 3: 第一次打脸**
    *   **事件**：曾经虐待女主的人找上门来，或者参加宴会有人嘲笑女主是野种。
    *   **高潮**：全家出动。爸爸："这是我的掌上明珠。" 哥哥们争着护短。
    *   **收尾**：坏人被扔出去。女主成为了这个豪门家族唯一的"小祖宗"。

## 2. 爽点回收表 (Payoff Checklist)

| 铺垫 (Setup) | 回收 (Payoff) |
| :--- | :--- |
| 残疾/失眠/暴躁的大佬 | 只有抱着女主才能睡着/腿好了/情绪稳定了（人形镇静剂） |
| 几个哥哥互相争斗，家庭由于睦 | 为了争夺"谁先抱妹妹"的权利，哥哥们居然和好了 |
| 女主想要天上的星星 | 爸爸直接买了一颗小行星命名为女主的名字 |
| 拥有锦鲤体质，随便指个股票 | 股票第二天涨停。爸爸："听闺女的，买！" |

## 3. 毒点黑名单 (Toxic Tropes to Avoid)

*   ❌ **熊孩子**: 萌宝必须懂事、乖巧，可以调皮但不能作恶。如果是熊孩子，读者会想打死她。
*   ❌ **只会哭**: 萌宝要有自己的逻辑，甚至比大人还通透。只会哭闹的小孩不可爱。
*   ❌ **团宠变团弃**: 无论发生什么误会，全家人对女主的爱不能变。不要写"全家误会女主"的虐心情节，要一直宠。
*   ❌ **恋爱脑**: 萌宝文前期重点是亲情。不要让三四岁的小孩搞对象，很雷。
