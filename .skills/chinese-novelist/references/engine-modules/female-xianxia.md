# 女频细分流派包：女频仙侠 (Female Xianxia)

> **核心逻辑**：修仙是为了救赎苍生，或者为了救赎那个他。情感线与事业线高度绑定。

---

## 1. 结构发动机 (Structure Engine)

### 黄金三章 (The Golden Three)
1.  **Chapter 1: 宿命相遇 (Destined Meeting)**
    *   **身份**：女主通常是低微小仙/废材弟子/魔族卧底。
    *   **男主**：高冷师尊/战神/魔尊。
    *   **事件**：女主在最狼狈的时候遇到了男主，或者女主无意中救了重伤的男主。

2.  **Chapter 2: 拜师学艺/共同历练 (Bonding)**
    *   **互动**：男主虽然冷漠但暗中护短。女主虽然弱小但性格坚韧。
    *   **冲突**：同门师姐/反派的嫉妒与刁难。

3.  **Chapter 3: 初露锋芒 (First Shine)**
    *   **高潮**：小秘境试炼/门派大比。女主爆发潜力（或动用禁术）保护了同伴/男主。
    *   **转折**：男主发现女主身上有特殊的秘密（身世/封印）。

### 核心差异点 (Vs Male Xianxia)
*   **男频修仙**：杀人夺宝，逆天改命，唯我独尊。
*   **女频修仙**：守护与牺牲，苍生与你，爱恨纠葛。

## 2. 爽点回收表 (Payoff Checklist)

| 铺垫 (Setup) | 回收 (Payoff) |
| :--- | :--- |
| 高冷师尊不近女色 | 师尊为了女主对抗整个修真界 |
| 女主被视为灾星/魔种 | 女主最终牺牲自己拯救苍生（然后复活） |
| 反派女配仗势欺人 | 女主觉醒神女血脉，血脉压制 |
