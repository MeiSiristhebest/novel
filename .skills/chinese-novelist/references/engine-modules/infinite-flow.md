# 高流量引擎包：无限流 (Infinite Flow)

> **核心逻辑**：主神空间 -> 任务发布 -> 生存/变强。通过由于不断变化的副本世界，提供极强的新鲜感和节奏感。

---

## 1. 结构发动机 (Structure Engine)

### 核心循环 (The Loop)
1.  **进入副本**: 被拉入主神空间/惊悚游戏，强制执行任务。
2.  **探索规则**: 发现副本的死亡规则或隐藏剧情（不仅仅是打怪）。
3.  **博弈破局**: 利用智谋、道具或团队配合，在绝境中求生。
4.  **结算强化**: 回到安全区，用积分兑换血统、技能、装备，为下一个副本做准备。

### 黄金三章 (The Golden Three)
1.  **Chapter 1: Yes or No**
    *   **事件**：主角在现实中感到通过绝望/无聊，点击了电脑上的弹窗。
    *   **觉醒**：在冰冷的车厢/废弃教室醒来，周围是一群神色各异的新人和资深者。
    *   **立威**：资深者讲解规则，刺头新人不信邪被抹杀，主角迅速接受现实。

2.  **Chapter 2: 初次危机**
    *   **事件**：丧尸/异形/鬼魂出现，队伍冲散。
    *   **行动**：主角冷静观察环境，发现了一把武器或一个安全点。
    *   **表现**：由于没有强化，主角只能靠智斗或利用地形反杀第一只小怪。

3.  **Chapter 3: 隐藏支线**
    *   **事件**：系统发布主线任务（存活3天）。
    *   **转折**：主角注意到一个不起眼的细节（NPC的这就话），触发了隐藏支线任务。
    *   **收益**：风险巨大，但奖励是新手唯一的神器。主角决定富贵险中求。

## 2. 爽点回收表 (Payoff Checklist)

| 铺垫 (Setup) | 回收 (Payoff) |
| :--- | :--- |
| 现实中主角是绝症患者/杀手 | 在副本中，绝症变成了抗毒体质，杀手本能让他如鱼得水 |
| 资深者想要把新人当炮灰 | 主角利用规则反坑资深者，继承了资深者的遗产 |
| 系统商店里昂贵的神级血统 | 主角在副本里通过隐藏任务，获得打折券或直接获得血统 |
| 必死的高难度开局 | 主角卡BUG通关，系统评价"S级"，全服通告 |

---

## 3. 毒点黑名单 (Toxic Tropes to Avoid)

*   ❌ **圣母泛滥**: 在主神空间救人是大忌，除非有巨大利益。为了救必死的龙套把自己搭进去是剧毒。
*   ❌ **强化杂乱**: 主角今天学魔法，明天学修真，后天变异形。体系要专一（如：智战流、修真侧、科技侧）。
*   ❌ **副本无聊**: 不要直接照抄电影剧情而不加改动。必须要有"魔改"或"蝴蝶效应"。
*   ❌ **现实剧情太长**: 读者想看的是副本里的冒险，回到现实装逼可以有，但不能喧宾夺主。
