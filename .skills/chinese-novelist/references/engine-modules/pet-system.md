# 高流量引擎包：御兽流 (Pet/Taming System)

> **核心逻辑**：养成 -> 进化 -> 羁绊。主角本身的战斗力可以不强，全靠宝宝打。强调"收集癖"和"养成感"。

---

## 1. 结构发动机 (Structure Engine)

### 核心循环 (The Loop)
1.  **契约**: 获得一颗死蛋/一只土狗，或者在野外捡漏一只变异兽。
2.  **培育**: 喂食特殊材料，进行魔鬼训练，挖掘潜力。
3.  **进化**: 满足条件，发生形态变化（光效拉满），种族等级跃迁。
4.  **对战**: 指挥宠兽进行宝可梦式的对战，属性克制与技能搭配。

### 黄金三章 (The Golden Three)
1.  **Chapter 1: 契约废材**
    *   **事件**：全民御兽的世界，主角在选宠仪式上只能选一只看起来最弱的（青虫/鲤鱼）。
    *   **冲突**：同学都选了火龙、雷狼，嘲笑主角。
    *   **钩子**：主角的系统能看到"隐藏进化路线"。这只青虫能进化成裂空座。

2.  **Chapter 2: 第一次进化**
    *   **事件**：主角利用系统，给宠物喂了奇怪的东西（如：辣椒）。
    *   **转折**：宠物痛苦挣扎，然后茧破，进化为"烈焰吐息虫"。
    *   **反馈**：外形变得帅气/呆萌，学会了第一个强力技能。

3.  **Chapter 3: 初战告捷**
    *   **事件**：之前的嘲讽者来找茬，发起御兽决斗。
    *   **高潮**：原本被认为是垃圾的宠物，一招秒杀了对方的稀有宠。
    *   **收尾**：宠物蹭蹭主角的脸求表扬。主角："我们的目标是世界最强御兽师！"

## 2. 爽点回收表 (Payoff Checklist)

| 铺垫 (Setup) | 回收 (Payoff) |
| :--- | :--- |
| 这个品种上限很低，大家都不要 | 主角培育出了变异种，不仅强，还带控制/毒 |
| 宠物性格奇葩（懒/贪吃/色） | 在关键时刻，这个奇葩性格反而成了致胜关键 |
| 传说中的神兽谁也没见过 | 主角的宠物即使是幼崽期，也散发出神兽威压，吓退万兽 |
| 敌人针对主角宠物的弱点制定战术 | 宠物进化了！属性变了！弱点消失了！ |

---

## 3. 毒点黑名单 (Toxic Tropes to Avoid)

*   ❌ **宠物甚至不如人**: 宠物必须可爱或者帅。如果长得丑还不可靠，读者不爱看。
*   ❌ **数量太多**: 贪多嚼不烂。重点塑造1-3只核心宠，其他的可以当工具宠。不要搞成亡灵天灾（除非是那个流派）。
*   ❌ **主角隐身**: 只有宠物打，主角在旁边喊"加油"。主角必须参与战斗（如：提供Buff，或者合体）。
*   ❌ **虐宠**: 即使是为了进化，也不能太残忍。要强调羁绊和爱。
