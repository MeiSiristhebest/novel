# 高流量引擎包：随身老爷爷流 (Grandfather System)

> **核心逻辑**：废材开局 -> 随身名师 -> 扮猪吃虎。主角负责惹事，老爷爷负责平事/教学。本质是**"我要把失去的尊严拿回来"**。

---

## 1. 结构发动机 (Structure Engine)

### 核心循环 (The Loop)
1.  **受辱**: 主角实力低微，被羞辱。
2.  **特训**: 老爷爷开启魔鬼训练，或者借力量给主角（代打）。
3.  **打脸**: 主角展现出超越等级的实力/见识。
4.  **危机**: 老爷爷沉睡/被发现，主角必须独立面对更强的敌人。

### 黄金三章 (The Golden Three)
1.  **Chapter 1: 天才陨落**
    *   **事件**：曾经的天才突然变成废材（修为倒退）。
    *   **冲突**：未婚妻退婚，家族冷眼，只有父亲/青梅竹马支持。
    *   **钩子**：主角对着戒指/项链发泄怒火，突然听到一声苍老的叹息。

2.  **Chapter 2: 师尊出山**
    *   **事件**：老爷爷现身（灵魂体），解释主角修为倒退的原因（被他吸走了）。
    *   **交互**：老爷爷为了补偿，传授神级功法/炼丹术。
    *   **转折**：主角从绝望瞬间转为狂喜，重以此誓要通过修炼复仇。

3.  **Chapter 3: 初试锋芒**
    *   **事件**：家族测试/购买药材遇到挑衅。
    *   **高潮**：在老爷爷指点下，主角一招击败了比自己高两级的龙套。
    *   **收尾**："三十年河东，三十年河西，莫欺少年穷！"

## 2. 爽点回收表 (Payoff Checklist)

| 铺垫 (Setup) | 回收 (Payoff) |
| :--- | :--- |
| 众人嘲笑主角不会炼丹 | 主角炼出极品丹药，震惊全场（老爷爷附体/指点） |
| 主角遇到绝境，敌人太强 | 老爷爷："把身体交给我"，瞬间爆发斗宗强者恐怖如斯的气息 |
| 拍卖会上没人识货的破烂 | 老爷爷指出那是上古神器，捡漏成功 |
| 所谓的大师在主角面前装逼 | 老爷爷随口指出大师的错误，大师跪地拜师 |

---

## 3. 毒点黑名单 (Toxic Tropes to Avoid)

*   ❌ **老爷爷夺舍**: 除非是暗黑流，否则老爷爷必须是真心对主角好，像父亲一样。
*   ❌ **过度依赖**: 主角遇到什么事都喊"老师救我"，会显得巨婴。关键时刻主角必须有自己的决断。
*   ❌ **老爷爷太弱**: 老爷爷必须是曾经的世界巅峰体验卡，不能连个新手村BOSS都搞不定。
*   ❌ **没有代价**: 借用老爷爷力量通常会有副作用（虚弱期/老爷爷沉睡），增加紧张感。
