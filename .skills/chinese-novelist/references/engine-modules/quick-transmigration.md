# 高流量引擎包：快穿流 (Quick Transmigration)

> **核心逻辑**：任务发布 -> 攻略副本 -> 结算奖励。节奏极快，多重世界观体验，主角通常是无情的任务机器或戏精。

---

## 1. 结构发动机 (Structure Engine)

### 核心循环 (The Loop)
1.  **接受任务**: 系统发布任务（如：拯救炮灰女配、攻略反派BOSS、打脸渣男）。
2.  **进入世界**: 快速交代原主背景和悲惨遭遇（拉仇恨）。
3.  **逆袭攻略**: 利用系统道具/自身演技/熟知剧情，扭转局势。
4.  **完成离开**: 不带走一片云彩，或者死遁（让攻略对象追悔莫及）。

### 黄金三章 (The Golden Three)
1.  **Chapter 1: 绑定与穿越**
    *   **事件**：主角死亡/被选中，绑定快穿系统。
    *   **背景**：快速介绍第一个世界的任务："原主被渣男抛弃，家破人亡。你的任务是让渣男爱上你然后甩了他。"
    *   **钩子**：主角睁开眼，正好在虐心名场面现场。

2.  **Chapter 2: 华丽反击**
    *   **事件**：渣男/反派正在羞辱原主。
    *   **行动**：主角接管身体，性情大变，一巴掌扇过去或冷冷回怼。
    *   **反馈**：渣男/反派惊讶："她怎么变了？"（引起了注意）。

3.  **Chapter 3: 布局**
    *   **事件**：主角分析局势，制定攻略计划。
    *   **高潮**：初次交锋，主角利用信息差坑了反派一把。
    *   **收尾**：主角对着镜子露出邪魅一笑，"好戏才刚刚开始"。

## 2. 爽点回收表 (Payoff Checklist)

| 铺垫 (Setup) | 回收 (Payoff) |
| :--- | :--- |
| 原主被全网黑，万人唾骂 | 主角拿出铁证反转舆论，全网道歉 |
| 渣男视原主为草芥 | 渣男爱主角爱得死去活来，主角转身离开 |
| 反派BOSS冷酷无情 | 反派BOSS为了主角对抗世界，主角却只把他当任务对象 |
| 每一个世界的技能积累 | 在修仙世界学的剑法，在末世世界一剑砍翻丧尸 |

---

## 3. 毒点黑名单 (Toxic Tropes to Avoid)

*   ❌ **沉溺剧情**: 快穿的精髓是"快"。一个世界不能写太长（通常30-50章），否则读者会腻。
*   ❌ **对路人动情**: 主角必须是"万花丛中过，片叶不沾身"，除非是1V1切片男主。
*   ❌ **原主太蠢**: 原主可以惨，但不能蠢得无可救药，否则主角接盘很恶心。
*   ❌ **同质化**: 避免连续两个世界都是古代宫廷或现代校园，要风格跳跃（古代->末世->星际）。
