# 五感描写库 (Sensory Library)

> **核心理念**：人类通过五感认识世界，写作也应该如此。
> **目标**：让读者不仅"看到"故事，还能"听到""闻到""摸到""尝到"。

---

## 一、五感的重要性

### 1.1 为什么需要五感？

```markdown
只用视觉（平面）:
> 房间很破旧。

加入五感（立体）:
> 房间很破旧。墙皮剥落，露出发黑的砖块（视觉）。
> 空气中弥漫着霉味和老鼠屎的臭味（嗅觉）。
> 脚下的木板咯吱作响，踩上去软得像要塌（听觉+触觉）。
```

### 1.2 五感使用频率建议

| 感官 | 使用频率 | 原因 |
|-----|---------|------|
| **视觉** | 60-70% | 人类最依赖的感官 |
| **听觉** | 20-25% | 制造氛围、节奏 |
| **触觉** | 10-15% | 身体代入感 |
| **嗅觉** | 5-10% | 记忆触发、环境质感 |
| **味觉** | 1-5% | 特定场景（吃饭、接吻） |

**警告**：不要平均分配，视觉永远是主导。

## 二、视觉描写库 (Visual)

### 2.1 光线与色彩

#### 阳光

| 时间/状态 | 描写词库 |
|---------|----------|
| **晨光** | 晨曦、熹微、第一缕阳光、透过薄雾、金色的细线 |
| **正午** | 刺眼、炽烈、白花花的、晒得发烫、影子缩成一团 |
| **夕阳** | 血色、橙红、余晖、镀金、最后一抹光 |
| **斜射** | 光柱、洒落、切割阴影、在尘埃中显形 |

**示例**：
```markdown
✅ 晨光透过破窗，在地上切出一道光柱。
✅ 正午的阳光像刀子，割在皮肤上。
✅ 夕阳如血，把半边天都染红了。
```

#### 月光/星光

| 类型 | 描写词库 |
|-----|----------|
| **月光** | 如水、银白、清冷、洒落、铺满、幽冷 |
| **满月** | 圆盘、明亮如昼、皎洁 |
| **残月** | 弯钩、一牙、像伤口 |
| **星光** | 稀疏、稠密、闪烁、冰冷、遥远 |

**示例**：
```markdown
✅ 月光如水，漫过窗台，漫过地板。
✅ 残月像一道伤口，挂在黑色的天幕上。
✅ 星星稀疏，冷得像要掉下来。
```

#### 人造光

| 光源 | 描写词库 |
|-----|----------|
| **烛光** | 摇曳、跳动、昏黄、摇摇欲灭、投下晃动的影子 |
| **油灯** | 豆大的光、昏暗、烟熏 |
| **火把** | 噼啪作响、火星飞溅、照亮方圆十米 |
| **霓虹灯** | 闪烁、刺眼、五颜六色、反射在水洼里 |
| **路灯** | 昏黄、拉长影子、把雾照成淡黄色 |

### 2.2 色彩描写

**原则**：不要只说"红色"，要说"什么样的红"

| 基础色 | 升级描写 |
|-------|----------|
| 红 | 血红、殷红、朱红、猩红、砖红 |
| 白 | 雪白、惨白、苍白、乳白、灰白 |
| 黑 | 漆黑、墨黑、乌黑、深黑、黑得发亮 |
| 蓝 | 湛蓝、深蓝、灰蓝、冰蓝、藏蓝 |
| 绿 | 碧绿、翠绿、墨绿、深绿、嫩绿 |
| 黄 | 金黄、土黄、昏黄、焦黄、枯黄 |
| 灰 | 铅灰、鼠灰、烟灰、银灰 |

**进阶技巧**：用对比强化色彩

```markdown
✅ 血在雪地上格外刺眼
✅ 白衣在夜色中像一抹鬼影
✅ 红裙在灰色的人群中晃过
```

### 2.3 形态与动作（视觉动态）

| 动作类型 | 动词库 |
|---------|--------|
| **液体** | 流淌、滴落、渗出、蔓延、溅起、涌出 |
| **烟雾** | 升腾、弥漫、缭绕、飘散、翻滚 |
| **火焰** | 燃烧、跳动、舔舐、吞噬、爆裂 |
| **风** | 卷起、吹拂、呼啸、掀起、扫过 |
| **人物** | 踉跄、疾驰、蹒跚、闪现、坠落 |

## 三、听觉描写库 (Auditory)

### 3.1 自然声音

#### 风声

| 强度 | 描写 |
|-----|------|

| 类型 | 描写 |
|-----|------|

```markdown
词库: 炸响、轰鸣、滚过天际、震耳欲聋、撕裂夜空、闷雷
```

| 骨折 | 咔嚓、脆响、骨裂声 |

| 地面 | 声音描写 |

|-----|------|

| 客观描述 | 情绪化描写 |
|---------|-----------|
| 风在吹 | 风像在哭泣 |
| 门响了 | 敲门声像催命符 |
| 很安静 | 安静得能听见心跳 |
| 很吵 | 嘈杂得让人想砸东西 |

## 四、嗅觉描写库 (Olfactory)

### 4.1 为什么嗅觉重要？

**特性**：嗅觉是最能触发记忆的感官

```markdown
示例:
> 血腥味。
> 他闻到的瞬间，就想起了三年前那个雨夜。
```

| 草木 | 草木的清香、松脂味、腐木的霉味 |
| 海 | 咸湿的海风、腥咸的海水味 |
| 雨 | 雨水的清新、潮湿的水汽 |

#### 食物气味

| 类型 | 描写 |
|-----|------|
| 肉 | 烤肉香、油脂的香味、焦香 |
| 酒 | 酒气、浓烈的酒精味、酒香 |
| 茶 | 茶香、清苦的味道 |
| 腐败 | 恶臭、腐烂的气味、发酸的 |

| 类型 | 描写 |
|-----|------|

| 汗 | 汗臭、酸臭、体味 |

## 五、触觉描写库 (Tactile)

### 5.1 触觉的重要性

**作用**：身体代入感最强的感官

### 5.2 温度

| 感受 | 描写词库 |
|-----|----------|
| **冷** | 冰凉、刺骨、冻得发麻、寒意渗入骨髓 |
| **热** | 灼热、滚烫、烫得起泡、热浪扑面 |
| **温暖** | 温热、舒适的温度、暖意 |
| **温差** | 一冷一热、温度骤降、从温暖到冰冷 |

**示例**：
```markdown
✅ 剑柄冰凉，握在手里像握着一块冰。
✅ 火焰近在咫尺，脸上的皮肤都快烤焦了。
✅ 她的手很暖，像冬天的炉火。
```

### 5.3 质感

| 质感 | 描写词库 |
|-----|----------|
| **粗糙** | 粗糙如树皮、磨砂的、扎手的、满是老茧 |
| **光滑** | 光滑如镜、丝绸般、细腻 |
| **湿润** | 潮湿、黏糊糊的、湿漉漉的 |
| **干燥** | 干裂、粗糙、起皮 |
| **柔软** | 柔软、蓬松、像棉花 |
| **坚硬** | 坚硬如铁、冰冷的金属质感 |

### 5.4 疼痛

| 类型 | 描写 |
|-----|------|
| **刺痛** | 像针扎、像刀割、刺痛感 |
| **钝痛** | 闷痛、沉重的痛、压迫感 |
| **灼痛** | 火烧般的、灼烧感 |
| **麻木** | 失去知觉、麻木、像不是自己的 |
| **剧痛** | 撕裂般的痛、痛得要命、痛得想晕过去 |

**示例**：
```markdown
✅ 刀口火辣辣的疼，像被人用烙铁烫过。
✅ 拳头砸在脸上，整个脑袋都嗡嗡作响。
✅ 腿已经麻了，像不是自己的。
```

### 5.5 身体感受

| 感受 | 描写 |
|-----|------|
| **疲惫** | 双腿沉重如灌铅、浑身酸痛、累得站不起来 |
| **饥饿** | 肚子咕咕叫、饿得前胸贴后背、胃在抽搐 |
| **窒息** | 喘不过气、胸口被压住、呼吸困难 |
| **眩晕** | 天旋地转、站不稳、眼前发黑 |

## 七、五感的组合使用

### 7.1 多感官叠加

**原则**：一个场景中至少使用 2-3 种感官

```markdown
❌ 单一感官（只有视觉）
> 房间很暗,他看到桌上有一把剑。

✅ 多感官组合
> 【视觉】房间很暗,烛光摇曳。
> 【视觉】桌上躺着一把剑,剑刃反射着微弱的光。
> 【嗅觉】空气中弥漫着血腥味。
> 【听觉】远处传来脚步声,越来越近。
> 【触觉】他握住剑柄,冰凉刺骨。
```

### 7.2 场景类型的感官侧重

| 场景类型 | 主感官 | 辅助感官 |
|---------|-------|---------|
| **战斗** | 视觉 + 听觉 | 触觉（痛）、嗅觉（血） |
| **环境** | 视觉 + 嗅觉 | 听觉、触觉 |
| **感情** | 视觉 + 触觉 | 嗅觉（香水）、听觉（呼吸） |
| **恐怖** | 听觉 + 嗅觉 | 视觉（阴影）、触觉（寒冷） |
| **用餐** | 味觉 + 嗅觉 | 视觉、听觉 |

### 7.3 实战示例

#### 示例 1：战斗场景

```markdown
【视觉】剑光如电,划破夜空。
【听觉】金属碰撞的声音刺耳。
【触觉】剑风擦过脸颊,留下一道火辣辣的痕迹。
【嗅觉】血腥味在空气中弥漫。
【视觉】血雾在月光下散开,像一朵诡异的花。
```

#### 示例 2：恐怖场景

```markdown
【听觉】走廊里传来脚步声——但他明明是一个人。
【触觉】后背一阵发凉,像有人在盯着他。
【嗅觉】空气中飘来腐烂的臭味。
【视觉】走廊尽头,一个影子在晃动。
【听觉】脚步声停了——就在他身后。
```

#### 示例 3：感情场景

```markdown
【视觉】她站在夕阳里,半边脸都是金色的。
【嗅觉】风吹来,带着她身上淡淡的茉莉花香。
【触觉】她的手很凉,他握住,想把温度传给她。
【听觉】她的呼吸声很轻,但他听得清清楚楚。
【视觉】她的眼睛红了。
```

## 八、五感使用的禁忌

### 8.1 不要过度堆砌

```markdown
❌ 感官过载
> 他闻到了血腥味(嗅觉),听到了惨叫声(听觉),看到了尸体(视觉),感到寒意(触觉),嘴里泛起苦味(味觉)……

✅ 精选感官
> 血腥味扑面而来。
> 他捂住鼻子,但还是闻到了——铁锈味,混杂着尸体腐烂的臭味。
```

### 8.2 避免陈词滥调

| ❌ 老套 | ✅ 新鲜 |
|--------|--------|
| 花香扑鼻 | 花香甜腻得像要把人淹没 |
| 刺骨的寒风 | 风冷得像刀子,割在脸上 |
| 鲜血淋漓 | 血从伤口涌出,温热的,带着铁锈味 |

### 8.3 不要无效感官

**无效感官** = 没有携带情绪/信息的感官描写

```markdown
❌ 无效
> 他闻到了空气的味道。(什么味道?有什么意义?)

✅ 有效
> 空气中弥漫着硝烟味——战斗刚结束不久。
```

## 十一、实战练习

### 练习 1：单一场景,多感官改写

**原文（只有视觉）**：
> 他走进房间,看到桌上有酒。

**改写目标**：加入至少 3 种感官

**参考答案**：
> 【触觉】他推开门,门轴发出刺耳的吱呀声。
> 【嗅觉】酒气扑面而来,浓烈得呛人。
> 【视觉】桌上歪歪斜斜摆着几个酒瓶,有的倒了,酒渍顺着桌腿滴在地上。
> 【听觉】滴答,滴答,像一个缓慢的倒计时。

### 练习 2：情绪驱动五感

**场景**：主角站在师父的坟前
**情绪**：悲伤、怀念

**参考**：
> 【视觉】墓碑上的字已经模糊了。
> 【嗅觉】泥土的清香混杂着枯草的味道。
> 【触觉】风吹过来,冷得像刀子。
> 【听觉】远处传来乌鸦的叫声,哑哑的,凄厉。
> 【触觉】他跪下去,手按在冰冷的石碑上,像要把温度传过去。
